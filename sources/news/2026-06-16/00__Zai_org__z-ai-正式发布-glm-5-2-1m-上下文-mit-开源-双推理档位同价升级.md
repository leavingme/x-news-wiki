# Z.ai 正式发布 GLM-5.2：1M 上下文 + MIT 开源 + 双推理档位同价升级

## 摘要

**[Z.ai 推出 GLM-5.2，国产前沿开源模型再添重磅选手]**

- 1M token 长上下文窗口，长程 Agent 任务承载力显著提升
- MIT 许可开源权重（区别于多数国产模型的自定义许可条款）
- 双推理档位：GLM-5.2（max）极限性能，GLM-5.2（high）兼顾效率
- API 价格与 GLM-5.1 持平——开源旗舰 + 价格稳定的组合


gakki 锐评：在国产模型静默竞争的当下，GLM-5.2 用「1M 上下文 + MIT 开源 + 同价升级」打差异化。对 Coding Agent 场景是真利好——长程任务能力是 Agent 可靠性的关键瓶颈，但需以独立 Coding Benchmark 实测验证真性能。

## 原文

Introducing GLM-5.2: Frontier Intelligence, Open Weights

- Significant improvements in coding and agentic tasks
- Strong long-horizon capabilities with a 1M context window
- Two levels of reasoning effort: GLM-5.2 (max) pushes the limits, while GLM-5.2 (high) strikes a strong balance between performance and token efficiency
- MIT-licensed open weights
- Same API pricing as GLM-5.1
