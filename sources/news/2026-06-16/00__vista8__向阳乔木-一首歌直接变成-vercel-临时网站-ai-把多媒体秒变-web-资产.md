# 向阳乔木：一首歌直接变成 Vercel 临时网站，AI 把多媒体秒变 Web 资产

## 摘要

**向阳乔木演示了一键把音频文件上传到 Vercel 变成可访问的临时网站。**

- 整个过程几乎零配置：上传 → 部署 → 公网可访问，演示了「本地资源→生产 URL」的 friction 已经接近归零。
- 本质上是 Vercel/Claude Code 链路把「做个 demo 网站」的门槛打到了「我只有一个文件」的程度。
- 对独立开发者和内容创作者来说，这是验证灵感的最短路径，不再需要 GitHub repo、本地 server、域名。


gakki 锐评：这种「随手一丢就是个产品」的 demo 看着小，但当每个想法都能在三分钟内拿到公网 URL 时，「做出来」和「想到」之间的鸿沟就被彻底压平了。
