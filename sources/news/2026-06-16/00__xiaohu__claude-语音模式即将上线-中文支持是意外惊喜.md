# Claude 语音模式即将上线，中文支持是意外惊喜

## 摘要

**Anthropic 终于补齐多模态语音短板，中文支持超出预期。**

要点：
- 设置页出现语音语言与风格配置入口
- 明确支持中文——是产品层面"非英语优先"的少有信号
- 据传底层采用 11Lab 模型（外部供应而非自研）

锐评：Claude 语音选择"接入成熟方案"而非自研，是符合 Anthropic 一贯边界的产品决策——把模型能力做透，把语音这种工程问题外包。中文化是更值得关注的信号，说明 Anthropic 在认真考虑中文市场，而不是只把中国当 API 套利场。

## 原文

看来 Claude 的语音模式即将推出

而且支持中文，出乎意料
