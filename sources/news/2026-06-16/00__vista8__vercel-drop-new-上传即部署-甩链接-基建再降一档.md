# Vercel drop.new：上传即部署，"甩链接"基建再降一档

## 摘要

**Vercel 把部署门槛压到"打开页面-拖入文件-生成链接"，内容分发形态被进一步压缩。**

要点：
- 打开 drop.new，上传 zip/文件/文件夹即生成可访问网站
- 测试上传 mp3 也能直接播放——轻量内容托管能力被内置
- 与"Agent 乐高化/甩链接即插即用"趋势形成基础设施层呼应

锐评：Vercel 持续在做"基础设施隐形化"——部署从 CLI 到 Git 到拖拽，每一次降门槛都让独立开发者的分发链路更短。当 mp3 都能直接 drop，文件分享/产品 demo/客户交付的形态都会变。
