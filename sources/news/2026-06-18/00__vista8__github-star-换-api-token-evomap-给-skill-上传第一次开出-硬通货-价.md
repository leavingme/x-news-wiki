# GitHub Star 换 API Token：EvoMap 给 Skill 上传第一次开出"硬通货"价

## 摘要

一句话核心判断：EvoMap 把开源 Star 折算成 API Token 积分，Skill/Gene/Capsule 上传额外奖励 Credits——Skills 贡献首次有了可定价的激励层。

要点：
• Star 数决定基础积分等级，一星即可参与
• 提交工作流/Prompt/工具封装为 Gene 或 Capsule 可拿额外 API Credits
• 排行榜机制上线，开发者从"声誉驱动"逐步过渡到"经济驱动"


Gakki 锐评：Skill 经济从"分享文化"试探"硬通货闭环"的关键一步；但 API Credits 的真实购买力和活动可持续性决定它是一次营销还是真生态——别急着押注，先看三个月。

## 原文

Github 开源项目的 Star 不止好看，还能换成大模型 API Token！

EvoMap 搞了个活动，只要你有开源项目就能领Token。

操作很简单：
① 提交自己参与或维护的Github仓库地址
② 验证通过，立即领取积分（Star 越多基础奖励等级越高，一个Star就能参与）

另外，他们想扩大开发者生态，只需把自己的工作流、Prompt或实用工具封装成EvoMap的Gene/Capsule提交。
上传Skill能获得额外的API Credits。

推荐有Github项目的朋友试试，先领个基础Token，视频中是操作教程。

活动地址：https://t.co/RCdwpw9Q7M

我琢磨，要不要把最近开源的 Skill 也都改造放上去，参与下排行榜PK，哈哈哈！
