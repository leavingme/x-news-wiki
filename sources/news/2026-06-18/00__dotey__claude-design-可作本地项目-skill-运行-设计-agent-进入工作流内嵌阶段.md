# Claude Design 可作本地项目 Skill 运行：设计 Agent 进入工作流内嵌阶段

## 摘要

**Claude Design 不再是独立工具，开始嵌入 Agent 工具链。**
- 宝玉提示：可在本地项目中以 Skill 形式加载 Claude Design，进而调用其他 Skills
- 配合 baoyu-design 的"本地生成 + Cursor/Codex 浏览器预览 + 可编辑 PPTX 导出"升级，设计生成能力开始与项目工作流深度耦合
- 内置浏览器标记功能支持二次修改，闭环"生成—审阅—再编辑"链路

锐评：设计 Agent 化的入口正在打开——Claude Design 从"独立工具"变成"Skill 单元"，意味着 Anthropic 在抢占 Agent 设计层的分发位。

## 原文

@30Thrd @claudeai got it. you can add other skills to your project if you can run claude design as a skill from a local project. it's worth a try.
