# 天策的 Token 经济学锐评：觉得 Token 贵的人，本质是 ROI 为负

## 摘要

**Token 单价不是问题，问题是用 Token 的人赚不回来。**

- 天策断言：所有抱怨 Token 贵的人，是自己的 ROI 为负；一旦 ROI 转正，越用越上瘾
- 这是用"单位经济性"取代"绝对成本"的视角——贵/便宜是结果，不是输入
- 对应判断：AI 工具真正的护城河不是把 Token 做便宜，而是让用户场景的 ROI 跑正


gakki 锐评：和 Sequoia 的"杠杆 100"逻辑一脉相承——Agent 经济的胜负不在模型成本表上，在用户账本上。
