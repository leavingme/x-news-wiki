# 歸藏点赞：Codex 补贴期仍守住服务稳定，把同行卷到降额降速

## 摘要

**在重补贴 + 用户暴涨的同时仍维持稳定服务,Codex 正在用'工程纪律'拉开身位。**
- 歸藏观察:Codex 即使在用户高速增长阶段,依然能保持稳定且快速的服务响应
- 对比鲜明:同期其他厂商普遍出现服务降速或额度下调,体验差距被用户侧放大
- 暗合主线:与宝玉定调的'模型能力才是王道'形成互证——Codex 的稳定性优势不能全甩给 Harness

服务稳定性是 Agent 工具的隐形护城河。模型层可以靠开源追平,但用户体验层的工程纪律需要时间沉淀。歸藏这条点评,是对 OpenAI 工程能力的一次隐性背书,也是对'Codex 与 Claude Code 之争本质在底层模型'论的侧面注脚。

## 原文

现在发现 OpenAI 在服务上是真的牛逼。

在 Codex 这么补贴的情况下，用户增长也很快，依然能提供非常稳定而且快速的服务。

最近其他几家都慢得不行了，再不然就是给你降额度。
