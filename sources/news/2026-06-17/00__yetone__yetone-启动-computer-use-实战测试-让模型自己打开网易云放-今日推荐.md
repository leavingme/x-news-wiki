# yetone 启动 Computer Use 实战测试：让模型自己打开网易云放「今日推荐」

## 摘要

**Computer Use 终于从演示视频走到日常 App 实操，yetone 用「打开网易云音乐、播放今日推荐」当 hello world。**

- 选择音乐类应用作为测试目标：UI 复杂、有状态变化、有登录态，典型的「真实世界」入口
- 隐藏的对照实验：同一指令在 Claude / Codex / OpenClaw 上的完成度差异
- 折射出 Computer Use Agent 的核心瓶颈：跨应用导航、登录态保持、对非结构化 UI 的容错


hakki 锐评：选音乐 App 而不是填表，是对 Computer Use 难度的清醒判断。但 hello world 级别的测试能给出的信息增量有限，要看到真东西还得跑出具体的失败/成功录像。
