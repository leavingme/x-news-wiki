# AI Native 圈已默认 HTML Slides：PPT 正在成为「落伍格式」的行业暗号

## 摘要

**一线观察：AI Native 圈正在用脚投票抛弃 PPT。**

在十字路口开放麦现场，演讲者普遍采用 HTML Slides 做现场演示，而传统公司代表仍坚持 PPT。这一分化背后是 HTML 原生适配 AI 代码执行、实时渲染、无版本依赖的结构性优势。

- HTML Slides 意味着演讲内容可被 AI 直接执行、修改、版本化——这是 PPT 无法实现的
- 工具选择本身是一种身份宣示：AI Native 圈用 HTML 定义自己「能动手不动口」
- 传统格式的惯性正在被 AI 编程能力打破，格式战争已从代码蔓延到演示层

gakki：「格式是认知的投影。还在用 PPT 的人，不是落后，是在用旧世界的语法写新世界的句子——语法对了，意思也错了。」
