# Anthropic $57万年薪 vs Claude Code：AI工具的「替代税」经济学

## 摘要

**Anthropic以57万美元年薪招聘工程师，有人在评论区灵魂拷问：为什么不直接用Claude Code？**

- 这个问题的潜台词是：当AI工具成本远低于人力成本时，企业是否还有动力雇佣顶级工程师？
- 现实是Claude Code能提升单人产出，但无法替代跨团队协作、上下文判断和商业决策——至少目前不能
- 「AI替代高薪工程师」的叙事和「AI提效后工程师反而更贵」的现实正在并行，两种逻辑都有数据支撑

57万不是 Anthropic 的溢价，是整个市场对「能驾驭AI的人」的定价。工具便宜了，但能驾驭工具的人更贵了——这是技术史上反复出现的反直觉规律。
