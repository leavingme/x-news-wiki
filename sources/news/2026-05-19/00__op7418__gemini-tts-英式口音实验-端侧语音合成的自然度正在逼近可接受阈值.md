# Gemini TTS 英式口音实验：端侧语音合成的自然度正在逼近可接受阈值

## 摘要

**[语音 Agent 的口音工程正在成为多模态 Agent 落地的前哨问题]**

歸藏(guizang.ai) 测试 Gemini TTS 并指出：忽略口音因素，表现不算差，但口音本身破坏了自然度。他将口音问题归因于选择了内置的英式口音模型。

信息增量：暴露了当前端侧 TTS 在口音适配层面仍需人工选择模型，尚未实现「语调节奏的本地自适应」——这对于语音 Agent 的落地体验是关键细节。

**gakki 判断：** 语音 Agent 的竞争已从「能说话」进化到「口音像人」阶段，Gemini TTS 在这个维度还差临门一脚。
