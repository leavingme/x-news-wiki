# 「AI 公司只分两种：生产 Token 的，和分销 Token 的」——天策重写 AI 公司分类法

## 摘要

**天策给出一个干净的 AI 公司二分法：上游做 Token 产能，下游做 Token 分发，中间没有第三类。**

- 核心论断：Skill、Agent、Workflow 这些当前最热的概念，单独看都不赚钱
- 真正的钱从 Token 流过：模型厂商、推理云、分发平台、订阅渠道才是利润中心
- 对应「工业时代」的分销-制造框架，类比传统消费品链路


gakki 锐评：这个框架是给「想做 Agent/Skill 创业」的人泼冷水——你的护城河如果不是 Token 成本或分发入口，那就只是「为他人做嫁衣」的应用层。但框架也忽略了一类公司：把 Token 转化为非 Token 资产（比如代码所有权、IP、品牌）的下游服务，这类才是真正赚钱的应用层。

## 原文

在这个世界上，所有AI公司只分为两种

要么是生产Token的，要么是分销Token的
不存在任何其他第三类公司

当你沉溺于Skill，沉溺于Agent
你会发现你根本赚不到钱

但是如果从Token的角度来看的话
就像上个时代的传统工业品一样

你只有顺着钱的流动，才能真正赚到钱
