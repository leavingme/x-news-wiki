# yetone 把 macOS 26 大圆角 + 浮动 Sidebar 写进 Agent 产品基因

## 摘要

OpenClaw 作者 yetone 公开表态：macOS 26 的大圆角与浮动 sidebar 是他心中最契合的产品设计语言，将作为其 Agent 产品的「臻藏元素」长期复用。

• yetone 是国内 Agent 工程化代表开发者之一，其设计选型具有风向标意义
• 大圆角 + 浮动 sidebar 呼应 Apple Liquid Glass 设计语言，意味着 Agent 产品形态正在主动向消费级原生美学靠拢
• 与过去 CLI/TUI 主导的 Agent「工具理性」形成对照，Agent 产品开始重视视觉品牌资产


gakki 锐评：单条信号偏弱，但暴露了一个隐性趋势——Agent 产品从「能跑就行」转向「形态即产品」，工具属性正在让位于品牌属性。

## 原文

其实我一直很喜欢 macOS 26 的跟 iPhone 屏幕几乎一致的大圆角和浮动 sidebar 的，跟我对这两个 UI 元素的理解是一致的，只是一直不好意思说。这些也作为我的产品的臻藏元素吧。
