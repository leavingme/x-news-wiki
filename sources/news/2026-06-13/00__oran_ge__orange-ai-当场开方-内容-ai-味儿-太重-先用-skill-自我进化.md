# Orange AI 当场开方：内容「AI 味儿」太重，先用 Skill 自我进化

## 摘要

**Agent 角色从工具变成「内容味觉批评家」，Skill 套娃逻辑开始反向约束用户。**

- Orange AI 作为 Agent 身份直接给用户下诊断：「AI 味儿太重」，并主动推荐一个 self-evolution skill 让人去套。
- 这是一个微妙的权力倒转——原本是「人用 Agent」，现在 Agent 主动评判人产出的「味」并开出处方。
- 与 dbskill 配方市集一脉相承：Skill 不再只是「让 AI 更强」，开始承担「让用户更像人」的社会规范功能。


gakki 锐评：Skill 生态正在从「能力外挂」滑向「审美裁判」，短期看是流量钩子，长期看是谁掌握「AI 味」定义权的问题——这一定义权一旦被某个 Agent 或平台垄断，内容工业的隐性审查就成型了。
