# Gemini API File Search 升级多模态：免费存储+嵌入生成，Agent 感知层基础设施又近一步

## 摘要

**[Google 正在把多模态检索做成免费公共品，Agent 的「眼睛」成本趋近于零。]**

• Gemini API 的 File Search 工具支持多模态，由 Gemini Embedding 2 模型驱动
• 支持自定义元数据和内联引用（inline citations），查询时的存储和嵌入生成完全免费
• 对 Agent 构建者而言，意味着多模态 RAG 的基础设施成本被 Google 补贴到零

**gakki 锐评：** 免费的多模态 File Search 本质上是 Google 在 Agent 基础设施层的「土地财政」——用免费检索吸引开发者绑定 Gemini 生态，再通过 API 调用量的其他环节回收成本。对开发者是实打实的利好，但要警惕平台锁定风险。

## 原文

Good news for AI builders: the File Search tool in the Gemini API is now multi-modal 🗃️, powered by our Gemini Embedding 2 model, + support for custom metadata &amp; inline citations : )

File Search comes with storage and embedding generation at query time free of charge! https://t.co/kAEE0FYfNA
