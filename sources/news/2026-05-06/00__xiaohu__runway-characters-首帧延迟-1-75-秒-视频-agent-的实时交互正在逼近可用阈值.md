# Runway Characters 首帧延迟 1.75 秒：视频 Agent 的实时交互正在逼近可用阈值

## 摘要

**从用户停止说话到角色返回第一帧约 1.75 秒，实时视频 Agent 的延迟正在进入可接受区间。**
- 1.75 秒的端到端延迟（含语音识别、推理、视频生成）对实时交互场景而言已接近可用
- 对比传统视频制作流程，这已是数量级的压缩，但距离「对话级流畅」仍有 gap

gakki 锐评：1.75 秒是个微妙的数字——不快到无感，但慢到你会意识到「这不是真人」。视频 Agent 的真正瓶颈不在延迟，在于当延迟存在时，交互设计能否让用户接受。
