# Codex App 支持一键导入其他 Agent 配置：Agent 工具链的可移植性正在落地

## 摘要

**OpenAI Codex App 上线 Import other agent setup 功能，支持从其他 Agent 工具迁移配置。**

- 操作路径清晰：Settings → General → Import，可选择性导入检测到的项目
- 这意味着 Agent 生态正从「锁定」走向「互通」，用户的技能包和工作流开始具备跨平台迁移能力
- 对标 Skills 生态的标准化趋势，配置可导入本质上是 Agent 的「用户数据可携带权」

gakki 锐评：当 Agent 开始互相导入配置，平台的护城河就不再是用户数据，而是运行时体验——这才是真正的差异化战场。

## 原文

操作步骤：

1、打开 Codex App 的 Settings
2、在 General 页面找到 Import other agent setup
3、点击 Import 或 Import again
4、查看 Codex 检测到的内容，选择要带入的项目
5、导入完成后，可以点击 View imported files 检查结果 https://t.co/DbJD9cUwt2
