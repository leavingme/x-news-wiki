# 小互实测无内置技能的内容抓取：Agent 原生能力正在逼近「技能外挂」的必要性门槛

## 摘要

**不依赖预置技能，纯靠 Agent 循环完成内容抓取——这正在成为衡量 Agent 基线能力的新标尺。**
- 小互展示了完整的执行路径：从内容抓取测试 → 客户端执行路径 → 终稿输出，全程未调用任何内置技能
- 这意味着 Agent 的通用推理 + 工具调用能力已能覆盖过去需要专门 Skill 封装的场景
- 对 Skills 生态的隐含挑战：当基座能力足够强，Skill 的价值将从「能不能做」转向「做得多快多稳」

gakki 锐评：无技能执行是能力证明，但生产环境要的是可复现性——裸跑能出活不代表应该裸跑，Skills 的真正护城河是确定性而非可能性。
