# Runway Characters 不只是数字人：视频生成公司正在悄悄长出 Agent 能力

## 摘要

**视频生成工具正在从「画面产出」跃迁到「角色行为编排」，Runway Characters 具备类似 Agent 的交互与响应能力。**
- Runway Characters 并非传统 TTS 头像，而是融合了实时对话、场景感知与角色行为编排的多模态系统
- 这意味着视频生成公司正在从工具层向 Agent 层渗透，角色不再是素材而是执行单元

gakki 锐评：Runway 选择在「角色」这个锚点上叠加 Agent 能力，比直接做通用 Agent 聪明——垂直场景的控制权比通用能力更有护城河。
