# Hugging Face 官方 CLI 支持一键读取论文：学术信息获取的终端化最后一公里

## 摘要

**论文阅读正在从浏览器行为变成终端命令，Agent 研究工作流的摩擦再降一级。**
- `hf papers read` 命令同时支持 arXiv 和 HF Paper URL，一行命令完成论文获取
- 对 Agent 而言，这意味着论文阅读可以无缝嵌入 terminal 工作流，无需切换浏览器上下文
- 从 curl + 解析 + 提取的三步流程压缩为一步，学术研究 Agent 的信息采集效率提升

锐评：小工具，大信号。当论文阅读变成 CLI 原生操作，Agent 的「研究-总结-推荐」闭环终于可以在终端内自完成了。

## 原文

如果你喜欢到Huggingface读最新的AI论文。

可以安装官方的cli：

curl -LsSf https://t.co/ET2pIYhJuU | bash

安装好以后，提示词：
hf papers read [论文编号或论文URL]

同时支持arxiv网站和Huggingface paper的URL。 https://t.co/TmsJOghXeg
