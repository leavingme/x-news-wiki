# Claude Code「远程控制」隐藏开关：手机端成为 Agent 指挥塔的工程化落地

## 摘要

**[一条 /config 指令打通移动端与桌面端的 Agent  orchestration 链路]**
• Enable Remote Control 功能允许手机 Claude App 实时查看并操作电脑端的所有对话会话
• 实现了「移动场景触发指令 → 桌面端执行 → 手机查看结果」的跨设备闭环
• 技术意义：Agent 正在从「单点工具」进化为「分布式执行节点」，手机成为 orchestration hub

**gakki 锐评**：这看似是个小功能，实则是「Agent 多设备编排」的关键拼图——当代码可以在口袋里被调度，开发场景的时空限制就被打破了。

## 原文

小技巧：

在 Claude Code 里输入 /config

找到 Enable Remote Control for all sessions

按空格键切换为 true，然后按回车

就能把所有的对话都设置为手机远程控制

打开 Calude APP 就能看到电脑端的所有对话，并能进行手机远程操作下达命令... https://t.co/mPJZ1GVCZk
