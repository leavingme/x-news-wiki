# Spatial-Agent：用空间科学理论替代网络搜索，地理推理Agent跑赢通用基线

## 摘要

**[一句话核心判断]** 通用Agent在城市规划、灾害响应等真实地理任务上仍然力不从心，Emory/Rutgers/UT Austin联合团队提出Spatial-Agent，把空间科学理论（方向、距离、拓扑）作为推理骨架，而非依赖网络检索或模式匹配。

- 技术核心：将自然语言问题拆解为 GeoFlow Graph 工作流，以空间科学基本概念为推理节点
- 评测表现：在 MapEval-API 与 MapQA 上击败强力基线（推文截断，未见具体数字）
- 信号意义：垂直领域 Agent 正在从「通用 LLM + 工具」走向「领域理论 + 推理图」范式

## 原文

Why do AI agents still struggle with real geospatial tasks like urban planning or disaster response?

Researchers from Emory, Rutgers, and UT Austin introduce Spatial-Agent—a new AI that grounds reasoning in spatial science theory instead of web search or pattern matching. 

It turns natural-language questions into step-by-step workflows called GeoFlow Graphs, using core concepts like direction, distance, and topology.

Result: On MapEval-API and MapQA benchmarks, Spatial-Agent beats powerful baselines like ReAct and Reflexion—and its workflows are fully interpretable.
