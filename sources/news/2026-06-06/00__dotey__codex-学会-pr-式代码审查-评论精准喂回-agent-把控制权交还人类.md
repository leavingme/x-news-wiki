# Codex 学会 PR 式代码审查:评论精准喂回 Agent,把控制权交还人类

## 摘要

**Codex 正在把 GitHub PR Review 的协作范式封装进 AI 编辑器,让「代码评论」成为 Agent 的精准上下文。**

- Codex 支持选择 Last turn 等代码修改范围进行审查
- 用户可对代码片段加评论,评论内容直接附加在会话左侧
- 提交评论即作为上下文发给 Agent,触发针对性修改
- 核心设计:不是让 AI 整体重写,而是让人能精准点出「这一段需要改什么」


Codex 在做的不是更强的 AI,而是「更克制的工程体验」——把修改决策权交还给开发者,这才是 Vibe Coding 喧嚣之后真正能跑生产环境的分水岭。

## 原文

Codex 里面可以 Review 代码的修改（比如选择Last turn就是上一次 AI 更改的），然后可以对部分内容进行评论，评论的内容会直接附加在左侧的会话内，提交就可以作为上下文发给 Agent，让它对评论处理 https://t.co/fvIFtQj0kx
