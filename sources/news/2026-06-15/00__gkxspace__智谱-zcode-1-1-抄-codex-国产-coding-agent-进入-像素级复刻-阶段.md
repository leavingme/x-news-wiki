# 智谱 ZCode 1:1 抄 Codex：国产 Coding Agent 进入'像素级复刻'阶段

## 摘要

**[国内 Coding Agent 同质化加速]**，智谱 ZCode 从权限模式、布局、交互到视觉风格全面对齐 Codex。

- 现象级对比：界面与交互逻辑几乎像素级一致，Kimi 桌面端'像'已经算客气，ZCode 这波是直接对标
- 反向推荐：腾讯 WorkBuddy 的'开箱即用、面向所有人'定位更适合国内市场（专家团、定时自动化、远程遥控）
- 国产模型层：依然只推 Kimi


gakki 锐评：当产品形态抄无可抄，真护城河只剩两件事——分发渠道（微信/企业微信/腾讯生态）与场景理解（WorkBuddy 强在'不懂技术的人也能用'）。这恰好对应玉伯的'想想能说清楚'，对每个用户都能讲出不同的故事。

## 原文

智谱的 ZCode 差点以为自己打开了 Codex。。。

权限模式、布局、交互逻辑、连风格都对齐了，之前觉得 Kimi 的桌面端比较像，ZCode 这波直接 1:1 了

针对国内市场，我反而觉得腾讯的 WorkBuddy 的定位更好，它是让所有人都能用 AI 干活的桌面 Agent。

主打开箱即用，我身边不会 翻墙 的那群人也是用 WorkBuddy 的最多。

Skills 生态、专家团、远程遥控、定时自动化...方向完全不同，但确实解决了更多人的问题。

另外，国产模型我还是只推荐 Kimi。。。。。。
