# 玉伯两字总结两年创业：祛魅+诚实，"诚实让 agents 特别听话"是隐性护城河

## 摘要

**玉伯给出 AI 创业方法论的极简版本：祛魅是第一性原理的起点，诚实是 agent 协作的隐性纪律。**

- **祛魅**：第一性原理是马斯克五步工作法的第一步，回归本源——"存在的不一定会继续存在"。这是对抗投资人和同行噪音的武器。
- **诚实的三层价值**：看清楚内心想做什么、让团队结构特别简单、**让 agents 特别听话**。
- **玉伯把"诚实"延伸到 agent 协作层**——这是少有人注意到的提法，意味着 prompt/harness 的可读性、可预测性，本质上是人对自己诚实程度的延伸。

延续玉伯近期对 AI 创业认知重构的系列思考。

**gakki 锐评**："诚实让 agents 特别听话"是玉伯这两年最有穿透力的一句——大多数人在抱怨 agent 不听话，实际上是在回避自己指令里的自欺。

## 原文

有人问我，一句话总结这篇文章，最大的收获是什么。我想了下，创业这两年最最重要的收获，是两个词：

第一个词是祛魅。创业很容易被各种声音干扰，有投资人的，有同行的，还有旧世界的各种指标。祛魅就是第一性原理，是马斯克五步工作法的第一步。回归本源，存在的不一定会继续存在，整个世界就会随着祛魅而明亮起来。

第二个词是诚实。祛魅的基础上，诚实可以让心特别静。诚实可以看清楚内心究竟想做什么。诚实可以让团队特别简单。诚实还可以让 agents 特别听话。

期待一个共同祛魅 + 诚实创业的时代到来。
