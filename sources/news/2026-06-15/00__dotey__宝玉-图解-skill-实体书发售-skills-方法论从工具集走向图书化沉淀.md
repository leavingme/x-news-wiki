# 宝玉《图解 Skill》实体书发售：Skills 方法论从工具集走向图书化沉淀

## 摘要

**Skills 范式正在从'工具集'升级为'方法论体系'，图书化是最强信号**

- 宝玉新书《图解 Skill —— AI 提效实战指南》正式发售，配套 GitHub 项目库 JimLiu/Illustrated-Agent-Skills 同步开源
- 书中沉淀了 info-digest（AI 资讯整理）、interview-analysis、interview-writing（播客访谈）等可复用 Skills
- 意味着 Skills 不再只是工程圈的私下玩法，而是有了系统化的中文教学载体


宝玉的写作工程化能力 + 开源项目 + 实体书的三件套，是 Skills 生态'基础设施层'形成的标志。当 Skills 能被画成图、能被装订成书，意味着它开始脱离'圈层黑话'阶段，进入规模化传播周期——下一步要警惕的是 Skills 教学化后的 GPTs 式通胀。
