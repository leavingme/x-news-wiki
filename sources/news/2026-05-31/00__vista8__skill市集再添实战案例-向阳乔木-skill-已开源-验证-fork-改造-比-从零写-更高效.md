# Skill市集再添实战案例：向阳乔木 Skill 已开源，验证「fork 改造」比「从零写」更高效

## 摘要

**[Skill 正在从工具演变为可传播的工件单位]**

向阳乔木发布已写好的 Skill 并开源，提供 npx 安装指令。这一动作比代码本身更有意义——它验证了 Skill 作为「可分发工作流单元」的可行性：发布者写好模板，用户 fork 改造，而不是从零构建。

关键信号：Skills 生态正在形成「模板市场」雏形，而不是封闭的工具库。

gakki：「Skill 的价值不在于工具本身，而在于它降低了改造的摩擦成本。能 fork 的 Skill 比能安装的 Skill 更符合 AI 协作逻辑——前者允许二次创作，后者只是下载。」

## 原文

Skill写好已开源，推荐star或fork试试：
https://t.co/FNMnrbFiLE

安装指令
npx skills add joeseesun/qiaomu-read-helper

前提是得安装飞书cli
