# 宝玉戳破 Agent 入口幻觉：超级助理的前提是用户先信任一个入口

## 摘要

**[单一 Agent 入口是产品终态，但用户信任路径是反的]**

宝玉提出：用户倾向统一入口，未来「什么事都跟超级助理 Agent 说」而非打开独立 App。这是对的方向，但忽略了一个现实——目前用户信任的入口是微信，不是某个 AI Agent。

信息增量：宝玉点出了入口收敛与用户信任之间的张力——Agent 要成为入口，先要解决「为什么我要把所有事交给它」，而不是「它能做什么」。

gakki：「Agent 入口的竞争本质上是信任入口的竞争。谁先占据『日常第一句话』的位置，谁就是真正的超级助理——在这个维度上，微信比任何 AI Agent 都领先三个身位。」

## 原文

关键是用户的入口在哪里，用户会倾向于一个统一的入口，就像未来每个人都会有一个贾维斯那样的超级助理 Agent，什么事都跟它说。

比如我要订机票不是打开订机票的 App，而是让 Agent 去帮我订，我只要说要求和确认结果。
