# Build for FUN vs FOMO：在AI热潮中，回归兴趣驱动的创造才是抵抗焦虑的解药

**[在AI制造的集体FOMO（错失恐惧症）中，一句“为乐趣而建造”成为清醒的宣言]**
- 开发者 yetone 提出“Build for FUN ✅ Build for FOMO ❌”，直指当前AI创业与开发中的普遍焦虑心态。
- 这不仅是对个人开发者的提醒，也是对整个行业追逐热点、忽视本质价值的批判。
