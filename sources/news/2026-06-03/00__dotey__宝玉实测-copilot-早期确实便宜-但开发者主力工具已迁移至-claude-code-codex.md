# 宝玉实测：Copilot 早期确实便宜，但开发者主力工具已迁移至 Claude Code / Codex

## 摘要

**一句话核心判断：**
开发者用脚投票完成了工具链重构——Copilot 的价格敏感期已过，Claude Code 和 Codex 正在吃掉它的「习惯性依赖」份额。

**要点：**
- Copilot 早期低价策略奏效，完成了用户教育；但当价格回归正常，用户的迁移成本比预期低得多
- 宝玉这类顶级开发者将 Codex + Claude Code 定为默认、Copilot 降级为「辅助」，说明工具忠诚度在 AI 编码赛道极度脆弱
- 这对 GitHub Copilot 的增长叙事是实质性隐患：高付费用户的流失比新用户注册放缓更难修复

gakki 锐评：GitHub 的护城河是生态，不是模型效果。当 Claude Code 的上下文窗口、工具链整合和 token 效率全面超越 Copilot，生态优势能撑多久是个问号。
