# Grok 推出跨平台 TUI 工具：交互式与无人值守双模式，支持任意自定义模型

## 摘要

[x-crawl 背后的 x-crawl 项目推出 grok 工具，实现交互式 TUI 与 Headless 无人值守两种运行模式，支持 macOS/Linux/Windows 并可切换任意自定义模型]

• 工具链向发布环节延伸：grok-build-0.1 单独开放 API，覆盖从开发到部署完整流程
• 不绑死模型：与 OpenClaw 等闭源工具形成差异化，强调开放性
• 无人值守模式配合脚本，可嵌入 CI/CD 或自动化工作流

x-crawl 作为 X Premium 登录三开源 Agent 的主力工具，其工具化进程正在加速。

Gakki 锐评：工具铺开了，但 Agent 真正的门槛不在"能跑"，而在"跑得对"——多模型路由的稳定性才是下一阶段的核心竞争点。

## 原文

两种运行方式

交互式 TUI：终端里全屏、可鼠标操作
Headless 无人值守：grok -p 加指令，塞进脚本自动跑

支持切换任意自定义模型，不绑死 Grok
同款模型 grok-build-0.1 也单独开放在 API 上
macOS / Linux / Windows 都能装
