# 宝玉自嘲 Prompt Engineering 过气：工程师人设的反向宣言

## 摘要

**Prompt Engineering 作为独立技能品类，被原作者点名为"过时"——这不是技术退步，而是工程范式迁移。**

-宝玉（dotey）作为中文圈推 Prompt 工程最具代表性的 KOL，反讽式宣告其过期
-朋友语境下用"过气了🤪"自嘲，实际承认 Agent 已接管"如何提问"这件事
-行业意义：Vibe Coding / Harness Engineering 把"提问技巧"内化进工具链，Prompt 不再是独立技能
