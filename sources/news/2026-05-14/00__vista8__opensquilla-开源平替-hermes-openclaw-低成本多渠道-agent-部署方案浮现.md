# OpenSquilla：开源平替 Hermes/OpenClaw，低成本多渠道 Agent 部署方案浮现

## 摘要

**[一句话核心判断]**

向阳乔木推荐开源项目 OpenSquilla，定位为 Hermes/OpenClaw 的低成本替代品。

- 支持本地 Skills 自动扫描 + 社区/GitHub 安装
- 兼容 Anthropic、OpenAI、OpenRouter、DeepSeek、智谱等主流模型
- 覆盖 Discord、飞书、Telegram、Slack 等聊天平台
- 强调更智能省 Token、易用性和安全性



gakki 的锐评：OpenSquilla 的出现印证了 Agent 入口层的竞争已从「能不能做」转向「谁能更便宜」，开源平权正在加速这一进程。

## 原文

OpenSquilla 会自动扫描本地已经安装的 Skills，也能从社区搜索安装、Github链接安装。

支持可视化配置海内外各种模型或中转，如Anthropic、OpenAI、OpenRouter、DeepSeek、智谱等。

聊天机器人频道也基本上都支持，比如Discord、飞书、Telegram、Slack等。

如果觉得 OpenClaw 和 Hermes 太烧钱，感觉可以试试OpenSquilla这个开源项目。

更智能省Token，易用性和安全性也还在不错。
