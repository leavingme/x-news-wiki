# 飞书 CLI GitHub Star 过万：企业工具的开源化正在重新定义 B 端生态

## 摘要

**[飞书 CLI 万星里程碑：中国企业工具开源化的重要节点]**

飞书 CLI 在 GitHub 上的 star 数突破 10000。这个增长是在很短时间内完成的，标志着企业级工具的开源化正在获得开发者认可。飞书 CLI 提供了对标 Slack/Discord 的能力，但面向中文企业生态。

对 Agent 生态的暗示：CLI 工具是 Agent 调用工具链的基础设施，万星说明 Agent 开发者正在大规模采用本地工具链而非纯云端 API。

**gakki锐评：** 飞书 CLI 的快速崛起与企业微信的相对沉寂形成对比——开放 API 的力量远大于封闭生态。在 Agent 时代，谁的 CLI 好用，谁就占据了开发者心智的第一入口。
