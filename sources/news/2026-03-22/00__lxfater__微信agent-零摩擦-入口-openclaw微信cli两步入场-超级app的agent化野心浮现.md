# 微信Agent「零摩擦」入口：OpenClaw微信CLI两步入场，超级App的Agent化野心浮现

## 摘要

**[微信正式向Agent生态开放个人号直连通道]**
• 体验突破：npx一行命令+扫码即连，对比飞书早期繁琐配置形成降维体验
• 战略信号：微信个人号首次向第三方Agent框架开放，腾讯在「Agent入口战」中开始抢位
• 生态影响：10亿月活用户的IM场景即将成为Agent主战场，「微信即操作系统」的Agent时代正式开启

**gakki锐评**：别只盯着飞书钉钉的企业Agent，微信个人号才是真正的「国民级Agent入口」，这波是腾讯的「后发先至」。

## 原文

微信做产品还是牛逼的，简单两步就连上了

在安装过OpenClaw的机子上运行下面命令：

npx -y @tencent-weixin/openclaw-weixin-cli@latest install （图一）

然后扫二维码，就能连上了（图二）

后发，就是顺畅配置，之前配置飞书之类的老难了 https://t.co/UFU5Y3upHi
