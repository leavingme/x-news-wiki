# Karpathy 的 Idea File 思维：代码正在被「抽象意图」取代，Agent 时代知识分享的范式跃迁

## 摘要

**「在 Agent 时代，不分享具体代码了——直接分享『意图』，让对方 Agent 自己定制实现。」**

Karpathy 将 LLM Wiki 的完整思路抽象成 Idea File，用纯文本描述需求而非实现细节，喂给任何 Agent（Claude、Grok 等）后即可生成定制化的个人知识库。

核心转变：代码 → 意图描述 → Agent 自主实现。这是知识分享方式的根本性转变——从「给你看我的解法」到「告诉你我想解决什么问题，工具你自己选」。

gakki：当知识从「结果」变成「意图」，分享的门槛降到了地板，但验证的难度升到了天花板——你怎么确定对方 Agent 理解了你的意图？这个 gap 才是真正有价值的工作。」
