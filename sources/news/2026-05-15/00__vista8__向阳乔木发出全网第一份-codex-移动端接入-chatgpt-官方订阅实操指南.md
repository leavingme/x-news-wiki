# 向阳乔木发出全网第一份 Codex 移动端接入 ChatGPT 官方订阅实操指南

## 摘要

**[向阳乔木率先填平了 Codex 与 ChatGPT 客户端之间的配置鸿沟]**

1. 官方订阅账号是必要条件，API 模式用户被排除在外——这意味着只有付费用户才能体验完整功能
2. 扫码登录强制使用原生相机，微信等第三方扫码工具失效——移动端入口限制比想象中更严格
3. 配置完成后可改电脑唤醒状态设置，暗示该功能设计之初就考虑了跨设备协作场景

这是全网首份具体可操作的 Codex 移动端接入指南，在各类插件和 API 模式教程泛滥的情况下，直击官方客户端订阅用户这一细分群体的配置痛点。

**gakki锐评：** Codex 在 ChatGPT 客户端里的这个"隐藏入口"设计，本质上是把 AI Coding 工具从极客玩具推向普通订阅用户的第一步——但强制原生扫码这一步已经筛掉了相当比例的国内用户，入口越窄，护城河越深，但天花板也越低。

## 原文

如何在ChatGPT 客户端用Codex？

很多人发新闻，就不发教程！其实配置稍微有点麻烦。

1. 更新Codex 本地客户端。

左侧会出现“设置 Codex 移动版”的入口。

注意！！！ 必须用官方订阅账号，API模式看不到这个入口。

2. 点击设置入口。
进去要求扫码，一定用苹果或安卓源生相机扫码。

（ChatGPT没找到扫码按钮，微信好像不行）

3. 登录ChatGPT账号（哪怕你App已登录账号）

4. 授权后搞定。
后续可改是否让电脑保持唤醒状态。

客户端下载地址见评论
