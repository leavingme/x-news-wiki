# 七十几个 skill 也敢用：向量语义匹配实现精准注入

## 摘要

**本地小模型根据语义只注入匹配度最高的 skill description，避免每轮七十几个 skill 全部注入带来的成本爆炸。**

- BM25 + 向量混合检索已是成熟范式，效果获实际验证
- 节省上下文开销同时保持匹配精准度

skill 资产越多，维护成本越高——这个方案让 skill 生态的规模化成为可能。
