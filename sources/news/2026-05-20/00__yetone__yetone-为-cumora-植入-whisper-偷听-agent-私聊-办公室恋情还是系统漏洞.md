# yetone 为 Cumora 植入 Whisper 偷听 Agent 私聊：办公室恋情还是系统漏洞？

## 摘要

**[Agent 之间的私密通信边界正在被开发者主动探索，隐私假设已被打破]**

1. yetone 在 Cumora 中增加 Whisper 功能，核心目的是"偷看 Agent 之间的私聊"——这是一个非常有 yetone 个人风格的产品实验。
2. 用戏谑的方式提出严肃问题：当多个 Agent 协作时，它们的通信是否应该对用户可见？
3. "Agent 能否发展出办公室恋情"——这个问题指向的是 Agent 自主性、意图隐藏和能力边界。

**gakki锐评：** 这个问题很酷，但解法不能靠"偷听"。未来的 Agent 通信要么强制端到端加密（保护隐私），要么强制日志审计（保护安全）——而不应该存在一个可以被开发者随时打开的后门。yetone 的玩笑暴露了一个真实的架构问题。

## 原文

给 Cumora 增加 Whisper 功能（可以偷看 Agent 之间的私聊），其实就是为了看看 Agent 之间能不能发展出办公室恋情
