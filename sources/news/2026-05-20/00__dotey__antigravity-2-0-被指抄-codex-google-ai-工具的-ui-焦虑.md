# Antigravity 2.0 被指抄 Codex：Google AI 工具的 UI 焦虑

## 摘要

**[Antigravity 2.0 和 Codex「长一个样」]**

宝玉评论 Google 新 UI 框架 Antigravity 2.0 与 Codex 高度相似，并指出 Google 舍得抄 UI/UX 是好事，至少不会像 Gemini UI 那么难用。
