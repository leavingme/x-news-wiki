# Codex App 插件化 Hack：Computer Use 和 Goal 指令的越狱式解锁

## 摘要

**一句话：Codex App 的插件机制意外撕开了一道口子，让 API 登录用户也能用上原本需要独立部署的 Computer Use 和 Goal 指令，顶部 Tab 化 UI 还顺带改善了多会话管理。**

要点：
- 即使纯 API 接入也能通过插件加载 Computer Use 能力，等效于降低了旗舰功能的门槛
- Goal 指令的加入让 Codex 从被动工具开始具备目标导向执行属性
- 开源社区正在用 Hack 方式缩短与官方功能发布的Gap，这对商业Agent平台的护城河是个警示信号


gakki 的锐评：开源社区的"越狱式解锁"正在变成Agent生态的常态——平台的功能领先窗口越来越短，护城河不能只靠功能，要靠数据锁和生态惯性。

## 原文

这个项目牛逼啊，增强（Hack）你的 Codex App。

即使你用API 登录Codex，也可以通过安装插件方式用上  Computer Use、添加上Goal指令。

界面样式也能调整，比如变成类似Chrome的顶部Tab。

设置任务开始、完成的声音提醒等等。

开源项目地址见评论区
