# Gemma 4 驱动 Open Duck：端侧多模态 Agent 的「轻量杀手级应用」现身

## 摘要

**【Google Gemma 4 落地实体机器人：视觉+对话+端侧部署三合一，Agent 硬件化进入快速迭代期】**

小互曝光由 Gemma 4 驱动的 Open Duck 机器人，具备视觉识别和对话能力，且运行在消费级硬件上。这是继开源模型端侧部署之后的又一次能力下放——从纯软件 Agent 延伸至实体硬件载体。

端侧多模态 + 低成本硬件 + 开源模型，这个组合意味着「视觉 Agent」的准入门槛正在被快速压缩。不需要云端算力，不需要昂贵 API，一个树莓派级别的设备就能跑一个会看会说的 Agent——这是 Agent 硬件民主化的前奏。
