# Dia 浏览器内测 Agent 晨报功能：AI 原生浏览器开始接管信息聚合入口

## 摘要

**AI 原生浏览器正在从「浏览工具」进化为「个人信息 Agent 平台」。**

- Dia 浏览器新功能 `coffeeonjosh`：新开 Tab 即可自动生成晨间早报，已支持 Gmail、Notion 等数据源接入
- 核心逻辑是用浏览器作为 Agent 入口，直接调用用户的个人数据流完成信息聚合
- 这与 Claude Chrome 插件、Gemini Desktop 超级入口的路径一致——浏览器正在成为 AI Agent 的默认运行时

浏览器厂商集体押注 Agent 入口，Dia 这条路走的是「个人数据聚合 + 自动摘要」，比单纯的 AI 搜索框更深一层。问题是：用户愿意把 Gmail 和 Notion 的数据权限交给一个浏览器插件吗？信任成本才是真正的瓶颈。

## 原文

如果你用Dia 浏览器，可以新开一个Tab，对话框中输入：coffeeonjosh，体验这个新功能。

会自动连接Gmail，也可以连接Notion等工具，制作一份晨间早报。
