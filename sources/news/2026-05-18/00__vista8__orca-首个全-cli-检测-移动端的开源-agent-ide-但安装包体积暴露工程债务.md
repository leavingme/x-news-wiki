# ORCA：首个全 CLI 检测 + 移动端的开源 Agent IDE，但安装包体积暴露工程债务

## 摘要

**[开源 Agent IDE 赛道又添新玩家，ORCA 的多 CLI 集成思路代表正确方向，但包体积问题暴露了 Electron 时代的技术债正在被 AI 时代继承]**

核心能力：ORCA 能检测本机已装的 Claude Code CLI、Codex CLI、Gemini CLI、Hermes、OpenClaw 等，并支持多账号切换（多人多订阅管理）。内置 Markdown 预览和文件拖拽对话。

亮点：移动端支持确实解决了一个实际痛点——远程批准危险操作的场景（ChatGPT 手机版内嵌 Codex 已验证需求）。

硬伤：安装包太大。这不是小事——AI 工具的目标用户恰恰是最在意启动速度的那批工程师，包体积大意味着信任损耗。

gakki：ORCA 踩到了点上（CLI 生态整合 + 移动端），但 Electron 的历史包袱不会因为套了个 AI 壳就消失。这波开源 Agent IDE 的竞争，最后赢的不会是功能最多的，而是最在意「工具感」的。

## 原文

又一个开源的 Agent IDE：ORCA

优势是直接提供iOS和安装移动客户端，支持多账号切换（比如多ChatGPT订阅），Token消耗、5小时重置显示。

也是检测电脑里装好的各种Cli，比如Claude Code CLI、Codex CLI、Gemini CLI、Hermes、OpenClaw等。

支持目录和文件拖拽对话，内置Markdown预览渲染。

开源地址：https://t.co/ZaiGQT91Ju
官网： https://t.co/DCYkpjWMep

解决了很多Tui工具小痒点，但缺点明显，安装包好大...
