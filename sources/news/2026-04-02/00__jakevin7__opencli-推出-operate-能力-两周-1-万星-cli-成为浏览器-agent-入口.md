# OpenCLI 推出 Operate 能力：两周 1 万星，CLI 成为浏览器 Agent 入口

## 摘要

**一句话核心判断：** OpenCLI 实现重大能力跃迁，支持 prompt 直接操作浏览器并沉淀为持久化 CLI，两周突破 1 万星和 100 名贡献者，标志 CLI 工具链正式进入「browser agent」时代。

Operate 能力打通 AI→浏览器→CLI 的闭环：prompt 直接驱动浏览器操作（填写表单、点击链接），执行结果可沉淀为可复用的 CLI 命令，AI 操作具备持久化价值。

社区增长验证工程方向：两周 1 万星 + 100 contributor 的速度表明，开发者对「浏览器自动化 + AI」的需求是真实痛点而非概念炒作。

OpenCLI 的野心是「AI Native CLI」：不只是工具，更是 AI 操作浏览器的标准接口层。

gakki：OpenCLI 正在做一件很聪明的事——不重复造轮子，而是给 AI agent 提供一个标准化的「手」。当 SKILL 体系成熟后，每个网页操作都可能成为一个可组合的 CLI 片段，复用壁垒就此建立。
