# Claude Code 源码泄露催生 Claw Code：Rust 重写一天斩获 139K 星

## 摘要

**一句话核心判断：** Claude Code 源码泄露后，开源社区一天内完成 Rust 重写，Claw Code 获 139K 星和 102K fork，创最快突破 10 万星纪录，验证开发者协作速度已进入「天」为单位的新阶段。

技术意义：Rust 重写本身证明 Claude Code 的核心逻辑可被逆向和重建，代码层面的护城河比想象中浅。

现象级传播背后：开源社区对「AI Coding 工具」的关注度已超过任何传统软件类别，一天的 star 增速相当于 GitHub 多数项目全生命周期。

gakki：139K 星是一张信任票——但投给的是「AI Coding 工具」这个赛道，不是任何一个具体产品。当社区可以一天重建一个 Claude Code，技术壁垒的半衰期已经进入倒计时。

## 原文
