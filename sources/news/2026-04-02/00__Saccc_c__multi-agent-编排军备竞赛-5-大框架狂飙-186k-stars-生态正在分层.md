# Multi-Agent 编排军备竞赛：5 大框架狂飙 186k Stars，生态正在分层

## 摘要

**[Multi-Agent 框架进入工程收敛期，生态位已初步划定]**

一份 Multi-Agent 编排框架的全景图：DeerFlow（字节，48.9k）、Agency Agents（63.5k）、Paperclip（34k）、oh-my-claudecode（12.9k）、Ruflo（27.1k）——合计超 186k Stars。关键信号：1）oh-my-claudecode 专精 Claude Code 生态，说明垂直整合开始；2）DeerFlow 支持飞书/Telegram/Slack 直接触发，说明入口已延伸；3）Ruflo 的向量记忆「记住成功模式并持续优化」——记忆层已成为标配。

**要点：**
- 186k Stars 总量代表开发者对 Multi-Agent 的关注度已从尝鲜进入工程验证阶段
- 垂直化（Claude Code 专用）和平台化（统一纳管）两条路线并行
- 记忆共享是下一个差异化战场

框架军备竞赛背后是入口和记忆的争夺，谁解决了这两个谁就是下一个基础设施。

## 原文

都2026年了，还在让你的Agent单兵作战，独自完成任务？

下面是我整理的近期爆火的开源Multi-Agent编排项目，让你的工作效率直接拉满：

1. oh-my-claudecode（12.9k⭐）
专为 Claude Code 打造的多智能体编排工具。
内置 32 个专业 Agent，支持自动拆解任务并行分发、执行直到验证完成。

2. DeerFlow（48.9k⭐）
字节出品的长任务 Agent 框架，主 Agent 动态派生多个子 Agent 并行处理复杂任务。
支持跨会话长期记忆，并能通过Telegram / Slack / 飞书 直接触发任务。

3. Paperclip（34k⭐）
AI 多智能体编排平台，把你已有的各种 AI Agent统一纳入一套规范的公司管理系统中协同运作。

4. Ruflo（27.1k⭐）
将 Claude Code 升级为多智能体群体开发平台，Agent之间自动分工、共享上下文、互相通信。内置的向量记忆会记住成功模式并持续优化任务。

5. Agency Agents（63.5k⭐）
144 个精心设计的 AI 专家 Agent 合集，覆盖工程、设计、销售等 12 个部门，由Agents Orchestrator 负责多 Agent 协调调度，高效完成任务。
