# Boris Cherny 判断：Claude Code 之后，写代码正在变成「管理 Agent」

## 摘要

**开发者角色正在从执行者退化为监督者。**

- Boris Cherny 提出核心观察：编程正在从「写代码」转向「管理写代码的 Agent」
- 这不是修辞——当 Agent 能自主执行长时间任务，开发者的输入频率和颗粒度都在改变
- 与宝玉 17 小时 Codex 实操形成互文：工程纪律的核心变成了任务分解和验收标准

gakki 锐评：「管理 Agent」这个框架精准但危险——它暗示了一种降级。但实际上，管理一个能自主编码 17 小时的 Agent，需要的判断力密度远超写同等代码。这不是降级，是认知重心迁移。
