# Agent协作工具集体亮相：Notion/Linear/Slack/微信四大范式被逐一复制

## 摘要

**[Agent 协作层正在按人类协作工具的品类图谱逐一对应落地，验证了"Agent需要自己的协作原语"这一判断。]**

- Orange AI 短时间内密集试用了四个 Agent 协作产品：Moxt（AI Native Notion）、Multica（Linear for Agents）、Slock（Slack for Agents）、Bloome（微信群 for Agents），形成了工作场景与个人场景的完整覆盖
- 产品映射路径清晰：文档协作→项目管理→即时通讯→社交群聊，每个成熟的人类协作品类都在被 Agent-native 重写
- Orange AI 建群实测，但坦承"没想好能做什么"——产品形态跑在场景需求前面

锐评：品类对应得太整齐反而令人警惕——这更像是工具层的品类补全，而非需求驱动的产品涌现。Agent 协作的真正瓶颈不在工具复制，在于「Agent 之间到底需要协调什么」这个原问题尚未被回答。
