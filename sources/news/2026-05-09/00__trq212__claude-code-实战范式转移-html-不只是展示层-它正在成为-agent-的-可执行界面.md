# Claude Code 实战范式转移：HTML 不只是展示层，它正在成为 Agent 的「可执行界面」

## 摘要

**HTML 在 Agent 工作流中的角色正在从「输出格式」变成「交互协议」。**

- Thariq 发现 Claude Code 中 HTML 的「不合理有效性」——它既是人类可读的界面，又能被 Agent 结构化解析，兼具可执行与可展示
- 宝玉随即翻译了同一篇文章，说明中文开发者圈也在快速跟进这个信号
- 这不是格式之争的重复，而是揭示了一个更深的趋势：Agent 工具链正在收敛到「一套格式同时服务人和机器」

HTML 作为 Agent 界面的胜出不是因为它「最好」，而是因为它同时满足了 two audiences 的最低需求——这恰恰是工程收敛的典型路径。
