# 玉伯一句话厘清 Agent-SaaS-Software 的包含关系：别再把 Agent 当新物种了

## 摘要

**Agent 不是 SaaS 的替代品，而是 SaaS 的子集——这个认知框架值得反复校准。**

- 明确层级：Agent ⊂ SaaS ⊂ Software，Agent 是 SaaS 的一种特殊形态
- 暗示当前"Agent 替代 SaaS"的叙事存在概念混淆，行业需要更精确的分类

gakki 锐评：看起来是咬文嚼字，实际是对当下 Agent 叙事泡沫的冷静校正。把 Agent 定位为 SaaS 子集，意味着评判标准应该沿用 SaaS 的商业逻辑（留存、ARPU、运行时归属），而不是凭空造一套新指标。

## 原文

@yangyi 可以搜索下 SaaS 的定义。以及 Software 的定义。Agent 是 SaaS 的子集，SaaS 是 Software 的子集。
