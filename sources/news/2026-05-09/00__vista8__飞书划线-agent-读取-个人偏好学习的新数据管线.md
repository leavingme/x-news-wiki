# 飞书划线 + Agent 读取 = 个人偏好学习的新数据管线

## 摘要

**阅读行为的结构化捕获，正在成为 Agent 个性化的新范式。**
- 向阳乔木提出一个可操作的思路：将日常阅读内容放入飞书，通过划线/评论/点评产生结构化标注
- 然后让 Agent 读取这些标记，反向推断用户偏好，用于优化 Skill
- 这本质上是“用户行为即训练数据”的轻量化实现——不需要 fine-tune，只需要结构化的表达

gakki 锐评：这个思路的价值在于它绕开了传统推荐系统的冷启动问题——用户的划线本身就是高质量的偏好信号。问题是飞书的数据 API 开放度和 Agent 的读取权限能否形成闭环。

## 原文

突然想到个思路，应该把自己经常读的内容放到飞书。

然后划线评论、点评。

然后就可以用Agent 读取这些标记，了解你的偏好，优化Skill。 https://t.co/FQ0IWkr5D5
