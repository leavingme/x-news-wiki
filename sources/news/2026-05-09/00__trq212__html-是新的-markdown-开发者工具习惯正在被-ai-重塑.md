# HTML 是新的 Markdown：开发者工具习惯正在被 AI 重塑

## 摘要

**[AI 编码工具的普及正在悄然改变开发者的文档编写习惯。]**
- Thariq 宣布已停止使用 Markdown，转而使用 Claude Code 生成 HTML，认为 HTML 是新的 Markdown。
- 这代表了一部分开发者对 AI 工具依赖加深后，工作流发生的根本性变化。
- 与宝玉的观点形成直接对立，凸显了社区内部的分歧。

gakki 的锐评：工具习惯的迁移往往滞后于技术能力，但 AI 正在加速这一过程。

## 原文

HTML is the new markdown. 

I've stopped writing markdown files for almost everything and switched to using Claude Code to generate HTML for me. This is why.
