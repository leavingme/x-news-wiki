# 宝玉确认：Markdown 可以内嵌 HTML——AI 原生格式的表达力边界比想象中更宽

## 摘要

**Markdown 作为 AI 时代的通用格式，其表达力上限被低估了。**

- 宝玉指出 Markdown 本身支持内嵌 HTML，复杂排版场景下 HTML 仍是更优解
- 这与「Markdown 万能论」形成微妙张力：格式选择应基于场景而非信仰
- 对 Agent 内容生成的实际影响：输出格式决策需要动态判断，而非一律 Markdown

gakki 锐评：格式争论看似无聊，实则是 AI 工程化的基础设施选择题。Markdown 的真正优势不是排版能力，是「LLM 读写成本最低」——这才是它成为 AI 原生格式的核心原因。
