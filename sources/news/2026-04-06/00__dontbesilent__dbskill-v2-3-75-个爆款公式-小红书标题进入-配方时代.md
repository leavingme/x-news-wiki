# dbskill v2.3：75 个爆款公式，小红书标题进入「配方时代」

## 摘要

**【内容创作套利正在工具化：Skill 市集开始长出护城河】**

dbskill 更新至 v2.3，上线小红书标题公式工具。核心创新：每个标题可追溯到具体公式编号，并附上该公式真实爆款案例，而非 AI 随机生成。

机制亮点：75 个公式 × 12 种心理触发器分类，单话题覆盖至少 3 种类型，输出 Top 3 推荐。与 content 诊断、hook 优化形成完整链路。

行业意义：Skill 市集开始出现「可验证的复用价值」而非「一次性 prompt 集合」，这是技能商品化的重要一步。

---
*gakki：真正有价值的不只是公式本身，而是「可追溯」这件事——它把隐性经验变成了可验证的资产。dbskill 这个方向走对了，但护城河在于持续验证和案例积累。copy 容易，但持续运营 75+ 真实案例的数据库才是壁垒。*

## 原文

dbskill 更新 V2.3：小红书标题不用自己想了

上一版更新了奥派经济聊天室，这一版加了个实用的——小红书标题公式工具。

用法很简单：给它一个话题，它从 75 个验证过的爆款公式里帮你挑最合适的，直接生成定制标题。

不是那种 AI 随便编的标题。每个标题都能追溯到具体的公式编号，旁边还附上这个公式原来在小红书上火过的真实案例，你可以自己判断靠不靠谱。

75 个公式按 12 种心理触发器分类——认知冲突、好奇缺口、恐惧损失、身份代入、数字锚定、结果承诺、社会证明、争议挑衅、场景条件、行动号召、权威借力、互动测试。你给一个话题，它至少覆盖 3 种不同类型，最后给你 Top 3 推荐。

和之前的 skill 是打通的。content 诊断完内容，可以直接推到 xhs-title 起标题；标题选好了，再推到 hook 优化开头。一条龙。

GitHub 项目地址：https://t.co/RCcIOduyUV

更新方式：

npx skills add dontbesilent2025/dbskill

或者在 Claude Code 里直接输入 /dbskill-upgrade。

装完之后输入 /dbs-xhs-title 或者 /小红书标题，给个话题试试。
