# Google AI Studio 吞进 Pro/Ultra 套餐：订阅即入口，API 末日已至

## 摘要

**[ AI Studio 不用单独 API key 了，订阅制把开发者工作流彻底锁进 Google 生态 ]**

Google 宣布 AI Studio 并入 Gemini Pro/Ultra 订阅体系——买了会员就能直接用 AI Studio，且额度比单独 API key 更高。Nano Banana 画图和开发小应用也在套餐内。

这意味着 API key 作为独立付费入口的价值被进一步侵蚀。Google 的逻辑很清楚：与其让开发者在按量计费和订阅之间跳来跳去，不如用订阅套餐一口价全包，降低决策摩擦，把用户锁进 Google 的使用习惯里。

对比 GitHub Copilot 同期收紧，Google 选择了相反的扩张策略——用更大的额度包吸引用户。

gakki：Google 和 GitHub 同期做了截然相反的动作，一个扩张额度、一个收紧入门。Google 的阳谋是「用订阅额度换 API key 的消失」——当会员额度足够大，谁还去管 API key 的细碎账单？平台整合的大幕已经拉开。

## 原文

Google 宣布 Google AI Studio

加入Google AI Pro / Ultra 订阅套餐

以前AI Studio 免费额度用完后，你需要开 API key、绑信用卡、走按量计费。

你只要订了 Gemini 的会员，不用再单独搞 API，直接登录 AI Studio 就能用而且额度还更高...
