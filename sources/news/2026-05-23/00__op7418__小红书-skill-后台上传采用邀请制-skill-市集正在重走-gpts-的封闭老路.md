# 小红书 Skill 后台上传采用邀请制：Skill 市集正在重走 GPTs 的封闭老路

## 摘要

**一句话核心判断：小红书悄然将 Skill 上传设为邀请制，意味 Skill 分发生态在中文平台再次倒向封闭围墙。**

要点：
- 小红书后台 Skill 上传通道目前仅限受邀用户，外部创作者无法自主上架
- 邀请制本质是平台早期筛选内容质量的惯用手段，但也意味着分发路径被平台牢牢掌控
- 对比 GitHub/MCP 的开放生态，封闭邀请制会压制第三方 Skill 创作者的积极性

 Skill 作为可组合工具单元，其价值取决于分发自由度。邀请制若长期延续，中文 Skill 生态将难以形成规模效应。
