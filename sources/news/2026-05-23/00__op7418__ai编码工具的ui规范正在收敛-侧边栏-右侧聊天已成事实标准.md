# AI编码工具的UI规范正在收敛：侧边栏+右侧聊天已成事实标准

## 摘要

**[AI编码工具的UI范式已收敛至同一张图纸]**

当「透明侧边栏+聊天面板」成为 Copilot、Manus、Codex 等产品的共同选择，这不是偶然——而是 macOS 人机规范在 Agent 软件层的强制显化。遵循规范的产品快速获得用户熟悉度红利，差异化的空间被压缩至图标排列层面。

更深的问题在于：这种收敛是创新的终局，还是新一轮同质化的起点？当交互层无法差异化，竞争必然向 Agent 能力层转移。

## 原文

左边侧边栏加右侧聊天几乎都已经是这类软件的设计规范了，不是常用的是啥软件就是抄啥。

Codepilot 之前的布局也是类似这样，透明的侧边栏的 Mac 的规范，遵循规范就这样

左边最顶上的那几个确实跟 Codex 像，但我感觉 Manus 之前就这样了
