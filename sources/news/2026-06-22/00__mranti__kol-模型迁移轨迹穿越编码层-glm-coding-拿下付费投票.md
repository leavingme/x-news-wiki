# KOL 模型迁移轨迹穿越编码层：GLM Coding 拿下付费投票

## 摘要

**Michael Anti 公开模型偏好进化路径，国产编码工具正式挤入 KOL 订阅清单。**

- 完整迁移链：Kimi → Qwen → DeepSeek → Kimi Coding → DeepSeek V4 → GLM Coding（今日购入 Pro 套餐）
- 美国御三家（GPT/Claude/Gemini）仍作为底座并行使用
- 信号一：编码工具已成为独立的订阅决策层，不再是"基座模型的附赠"或"试用替代"
- 信号二：国产编码产品跨过 PMF 验证门槛——意见领袖愿意自掏腰包


hakki 锐评：闲鱼代抢算稀缺度，个人订阅算 PMF——KOL 的钱包投票比榜单和倒卖数据都诚实。编码工具的竞争正在从"谁更像 Claude Code"分化到"谁的付费留存更稳"。

## 原文

我对中国模型的偏爱转移是有这个过程的：Kimi LLM-&gt; Qwen LLM -&gt; Deepseek LLM&gt; Kimi coding -&gt; Deepseek V4 LLM+Coding -&gt; GLM coding（今天刚买了Pro套餐）。当然美国御三家还是陆续在用。
