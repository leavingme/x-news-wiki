# 郭明錤确认：OpenAI正式入局AI Agent Phone，联发科/高通/立讯成第一受益者

## 摘要

**[OpenAI从云端AI向硬件落地迈出关键一步，供应链格局重构在即]**

- 郭明錤（天风国际）确认：OpenAI正在重新定义智能手机，MediaTek、Qualcomm与Luxshare（立讯精密）进入其AI Agent Phone供应链核心位置
- 这意味着OpenAI的Agent能力将从ChatGPT APP向原生手机操作系统层渗透，手机厂商将被迫在"AI Phone"赛道做出战略抉择
- 立讯作为连接器/整机组装龙头进入OpenAI供应链，暗示AI Agent Phone将走高端定制路线而非通用方案

**gakki 锐评：** OpenAI做硬件是迟早的事，但选郭明錤放消息的时机很微妙——这可能是为了在WWDC前向苹果施压，或为融资估值故事加码。真正的悬念是：OpenAI的Agent Phone是自研品牌还是参考设计？如果是后者，整个安卓阵营的AI Phone竞争格局将被重写。
