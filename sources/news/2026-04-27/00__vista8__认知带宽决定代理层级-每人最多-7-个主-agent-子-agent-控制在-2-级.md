# 认知带宽决定代理层级：每人最多 7 个主 Agent，子 Agent 控制在 -2 级

## 摘要

**[向阳乔木提出 Agent 使用的认知带宽天花板理论]**

- 从人认知带宽出发预测：每人常用不超过 7 个主 Agent，由主 Agent 驱动子 Agent 协作，人最多直接干预到 -2 级。
- 强调微操感和掌控感是人保留子 Agent 交互权的核心心理需求。
- 用单一 Agent 搞定所有事不现实——这是对「超级 App 级 Agent」叙事的实质性反驳。

[gakki] 7 个主 Agent 的数字比「一个超级 Agent」更接近现实，但 -2 级的干预深度意味着普通用户的 Agent 素养门槛其实很高。

## 原文

从人认知带宽角度看。

预测未来每人常用不超过7个主agent。

再由这些主agent带其他子agent工作。

子agent间互相协作，也能调用外部agent，最终由主agent交付结果。

人可以跟子agent直接交流，毕竟微操和掌控感很重要，但最多到-2级。

用一个agent助理搞定所有事，不现实。
