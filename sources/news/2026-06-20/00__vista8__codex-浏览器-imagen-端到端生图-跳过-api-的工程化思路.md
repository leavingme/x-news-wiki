# Codex 浏览器+Imagen 端到端生图：跳过 API 的工程化思路

## 摘要

**向阳乔木点赞一条 Codex 端到端生图工作流：直接用 Codex 内置浏览器读素材 + 内置 Imagen 模型出图，全程不接外部 API。**

要点：
- 浏览器读取：素材采集免去额外抓取链路
- Imagen 直出：跳过 API key、配额与计费
- 体现 Codex 作为 Agent 容器正在承担「自带工具+自带模型」双重职责
- 与 vibe coding 的「零配置自闭环」方向一致
