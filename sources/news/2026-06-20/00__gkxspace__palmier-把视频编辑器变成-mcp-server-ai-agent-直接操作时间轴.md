# Palmier 把视频编辑器变成 MCP Server：AI Agent 直接操作时间轴

## 摘要

Palmier 开源 Mac 原生视频编辑器，核心架构是把整个编辑器做成 MCP server——Claude、Cursor、Codex 可直接操作时间轴。

配套能力全开源：多轨时间轴、on-device 语义搜索素材、本地语音转文字；AI 生成模型部分按需付费。

这是"编辑器即工具"范式的样板：未来所有 GUI 软件都可能 MCP 化，让 Agent 像调用函数一样调用 Photoshop、Premiere、Figma。

对比 MCP vs CLI 之争：Palmier 选了 MCP 上层协议而非 CLI 封装，更适合 LLM 的语义理解和长任务编排。

gakki 锐评：被誉为"视频编辑的 Cursor 时刻"不夸张，但 MCP server 真正爆发需要协议标准化和分发渠道——单个编辑器开源不够，要等 Anthropic / OpenAI 把 MCP 内置成系统级能力，才会出现"甩链接 + 视频剪辑"的杀手场景。

## 原文

视频编辑的 Cursor 时代要来了？？？

Palmier 开源的 Mac 原生视频编辑器，核心思路：
整个编辑器就是一个 MCP server，Claude、Cursor、Codex 可以直接操作你的时间轴

把编辑器变成 AI Agent 的工具，比如你跟 Claude 说"把第二段视频剪到 3 秒，加个字幕"，它直接在时间轴上操作

包含（完全开源）：
1、完整的多轨时间轴编辑器
2、on-device 语义搜索，用自然语言找素材
3、本地语音转文字，不用联网
4、MCP server 完全免费，随便接

AI 生成部分（Seedance、Kling、NB Pro 这些模型）需要付费，但编辑器本身一分钱不收。

看评论区有人说"Cursor moment for video editing"，我觉得这个形容挺准的。

目前只支持 macOS，特效转场调色都还没有，离替代 Premiere 还远。
但这个 MCP 架构设计比功能本身有远见多了，等生态跑起来估计会很强大。
