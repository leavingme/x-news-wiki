# yetone 一句话点破 Raft 取巧位：不碰 Agent 本体，专做周边

## 摘要

yetone 指出 Raft 做得好的关键是它'不做 Agent 本身'。

- Agent 生态的分工正在细化：有人造 Agent 引擎，有人造 Agent 配套
- Raft 选择做 Agent 周边的策略，本质是规避与 Claude Code / Codex / Hermes 的正面竞争
- 这条路径对独立开发者有借鉴意义——Agent 时代不是只有造 Agent 一种活法


gakki 锐评：'不碰 Agent 本体'是一种清醒的定位，但也是天花板的天花板——周边生意的护城河永远比不过引擎层，需要随时警惕上游把你吸收掉。
