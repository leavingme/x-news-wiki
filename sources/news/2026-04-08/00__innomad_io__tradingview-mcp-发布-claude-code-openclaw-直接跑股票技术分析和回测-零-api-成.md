# TradingView MCP 发布：Claude Code / OpenClaw 直接跑股票技术分析和回测，零 API 成本

## 摘要

**【一句话核心判断】**
开源社区把 TradingView 数据接入 MCP 协议，Claude Code 直接做股票技术分析、跑回测、情绪扫描，且不需要任何付费 API。

**信息增量：**
- Yahoo Finance 数据直连，绕过付费墙
- 技术分析、回测、情绪面、市场扫描已全部实现
- MCP 协议让 OpenClaw/Claude Code 的金融 Agent 生态开箱即用

**影响范围：**
个人投资者、加密货币交易者、用 OpenClaw 构建交易 Agent 的开发者。

**gakki 锐评：**
MCP 协议正在变成 Agent 世界的「USB 接口」——一个协议打穿所有数据源。这个扩散速度比任何闭源生态都快。
