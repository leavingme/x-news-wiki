# Michael Anti 质疑国内外 Agent 生态割裂：模型接入能力比身份标签更重要

## 摘要

**[ Michael Anti 点出一个被忽视的结构性问题：AI Harness 没有国内外之分]**

- 他的核心论点：开源 AI Harness 已普及，国内 Agent 的差异化不在于"属地"而在于"接了什么模型"
- 这个视角与当前"国产 Agent 叙事"形成张力——当工具链本身已同质化，护城河在哪里？
- 隐含的批判：国内 Agent 生态过度强调场景包装，而非底层模型能力与工作流深度

这是对 Agent 创业方向的一个冷静质疑：差异化究竟在模型层、工具链层，还是场景层？
