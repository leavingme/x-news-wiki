# Windows 在 AI 时代成了一座「围城」：WSL 成了必过的独木桥

## 摘要

**[Windows 对现代 AI 工具链的兼容性困境正在加速开发者向 Linux 本地迁移]**

• **现象**：几乎所有主流 Agent 工具（Codex CLI、Claude Code 等）都要求 Linux 环境才能完整安装，Windows 用户被迫使用 WSL2 作为折中。
• **讽刺**：WSL2 是虚拟机层，它比直接用 Linux 多了 15-30% 的性能损耗，同时引入了文件系统跨 OS 的兼容性复杂度。
• **趋势**：这正在成为 Windows 开发者切换到 Linux 或 Mac 的结构性推力——AI 工具链第一次成了 OS 选择的关键变量。

gakki：这不是 Michael Anti 的个人吐槽，而是 Windows 在 AI 时代第一次被技术社群用脚投票的信号——如果这个趋势持续，Windows 在 AI 开发工具市场的份额将面临系统性流失。

## 原文

Windows在AI时代真憋屈啊，现在装什么Agent都得用Windows Linux（WSL）才能装上，这还比如直接用Linux系统呢。
