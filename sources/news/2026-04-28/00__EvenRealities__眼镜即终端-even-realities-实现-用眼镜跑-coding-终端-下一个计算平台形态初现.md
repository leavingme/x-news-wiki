# 眼镜即终端：Even Realities 实现「用眼镜跑 Coding 终端」，下一个计算平台形态初现

## 摘要

**[一句话核心判断]**

Even Realities 发布的 Terminal Mode 让用户可以直接用眼镜设备运行编码终端，这是可穿戴设备首次在「生产力工具」层面实现场景落地，而非健身或通知类轻应用。

**后续要点：**
- 技术突破：眼镜设备原生支持命令行操作，跳过了手机/电脑的中转，真正实现「随身工作空间」
- 场景验证：coding terminal 是一个极端考验输入效率和显示密度的场景，选择这个切入说明硬件已达可用阈值
- 平台信号：可穿戴设备正在从「感知配件」进化为「执行终端」

**gakki 锐评：**
这个产品的战略意图比技术本身更有趣——眼镜作为「随身穿戴的智能终端」，与 AI Agent 的结合意味着「空间即工作区」可能不再是概念。但核心问题是：续航、散热和隐私。coding 任务重时眼镜的电量能撑多久？长时间佩戴发热用户体验如何？这些细节决定了这是玩具还是工具。
