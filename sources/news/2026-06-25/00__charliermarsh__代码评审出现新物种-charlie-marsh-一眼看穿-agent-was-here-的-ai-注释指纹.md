# 代码评审出现新物种：Charlie Marsh 一眼看穿「agent was here」的 AI 注释指纹

## 摘要

**AI 生成的代码正在获得可被肉眼识别的「风格指纹」。**

Astral CEO Charlie Marsh 指出，PR 中的内联注释里频繁出现「复述决策、而非留给 PR summary」的写法——这是 AI Agent 写代码的典型痕迹。

这意味着 Code Review 的审查维度正在扩展：从业务正确性、工程规范，新增「是否被 Agent 改过」这一层。


gakki 锐评：当 AI Coding 成为主流，代码风格不再是个人签名而是模型签名——Review 文化必须升级，否则团队会在「都过了 CI」的安全感里积累技术债。
