# Karpathy 站台 FDM1：VPT 范式从游戏扩展到「知识工作 / 电脑使用」

## 摘要

**Karpathy 罕见公开背书 SI 团队将 VPT（Video PreTraining）范式扩展到 FDM1，明确指向「知识工作 / 电脑使用」场景。**

- VPT 当年靠 Minecraft 视频无监督预训练实现了具身智能突破，现在被映射到「人操作电脑的视频」上——预训练数据从游戏扩展到真实 GUI 交互。
- FDM1 如果是「Foundation Desktop Model」类命名，意味着通用电脑操作 Agent 正在走向基础模型化，而非小作坊式微调。
- Karpathy 的兴奋度本身就是信号：他亲自参与/关注的赛道，往往是 12-18 个月内的主战场（如 GPT、nanoGPT、llm.c）。
