# 钉钉CLI宣布开源

## 摘要

一行命令调用钉钉全部产品能力，开放AI表格、日历、日志、待办等

## 原文

卧槽，钉钉宣布其 CLI 直接开源

一行命令调用钉钉全部产品能力

首批开放 AI 表格、日历、日志、待办、机器人、通讯录、DING 消息、考勤、开放平台文档、工作台共 10 项核心产品能力...

原生支持 Claude Code、小龙虾 等主流 AI Agent 执行环境。

三个关键设计：

① 所有输出都是结构化 JSON：AI Agent 不需要解析自然语言返回值，直接拿到结构化数据就能用。

② 内置 Agent Skills： 安装脚本会自动把每个产品的 SKILL.md 文件安装到 ~/.agents/skills/dws 目录下。Claude Code、小龙虾等会自动发现这个目录，开箱即用。

③ 支持 dry-run：加 --dry-run 可以预览操作但不执行，Agent 在执行敏感操作前可以先确认。

应该是国内首个面向 AI Agent 开源全产品能力的国民级应用...🫡
