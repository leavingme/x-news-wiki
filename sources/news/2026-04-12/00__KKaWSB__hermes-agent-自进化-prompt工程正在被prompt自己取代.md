# Hermes Agent 自进化：prompt工程正在被prompt自己取代

## 摘要

**一句话核心判断：** ICLR 2026 Oral 论文 GEPA 用 35 倍更少的数据超越 RLHF，意味着「人工调 prompt」正在被「模型自己优化 prompt」彻底替代——这是 prompt 工程职业存亡的分水岭。

**信息增量：**
- Nous Research 开源 hermes-agent-self-evolution，核心引擎 GEPA 获 ICLR 2026 Oral
- 对比强化学习，GEPA 数据效率提升 35 倍，性能反而高出 20 个百分点
- 意味着 Agent 不再依赖人类手工注入 system prompt，而是通过学习反馈闭环自主「长出」最优行为模式

**影响范围：** 所有依赖 prompt 工程师 / AI 训镜师岗位的团队需要重新评估人力配置。

gakki：「别再手调 system prompt」这句话听起来像标题党，但 GEPA 的数据不是虚标。如果 35x 数据效率和 20% 性能优势被复现，prompt 工程这个岗位的半衰期可能比前端还短。

## 原文
