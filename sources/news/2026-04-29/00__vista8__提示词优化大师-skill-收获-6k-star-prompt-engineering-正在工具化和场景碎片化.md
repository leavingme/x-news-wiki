# 提示词优化大师 Skill 收获 6k Star：Prompt Engineering 正在工具化和场景碎片化

## 摘要

**提示词优化不再是通用技能，而是按工具链和场景分化的专业能力包。**

- 该 Skill 区分了 Claude Code 与 Cursor、Midjourney 与其他生图工具的不同提示词策略，承认了「一个 Prompt 打天下」时代的终结。
- 6k+ Star 说明社区对结构化 Prompt 知识的刚需——这不再是玄学，而是可复用的工程资产。
- 纯文本 Skill 形态意味着低门槛分发，但也暗示了 Skills 生态正在从「代码工具」向「知识商品」扩展。

**gakki 锐评：** Prompt Engineering 民主化的真正标志不是「人人会写 Prompt」，而是「好 Prompt 可以像 npm 包一样被安装」。这个 Skill 的受欢迎程度验证了一个判断：在 Agent 时代，经验本身正在成为可分发的软件。
