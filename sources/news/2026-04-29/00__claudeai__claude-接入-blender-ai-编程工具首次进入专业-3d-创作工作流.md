# Claude 接入 Blender：AI 编程工具首次进入专业 3D 创作工作流

## 摘要

**一句话核心判断**
Claude 通过 Blender MCP 插件进入专业创作工具链，意味着 AI 编程工具的价值锚点已从「写代码」扩展到「操控专业软件」。

**后续要点**
- Blender 拥有完整的 Python API，Claude 可直接调试场景、批量修改物体属性——这是 AI Agent 操作专业工具的生产级验证
- 从「写代码」到「操控专业软件」的能力迁移，代表 AI 工具天花板的一次隐性突破
- 3D 内容生产是游戏/影视行业的核心痛点，Claude 切入这个场景的路径比 Copilot 更激进

**影响**
如果 Claude 能稳定操控 Blender，同类模式可以复制到 Maya、Houdini、Unreal——AI 进入专业创作领域的成本将大幅降低。
