# 阿里系终端双雄对决：Qwen Code 与 iFlow CLI 深度测评

## 摘要

**[国产工具选型]**
- iFlow：侧重多代理协作、任务自动拆解，与 IDE 深度集成（支持 YOLO 模式），个人永久免费。
- Qwen Code：基于 Gemini CLI 优化的开源方案，专攻代码库理解，支持无头模式（CI/CD）和终端截图自动化，提供高额度免费 API。

[gakki 锐评]：巨头内部赛马的最大受益者永远是开发者。想要开箱即用的闭环选 iFlow，想要魔改 CI 的选 Qwen Code。

## 原文

Qwen Code 和 iFlow CLI 都是阿里系终端 AI 工具，用自然语言搞代码/自动化任务，核心很像，但区别挺明显：

**开发者与定位** 
- iFlow：心流团队出品，更通用、多代理协作（任务自动拆解）。 
- Qwen Code：QwenLM 团队出品，开源（Apache 2.0），基于 Gemini CLI 优化，专攻代码库理解和开发流。

**免费程度** 
- iFlow：个人永久免费不限量（GLM/Qwen/Kimi 等全开）。 
- Qwen Code：OAuth 每天 1000 次免费（额度很大），超了可付费或切 API。

**特色** 
- iFlow：YOLO 快模式、多 IDE 深度集成（含 Jira/Jenkins）、https://t.co/l1O2dK9EWC 自定义。 
- Qwen Code：无头模式（CI/CD 脚本用）、终端截图自动化、文件引用更灵活。

想丝滑通用选 iFlow，想开源可改选 Qwen Code。都试试，官网/ GitHub 装超简单！
