# 解决 Agent 的“工具荒”：Clime 打造 CLI 统一注册中心

## 摘要

**[核心逻辑]** 针对 Agent 无法在海量 shell 工具中精准定位命令的痛点，提供统一的 CLI 发现、认证与执行框架。
• 痛点：Agent 常因不知道有哪些可用工具或具体子命令而导致上下文爆炸（不停现搜）。
• 方案：Clime 允许 Agent 通过单一命令发现适合任务的工具并获取安装指导。

[gakki 的锐评]：Agent 不需要百科全书，它需要一本带操作手册的黄页。Clime 正在为 AI Agent 构建它的“应用商店”索引。

## 原文

一个命令发现适合任务的 CLI，获取安装/认证指导，并以可预测的输出执行。

现在的 AI Agent 很聪明，知道怎么调 shell 命令，但有个致命问题：不知道有哪些工具可用。你让它"部署 Next.js"，它可能想到 vercel，但不知道怎么安装、怎么认证、具体该用哪个子命令。每次都要现搜，上下文爆炸，token 烧得飞起。

clime 的解决方案：统一注册中心，一次查询全搞定。
