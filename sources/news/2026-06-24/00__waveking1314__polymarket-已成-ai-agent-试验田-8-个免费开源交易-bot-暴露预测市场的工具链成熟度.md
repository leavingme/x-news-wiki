# Polymarket 已成 AI Agent 试验田：8 个免费开源交易 Bot 暴露预测市场的工具链成熟度

## 摘要

**预测市场的 Agent 化已经不是概念，而是工具链溢出阶段。**

- 8 个 Polymarket/Kalshi 开源 Bot 覆盖延迟套利���流动性做市、天气修正、DCA 等多种策略，Cambridge 黑客松项目 CloddsBot 单仓库即含 118+ 策略模板
- 天气模型用机器学习修正多源预报误差、再做预测，思路与量化对冲基金校准信号源同源
- 这一批 Bot 的真正意义不在策略本身，而在「预测市场 API + 链上结算」已经足够标准化，让散户也能 30 分钟跑起来一个会自己下单的 Agent

## 原文

兄弟们，Polymarket 现在最离谱的地方是：

很多人还在手动盯盘口，GitHub 上已经有人把各种交易 bot 全开源了。

整理 8 个免费的 Polymarket / 预测市场 bot，覆盖不同玩法，基本都有英文安装教程和使用说明。

1. CloddsBot
118+ 个预测市场策略和工具包，包括 Binance-Polymarket 延迟、Momentum、Penny Clipper、Smart Routing、DCA、Expiry Fade 等
据说是剑桥计算机学生做的 hackathon 项目
https://t.co/iryy3WUate

2. polymarket_lp_tool
自动管理 Polymarket 限价单，主要用来最大化流动性奖励
https://t.co/X5lq0h5374

3. Kalshi Weather Model
机器学习天气模型，不是盲信预报，而是学习不同天气源过去在不同城市的误差，再修正新预测
https://t.co/WS3ScC78jg

4. Polymarket-Kalshi Arbitrage Bot
自动搜索 Polymarket 和 Kalshi 之间的套利机会
https://t.co/4YRs9Ugt2v

5. Polymarket-bot
Smart Money 策略，筛选高 PnL、高胜率、表现稳定的钱包，再生成自动跟单列表
https://t.co/J7Bo6ZX8Ae

6. Prediction Markets Trading Bot Toolkits
工具箱级别项目，包含跟单、套利、鲸鱼提醒、做市、spread farming、体育交易等
https://t.co/vJkflXuyQu

7. PolyWeather
中国开发者做的天气 bot，实时分析天气预报、机场数据、航空观测 METAR，生成指定城市和日期的天气报告
https://t.co/73mdNbjSfq

8. Awesome Prediction Market Tools
30+ 个预测市场免费交易 bot 和服务合集
https://t.co/Ra0p8TUxaZ

很多项目都有 Dry Run 模式，可以先在真实市场测试逻辑，不用一上来拿真钱硬冲。

一句话：

Polymarket 越来越不像纯预测平台，反而越来越像一个被开发者、量化脚本和 AI Agent 攻占的交易所。

别无脑实盘，先看代码，先跑模拟，先理解策略。

如果不想研究，可以直接用Polymarket上最快的跟单bot跟单聪明钱
