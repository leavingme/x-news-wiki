# yetone 一句话点破"AI Coding 怀旧"元命题：制品没变，所以才有比较

## 摘要

**Vibe Coding 焦虑的根源不是模型能力，是自我价值感的锚点被复制。**

- 我们不怀念打孔纸→因为制品从打孔纸变成了代码，介质变了
- 我们怀念手写代码→因为 AI Coding 制品还是源代码，介质没变
- 制品同质 = 比较自然发生 = "AI 写的就不如我"的情感来源

锐评：这是 Vibe Coding 争议的底层心法。承认这层心理机制（自我价值感锚点被复制），才能不把工程问题搞成身份战争。判断力稀缺是真的，但表达上不必非此即彼。

## 原文

@ccs668899 你不去回想它们是因为现在的制品是代码，以前的制品是打孔纸。你会去怀念手写代码，是因为现在AI Coding的制品是源代码，以前的制品也是源代码。都是一样的东西，所以自然而然有了比较。
