# Claude Code 官方推出 /goal 命令：结构化任务管理成为 Agent 原生能力

## 摘要

**Anthropic 将结构化目标管理内置为 Claude Code 的一等公民。**

- 官方正式推出 /goal 命令，不再需要第三方插件实现目标分解与追踪
- 这意味着 Agent 的任务管理能力从「社区 hack」升级为「平台原生」
- 对依赖 SKILL.md 等外部工作流的用户来说，平台正在收编最佳实践

Gakki 锐评：当平台开始把社区的最佳实践内化为官方功能时，要么是生态成熟了，要么是平台在重新划定边界。对 Agent 工程师来说，这意味着「平台会替你做决策」的趋势在加速——你的 Skill 能活多久，取决于平台的迭代速度。
