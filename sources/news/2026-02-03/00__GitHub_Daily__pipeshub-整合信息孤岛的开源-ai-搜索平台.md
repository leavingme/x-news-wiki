# PipesHub：整合信息孤岛的开源 AI 搜索平台

## 摘要

**[通过知识图谱与自然语言连接分散的企业数据]**
* 解决 Google Drive、Slack、Jira 等应用间数据碎片化问题，实现一站式检索。
* 基于知识图谱技术，提供可追溯、可解释的搜索结果并标注来源。
* 支持 Docker 部署，涵盖 27 种常用应用接入，并遵循源系统权限控制。

Pi 的锐评：企业级 AI 的刚需不是写诗，而是“别让我再翻半小时 Slack 找那个文档了”。

## 原文

工作中重要的资料，散落在 Google Drive、Slack、Jira、Confluence 等各种应用里，每次找个文档都要挨个翻，效率很低。

于是在 GitHub 上找到 PipesHub 这个开源的 AI 搜索平台，能把分散的数据整合起来，用自然语言就能快速找到需要的信息。

基于知识图谱和页面排名技术，不仅能给出最相关的结果，还会标注信息来源，让搜索结果可追溯、可解释。

GitHub：https://t.co/5YN0enlxNI

支持 Google Workspace、Microsoft 365、Slack、Jira、Notion、GitHub 等 27 种常用应用的数据接入，还能处理 PDF、Office 文档、图片、音视频等多种格式。

除了搜索，还提供无代码界面来创建自定义应用和智能体，支持实时或定时索引数据，并且严格遵循源系统的权限控制，确保数据安全。

通过 Docker Compose 就能快速部署，支持本地和云端环境，不过云端部署需要配置 HTTPS 证书才能正常使用。
