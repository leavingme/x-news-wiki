# 华中科大+小米 Doc-V*：OCR-free 多页文档 Agent，先缩略图概览再按需精读，域外准确率提升 47.9%

## 摘要

**多页文档理解正在从「全量 OCR」进化为「视觉 Agent 主动检索」，架构层面解决了长文档的 Token 爆炸问题。**

- Doc-V* 采用 coarse-to-fine 策略：缩略图概览 → 主动定位相关页 → 存入工作记忆 → 精读推理
- 域外场景准确率比 RAG 基线提升 47.9%，接近闭源模型水平
- 关键设计：Agent 自主决定「读什么」而非暴力全扫，Token 效率质变

这不只是一个 OCR 替代方案——它验证了 Agent 范式在文档理解上的结构性优势。当「看什么」由 Agent 决定而非预设管道，长文档理解的边际成本将从线性变为对数级。

## 原文

Struggling with AI that can't handle long, multi-page documents? 

Researchers from HUST and Xiaomi introduce Doc-V*, an OCR-free agent that starts with a thumbnail overview, then actively retrieves and reads only the relevant pages, storing evidence in a working memory. 

It outperforms open-source baselines and approaches proprietary models, boosting out-of-domain accuracy by 47.9% over RAG.

Doc-V*:Coarse-to-Fine Interactive Visual Reasoning for Multi-Page Document VQA
