# Cursor 3 把 PR Review 全链路收编进 IDE：代码审查正在从「跳转 GitHub」变成「不离开编辑器」

## 摘要

**Cursor 把 PR 创建到合并的完整流程内嵌到编辑器中，消除上下文切换成本。**

• 支持评论、diff、commit、review 状态一站式查看
• 文件树和变更选择器加速大型 PR 浏览
• AI Coding 工具的竞争焦点从「生成代码」扩展到「代码生命周期管理」

gakki 锐评：这步棋的本质是 IDE 想成为开发者的「唯一工作台」——当 PR Review 也不需要跳出 Cursor 时，GitHub 的入口价值正在被稀释。竞争从「AI 写代码」升级到「AI 管代码」。

## 原文

A new PR review experience is now available in Cursor 3.

Take PRs from creation to merge, all in one place.

You can see comments, diffs, commits, and review status to understand what changed and next steps. Navigate larger PRs more quickly with the file tree and changes picker. https://t.co/ZOFrf7mPSn
