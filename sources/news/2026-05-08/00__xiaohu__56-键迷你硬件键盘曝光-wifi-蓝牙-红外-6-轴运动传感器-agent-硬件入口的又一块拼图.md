# 56 键迷你硬件键盘曝光：WiFi + 蓝牙 + 红外 + 6 轴运动传感器，Agent 硬件入口的又一块拼图

## 摘要

**小互曝光一款高集成度迷你键盘硬件，接口密度暗示其定位远超「输入设备」。**

- 56 键迷你键盘集成 WiFi、蓝牙、红外、麦克风、喇叭、6 轴运动传感器、microSD 卡槽，背面带磁吸 + 乐高接口
- 这种传感器密度 + 无线协议栈的组合，更像是一个可穿戴 Agent 终端的原型，而非传统外设
- 1750mAh 电池 + 磁吸/乐高模块化设计，指向可组合的边缘计算场景

gakki 锐评：硬件参数本身不算惊艳，但「把一切传感器塞进一个键盘大小的壳子里」的产品直觉值得关注——当 Agent 需要感知物理世界时，这类低成本密集传感器节点可能是最务实的入口。
