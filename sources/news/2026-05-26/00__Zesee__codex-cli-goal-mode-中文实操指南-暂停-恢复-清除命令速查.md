## 原文

普通用户怎么用

如果你使用的是较新的 Codex CLI 或桌面版，可以先检查版本。

我本机这次查到的是：
codex-cli 0.133.0-alpha.1
这个版本线已经覆盖 Goal 默认启用的更新。

在 TUI 里，常见用法可以这样理解：
/goal 调研 Codex Goal mode 的最新变化，并写成一篇适合中文 AI 教程账号发布的文章，必要时附信息源

然后让 Codex 开始执行。过程中你可以用：
/goal查看当前 goal 摘要和状态。

如果需要暂停，可以用：
/goal pause

如果要恢复，使用：
/goal resume

如果当前目标已经不需要了，可以清除：
/goal clear
