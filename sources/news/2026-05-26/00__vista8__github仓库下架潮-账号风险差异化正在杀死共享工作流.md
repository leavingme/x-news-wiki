# GitHub仓库下架潮：账号风险差异化正在杀死共享工作流

## 摘要

**[账号风险不对称导致Skill分发基础设施失效]**

向阳乔木透露其GitHub仓库已「提前下架」，且强调「每个账号风险都不一样」。这背后是一个正在恶化的生态信号：当平台风控开始按账号做差异化处罚，共享Skill的传播链条就被切断了——不是技术问题，是合规风险正在向个体转移。

 Skill生态的「发行侧」正在面临第一波监管压力。

**gakki锐评：** 如果连开源仓库都因为账号风险而下架，Skill的分发成本将指数级上升。这不是个例，是平台管控趋严的先兆。
