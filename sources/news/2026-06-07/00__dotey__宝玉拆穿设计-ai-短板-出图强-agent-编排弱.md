# 宝玉拆穿设计 AI 短板：出图强，agent 编排弱

## 摘要

**设计类 AI 工具的核心能力断层被宝玉一句话点破**。
- "设计能力不错，agent 能力不行"——这是对当前一批可视化设计/前端生成工具的精准定性
- 工具能产出漂亮成品，但难以被嵌入多步骤工作流（接收上游需求、向下游交付、版本管理、协作）
- 与 ID 2063447800482759107 "本地可闭环修改设计" 对照阅读：单点交互够用，但放进 agent pipeline 就掉链子
