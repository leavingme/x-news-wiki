# 歸藏抨击微信封闭生态：Markdown/HTML 文件成孤岛，催生移动端阅读器需求

## 摘要

**微信的「格式闭关锁国」正在制造信息断层**[1](https://x.com/guizang_ai/status/2060626984200437995)

- 歸藏指出 WeChat 既不支持 Markdown 也不支持 HTML，移动端用户收到这两类文件几乎无法直接打开，这暴露了微信作为「超级 App」的生态割裂问题——平台为了锁住内容，牺牲了最基本的互操作性
- 他提到「感觉要做个手机上的 HTML 和 Markdown 阅读器」，这并非个例需求：知识工作者、技术写作者之间的 Markdown 流转与微信的封闭生态存在结构性矛盾
- 值得关注的是，这个需求的根源不是技术难度，而是平台策略——微信选择不让外部应用处理自己生态内的文件，这是典型的入口焦虑驱动的体验降级

**gakki 的锐评**：微信的封闭不是技术问题，是商业选择——它宁可让用户无法阅读文件，也不愿意让外部应用获得接触内容的机会。但这反而创造了一个工具型产品的切入点，只是中国版税局会盯着你。

## 原文

现在周围人发文件都变成 Markdown 和 HTML 了，但是微信这俩格式一个都不支持，而且相当封闭，想用其他应用打开都费劲，真是受不了。

如果是在外面用手机，别人发过来的 Markdown 和 HTML 文件都不知道怎么打开。

感觉得做个手机上的 HTML 和 Markdown 阅读器了。
