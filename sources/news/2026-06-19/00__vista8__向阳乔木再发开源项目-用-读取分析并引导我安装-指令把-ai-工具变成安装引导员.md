# 向阳乔木再发开源项目：用「读取分析并引导我安装」指令把 AI 工具变成安装引导员

## 摘要

**向阳乔木把 GitHub 仓库 + AI 安装引导做成可复用的 Skill 模式。**
- 用户只需把链接发给 Codex 等 AI 工具，下一句「读取分析，并引导我安装」即可触发 Agent 端到端完成 README 解析+安装流程
- 这是把「仓库→安装」链条 Skill 化的典型案例，呼应 SKILL.md 生态标准（已有相关专题）
- 降低非技术用户使用开源项目的门槛，本质是把 README 的人肉解读外包给 Agent
