# ChatGPT 网页版开启开发模式：第三方 MCP 可读本地文件，变向获得双倍 Codex 额度

## 摘要

**[OpenAI 悄悄把 ChatGPT 变成 MCP 客户端——网页端也能调本地文件]**

- 开启路径：ChatGPT 网页端 → 设置 → 应用 → 高级模式 → 开启开发模式 → 添加 MCP
- 核心用途：Codex 额度耗尽时，切换 ChatGPT 网页端继续开发
- 实质效果：变向获得双倍 Codex 配额（向阳乔木原话"有点诡异"）
- 把 ChatGPT 从"问答工具"延伸到"开发环境"


gakki 的锐评：OpenAI 在用 ChatGPT 网页端做"应用平台"——开发模式 + MCP 是把浏览器变成 IDE。这步棋比 Codex CLI 威胁更大，因为 ChatGPT 用户基数 10 倍于 Codex，意味着 MCP 标准争夺里 OpenAI 已经占了入口先手。

## 原文

有意思，开发一个MCP，连接到网页版的ChatGPT。

使用这个MCP就能读取本地的文件。

Codex额度用光，无法重置时，也可以继续用ChatGPT开发。

变向获得了双倍codex额度，虽然有点诡异。

配置注意事项：

打开 ChatGPT网页端，设置-&gt;应用-&gt;高级模式-&gt;开启开发模式，添加MCP
