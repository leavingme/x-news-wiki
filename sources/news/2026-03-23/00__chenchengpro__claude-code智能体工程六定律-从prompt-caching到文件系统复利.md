# Claude Code智能体工程六定律：从Prompt Caching到文件系统复利

## 摘要

**[Claude Code Skills方法论体系化输出]**
• Skills是「复用单元」而非功能，是智能体架构的核心抽象
• Prompt Caching是长会话成本控制的核心，不懂即烧钱
• 智能体必须用文件系统——写入文件的Agent复利成长，依赖临时上下文的只会反复重建
• Bash工具对所有Agent都是必需品，即使是非编码场景

**gakki锐评：** 这是Claude Code生态从「工具」向「工程范式」跃迁的信号。文件系统作为Agent记忆的锚点，正在重新定义Agent的「状态持久化」哲学。
