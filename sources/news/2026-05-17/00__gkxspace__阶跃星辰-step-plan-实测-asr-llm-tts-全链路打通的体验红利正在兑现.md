# 阶跃星辰 Step Plan 实测：ASR+LLM+TTS 全链路打通的体验红利正在兑现

## 摘要

**阶跃星辰 Step Plan 以 ASR+LLM+TTS 全链路套餐形式交付，实测将多件「以前嫌麻烦不会去 LLM 化的小事」自动化。**

- 具体场景：英文录音→带时间戳中文笔记（54秒）、长文→双人对谈 mp3、独白→7种情绪 TTS 演绎
- 速度达 100-300 tok/s，额度充足，表明推理成本控制已到可交付水位
- 组合定价策略规避单模态订阅的性价比焦虑，是 AI 产品定价的进化方向

「全链路套餐」比单点 API 更容易让用户产生「值回票价」的感知，这是 AI 产品定价的下一章。

## 原文

我最近尝试了阶跃星辰的 Step Plan，也是打算长期订阅了，一个套餐包含：ASR + LLM + TTS​
​
而且实测体验很好，简单讲一讲具体我的玩法（具体放评论区）：​
​
1、12 分钟英文录音 → 带时间戳中文笔记 (54 秒)​
2、Paul Graham 12000 字长文 → 8 分钟双人对谈 mp3​
3、同一段独白 → 7 种情绪 TTS 演绎​
4、鲁迅《孔乙己》节选 → 多音色有声书剧场​
5、13 分钟英文 NPR → 中文重制版播客​
​
速度达到100-300 tok/s，额度给的也是非常足，用这套组合把很多我“以前嫌麻烦不会去 LLM 化的小事”全自动化了。
