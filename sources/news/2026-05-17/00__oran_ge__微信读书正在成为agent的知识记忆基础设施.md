# 微信读书正在成为Agent的知识记忆基础设施

## 摘要

**[微信读书 Skill 揭示 Agent 知识管理新路径]**

- Agent 可以把微信读书当作可查询的向量数据库，基于真实阅读记录做兴趣推荐而非关键词匹配，且会判断用户是否真的会对某本书感兴趣
- 「连接纸书」功能打通了纸质书与数字阅读的数据割裂，形成完整的阅读知识图谱
- 这是知识管理从「收藏夹囤积」到「Agent 可消费的知识资产」的关键转折

gakki：微信读书如果开放 API 给第三方 Agent 调用，配合真实阅读行为数据做推荐，将成为中文世界最有价值的个人知识图谱基础设施，比任何笔记工具都更贴近「真实偏好」。

微信读书的 Skill 是个好东西
