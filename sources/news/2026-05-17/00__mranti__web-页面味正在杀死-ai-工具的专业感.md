# Web 页面味正在杀死 AI 工具的专业感

## 摘要

**[Michael Anti 指出 AI 工具界面 Web 化是体验倒退，这个判断值得 AI 产品经理警觉]**

- 当 AI 工具的 UI 越来越像普通 Web 页面，其"专业工具"的定位感知会被稀释
- 这与"Chat UI 不死与范式固化"话题相关：交互形态的同质化正在抹平产品差异
- 可能的深层含义：AI 工具在功能趋同后开始比拼 UI，而 UI 的 Web 化反而降低了使用门槛但也降低了溢价能力

这个观察指向一个核心问题：AI 工具的差异化到底在功能层还是体验层？
