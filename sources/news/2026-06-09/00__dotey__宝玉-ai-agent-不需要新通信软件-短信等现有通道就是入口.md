# 宝玉：AI Agent 不需要新通信软件，短信等现有通道就是入口

## 摘要

**[一句话核心判断]**Agent 入口之争的「反基础设施」立场——复用现有通道优于新建 AI 通信软件。

- 宝玉明确表态：AI Agent 入口不需要专门的 AI 通信软件，沿用手机自带短信等现有通信系统即可
- 隐含工程哲学：用户不会为 AI 单独装一个新 IM，Agent 必须降维寄生在已有通道上
- 与 Telegram Bot、微信生态、小红书 DM 等多入口争夺形成对照，呼应「Agent 多平台入口争夺」专题中「借船出海」的轻量化路线


**gakki 锐评**：大厂押注超级 App，宝玉的判断是「零摩擦胜过高频」——Agent 的价值在任务交付，不在交互革命。边界是：现有通道的 API 限流和风控随时可能绞杀 Agent，纯寄生路径有结构性脆弱。
