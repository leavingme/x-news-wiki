## 原文

Claude 打通 Excel & PPT：跨文件对话贯通 + Skills 工作流复用

咋感觉这件事微软早就该做😂但没做！却让 Claude 这个第三方包抄了。。看看具体功能
https://t.co/x0ljnG38Nt

1. 跨文件上下文贯通
Claude 现在可以在 Excel 和 PowerPoint 之间共享完整的对话上下文。这意味着：
· 在 Excel 中完成的分析，Claude 会记住并直接应用到 PPT 制作
· 用户无需反复解释同一数据集的背景
· 一个连续对话可跨越多个 Excel 和 PPT 文件

实际场景：金融分析师可以从 Excel 工作簿拉取可比公司财务数据，构建交易对比表，然后将估值摘要直接放入 pitch deck，并起草给 MD 的邮件——全程无需切换标签或重复说明数据集。

2. Skills —— 一键复用最佳实践
Skills 将完整工作流转化为一键操作。一旦团队中的某人找到了正确的方差分析方法或使用公司模板撰写客户报告的方法，将其保存为 Skill 后，未来可立即复用。
Excel 预置技能（覆盖金融分析高频场景）：
· 审计模型：检查公式错误和资产负债表完整性
· 模板构建：LBO、DCF、三表模型
· 可比公司分析
· 数据清洗（支持选定区域、活动工作表或整个文件）

PowerPoint 预置技能（分析后的展示层工作）：
· 构建竞争格局报告：市场定位、同行深度分析
· 用新信息更新现有报告
· 审查投行报告：数字一致性、数据-叙述对齐、语言润色

部署方式：通过 Financial Analysis 插件自动安装，现有 Skills 和 MCP 连接器均可直接使用。Instructions 功能可设置持久性应用级偏好（如公司数字格式、单条 bullet 限制等）。

3. 部署灵活性：三大云平台支持
Claude for Excel 和 PowerPoint 现在支持通过 LLM 网关接入三大云平台：
· Amazon Bedrock
· Google Cloud Vertex AI
· Microsoft Foundry
组织可以在现有合规框架内使用 Claude，无需改变现有的 AI 治理架构。

4. Agent Mode 原生集成 Excel
Microsoft 365 Copilot 客户现在可以在 Excel 中与 Copilot 并排使用 Claude 的 Agent Mode，实现工作簿的协同构建、编辑和分析。
