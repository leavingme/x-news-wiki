# Origins Network 拿的不仅是钱：AWS/腾讯云/阿里云同时站台意味着什么

## 摘要

**[一句话核心判断]**

Origins Network 800万美元融资背后，三大云厂商同时入局背书，这个组合本身就是一种信号——不是叙事、是资源协同的预判。

• **云巨头的真实动机**：AWS/Tencent Cloud/Alibaba Cloud 不会为空气站台。他们看的是「链上 AI 训练」这个场景能否把自己的 GPU 资源池进一步变现，同时卡位 Web3×AI 的基础设施层。
• **闲置 GPU 接入链上 AI 训练**：这个方向如果跑通，本质上是把算力代币化的另一种表达——不是发币，而是把 GPU 使用权变成可验证的链上资产。

当云厂商开始公开押注某个 Web3×AI 基础设施，意味着这个方向的「合规可能性」已经被悄悄评估过了。

**gakki锐评：** 800万在加密圈不算大，但三大云同时出现才是重点——这不是投资，是站队。

## 原文

今年都没有啥行情，只能撸撸老马的工资，逛一逛有什么值得投的，昨天偶尔看到Origins Network仔细看了下

Origins Network 刚完成 800 万美元融资，我去看了一下，投资人这波确实挺有意思。

Animoca Brands、TBV、Candaq、Castrum Istanbul、CoinVestor Ventures这些机构一起下场，还有家族办公室和天使跟进。战略合作方直接拉来 AWS、Tencent Cloud 以及 Alibaba Cloud，云端巨头亲自站台，这种背书可不是随便讲讲叙事就能拿到的。

他们在做的其实挺直接：一条专为 AI × Web3量身打造的模块化公链。Sharding 1.0 版，吞吐量更高、成本更低，还保持以太坊级别的安全性。

最硬核的是，用户能把闲置 GPU 直接贡献给网络，跑链上 AI 训练，而且算力过程可验证。ORC20 铭文也支持，比传统 NFT 便宜得多、去中心化程度更高，生态冷启动挺方便。

生态已经实打实上线跑起来了：OrtradeX（去中心化交易所）、OrClaw AI（AI 应用层）、Origins Cloud（GPU 算力市场）、Or Wallet（用户入口钱包），整个闭环都在转，不是还在画大饼的阶段。

$OR 代币设计也算干净：治理 + 质押，平台收入直接回购销毁，固定供应量零通胀，最后把网络经济交给社群共同拥有。

一句话概括，这波融资 + 顶级机构 + 云巨头背书，叠加已经落地的技术和跑起来的生态，Origins Network 在 AI × Web3 这条路上至少把“可验证算力 + 模块化公链”给落地了。值得持续盯一眼。
