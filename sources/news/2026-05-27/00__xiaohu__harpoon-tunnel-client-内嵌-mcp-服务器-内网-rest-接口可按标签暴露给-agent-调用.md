# Harpoon：tunnel-client 内嵌 MCP 服务器，内网 REST 接口可按标签暴露给 Agent 调用

## 摘要

**Harpoon 是 tunnel-client 里内嵌的一个 MCP 服务器，允许将内网 REST 接口按标签暴露给 Agent 调用，但调用方不能乱选目标，并非通用代理。**

要点：
- 内网接口"安全出洞"的一种路径，非通用代理
- 解决内网服务被 Agent 调用时的访问控制问题
- 标注来源标签，过滤调用范围，属于精细化权限控制

gakki：「让 Agent 碰内网服务，Harpoon 这个路子比直接开端口高明——标签即权限，接口即工具，Agent 调用变成有审计的受控行为。」

## 原文

还有个小彩蛋叫 Harpoon，是 tunnel-client 里内嵌的一个 MCP 服务器，可以让你把内网里一小撮 REST 接口配进去，按标签暴露给 Agent 调用，但调用方不能乱选目标，不是通用代理。
