# 向阳乔木 Suno Skill 改样式时不慎丢失功能，正在修复中

## 摘要

**[Skill 工具在实际使用中暴露的脆弱性：功能稳定性问题]**

- 向阳乔木明确表示「等我更新一版」，承认昨天改样式把功能改丢了
- 这是 Skill 作为生产工具的真实一面：迭代过程中功能丢失，说明 Skill 的版本管理和测试机制尚未成熟

**gakki**：Skill 不是魔法，是需要版本控制的代码。改样式改丢功能这件事，暴露了 Skill 市集缺乏自动化测试和灰度发布的工程短板。用 Skill 搭产品，绕不开工程纪律这一关。
