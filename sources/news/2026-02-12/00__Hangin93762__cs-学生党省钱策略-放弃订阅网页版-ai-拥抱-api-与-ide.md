# CS 学生党省钱策略：放弃订阅网页版 AI，拥抱 API 与 IDE

## 摘要

**[核心逻辑]**
- 资深用户建议 CS 学生放弃 ChatGPT/Claude 昂贵的网页订阅费。
- 推荐使用 Perplexity 进行学术搜索，省下的钱投入到 Coding Plan 和 API 调用上。

[gakki 的锐评]
这就是“生产力思维”对“消费级思维”的降维打击。别在聊天框里浪费钱，去代码里烧算力才是正经事。

## 原文

分享一则自己使用ai的小经验（非广告）：
如果你是一个CS相关专业的学生或者并不需要大量使用网页版ai进行各类复杂工作流进行工作的用户，我并不推荐订阅chatgpt /gemini / claude的网页版，更不推荐这几家都订阅；
如果为了极致的省钱，可以使用 Qwen / GLM等便宜的中国模型，这个基本上可以做到不花钱；如果想要使用高性能的美国模型，可以订阅一个perplexity就可以了，perplexity中可以使用chatgpt gemini claude kimi等高性能模型，并且它的搜索、research功能是我体验非常SOTA的，尤其是从事research的研究生博士生非常需要，如果有学生会员的话每个月只要10美元；
省下来的钱可以用来购买AI IDE的会员或者各家的Coding Plan
