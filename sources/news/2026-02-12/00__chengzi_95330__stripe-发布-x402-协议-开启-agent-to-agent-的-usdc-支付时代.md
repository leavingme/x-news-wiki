# Stripe 发布 x402 协议：开启 Agent-to-Agent 的 USDC 支付时代

## 摘要

**[核心逻辑]**
- 基于 Base 链，利用 HTTP 402 状态码实现 AI Agent 之间的自主支付。
- 单笔成本仅 0.01 美元，实现 2 秒级确认，无需人类干预授权。
- CoinGecko 等平台已率先接入，允许 Agent 实时购买数据 API。

[gakki 的锐评]
支付是 Agent 闭环的最后一块拼图。当机器可以自己付钱买算力和数据，它们才真正拥有了在这个数字社会独立行动的“经济主权”。

## 原文

Crypto x AI Agent赛道密集爆发，Agent经济的基础设施正在加速成型。

1. Stripe上线x402支付协议，基于Base链

让AI Agent用USDC自主完成支付，无需人工介入

利用HTTP 402状态码，单笔成本约$0.01，2秒确认

CoinGecko已率先接入：每次API调用$0.01 USDC，无需注册

2. ERC-8004正式上线以太坊主网

定义了三个链上注册表：身份(Identity)、声誉(Reputation)、验证(Validation)

测试网阶段已有10,000+ Agent注册，20,000+反馈条目

3. Coinbase推出Agentic Wallets

首个专为AI Agent设计的钱包基础设施

Agent可自主持币、交易、赚取收益、管理DeFi仓位

4. Kris Marszalek花 $7000万买下 https://t.co/rCBRbjbNJD 域名，超级碗投放广告，推出个人AI Agent平台。

域名史上最贵公开交易。Agent叙事已经从开发者圈层破圈到主流市场。
