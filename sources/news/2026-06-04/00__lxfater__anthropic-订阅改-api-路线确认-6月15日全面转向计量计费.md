# Anthropic 订阅改 API 路线确认：6月15日全面转向计量计费

## 摘要

**一句话核心判断**
Anthropic 确认 6 月 15 日所有订阅套餐 token 不再可用于第三方，第三方调用全部转 API 计费——大模型公司的盈利压力正式传导到生态层。

**信息增量**
1. 这不是临时政策调整，而是明确的商业路线切换，意味着 Anthropic 正在系统性地收紧 API 边界
2. 订阅转 API 后，成本从「月费固定」变为「按量弹性」，对高频调用者影响最大
3. 与 OpenSquilla 的分档路由方案同期出现，说明市场正在快速形成「绕过订阅税」的工具链

**gakki 锐评**
Anthropic 需要上市，所以必须盈利——这是公开的秘密。但每一次「API 收紧」都在给开源生态和创新者发令枪：谁能更好地组织模型资源，谁就能拿走被吃掉的那块利润。

## 原文

6月15日，Anthropic所有的订阅套餐的TOKEN都不能被第三方使用了，全部转为API计费。
大模型公司们为了上市，必须盈利，现在是时候想想如何节约TOKEN

大部分人的思路可能都是换个便宜模型，然后眼睁睁看它变笨
但 OpenSquilla 反其道而行之

这个开源 Agent 给每个请求按难度打分，分四档派发：
简单活走便宜模型
硬活照样上最强的

控制台里就能看到这套分档，T0 走 deepseek-flash，T3 才上 claude-opus：
省钱，但不降智！！

官方测下来，输入 token 比 OpenClaw 少 44%，172 万对 306 万
综合成本官方称能降 60 到 80%。

这还不是最厉害的，他们现在发明一个东西叫MetaSKill
