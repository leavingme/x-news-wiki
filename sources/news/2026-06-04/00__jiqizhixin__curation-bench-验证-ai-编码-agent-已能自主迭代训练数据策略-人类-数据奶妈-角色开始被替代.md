# CURATION-BENCH 验证 AI 编码 Agent 已能自主迭代训练数据策略：人类「数据奶妈」角色开始被替代

## 摘要

**[一句话核心判断]**
AI 编码 Agent 不再只是执行工具——它已具备自主定义和迭代数据筛选策略的能力，意味着模型训练流程中最后一个需要人类密集干预的环节正在被自动化。

**信息增量：**
- CURATION-BENCH 由 Virginia Tech、UIUC、UW-Madison、UC Berkeley 四校联合提出，专门评测通用编码 Agent 对数据策略的自主掌控能力
- 无需人工手把手引导，使用开箱即用的 Agent 配合 scaffolded 架构，十次迭代即可匹配已发布强基线
- 真正的突破在于方法引导式探索（method-guided exploration）——Agent 能自主「发明」数据策略，而非简单复现人类规则

**影响：**
- 这条路打通后，合成数据生成（Synthetic Data）和数据飞轮（Data Flywheel）将形成自驱闭环，大幅降低模型对人工标注的依赖
- 对数据标注行业、合成数据创业公司、以及 AI 基础设施提供商都是明确的预警信号

## 原文

What if an AI agent could curate its own training data—without human hand-holding?

Researchers from Virginia Tech, UIUC, UW-Madison, and UC Berkeley present CURATION-BENCH, a benchmark that gives generalist coding agents command-line access to inspect, implement, test, and revise data policies autonomously.

Their out-of-the-box agents matched strong published baselines in just ten iterations. But the real breakthrough? A scaffolded agent that forces method-guided exploration autonomously invented a data-selection policy that outperforms those same baselines—using only one-tenth the data budget.

The catch: agents can run the curation loop, but reliable data research still needs structured nudging, not just open-ended prompting.

Can Generalist Agents Automate Data Curation?
