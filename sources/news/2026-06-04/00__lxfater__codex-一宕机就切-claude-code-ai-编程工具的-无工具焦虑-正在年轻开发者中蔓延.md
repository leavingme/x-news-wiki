# Codex 一宕机就切 Claude Code：AI 编程工具的「无工具焦虑」正在年轻开发者中蔓延

## 摘要

**铁锤人自述：昨夜 Codex 不可用，迫使他临时订阅了 Claude Code，并坦言「AI 只要不能用，就浑身刺挠」。**

信息增量：这不是简单的工具切换记录。它揭示了一个正在成型的用户行为模式——当单一 AI 编程工具成为「必需品」而非「可选项」时，平台稳定性直接锚定用户留存与支出决策。宕机触发付费转化，说明用户黏性已超越品牌忠诚，演变为对「持续可用性」的条件反射式依赖。

对谁有影响：Anthropic 的 B2C 订阅收入模型获得了一个意外验证样本——竞品故障成为其获客渠道。Codex 的 SLO（服务级别目标）正在成为用户留存的关键指标，而非模型能力本身。

gakki 的锐评：工具焦虑和工具成瘾只有一线之隔。当「换工具」变成肌肉记忆，用户的议价能力其实在下降——他们不是在做选择，而是在找下一个锚点。
