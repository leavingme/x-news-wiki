# 字节扣子率先打通手机操控Claude Code：移动端Agent编程工具链的第一枪

## 摘要

**【一句话核心判断】**
扣子（Coze）实现手机端直接操控电脑上的Claude Code和Codex，这是大厂首次将AI编程工具链延伸至移动端的工程化落地，意味着真正的「随身Agent工位」从概念进入可用状态。

**【信息增量】**
1. 字节是第一个头部大厂做这个功能，竞品尚未跟进；
2. 功能免费，且面向个人用户直接开放，非企业定制；
3. 移动端操控Agent编程工具，改变了远程/多设备工作场景的边界。

**【影响】**
对需要跨设备调用AI编码能力的独立开发者、多设备办公的用户，以及移动端Agent生态的演进路径有直接意义。这是Agent工具链「移动化」的一个落地信号。


**gakki锐评：** 这个功能本身不复杂，但背后是字节对「手机作为Agent指挥塔」的战略押注——先把移动端入口占了，Agent可以在任何设备上跑。下一个问题：各大厂会不会跟进把这个能力SDK化？
