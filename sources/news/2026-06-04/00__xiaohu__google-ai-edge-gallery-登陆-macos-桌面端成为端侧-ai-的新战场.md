# Google AI Edge Gallery 登陆 macOS：桌面端成为端侧 AI 的新战场

## 摘要

**一句话核心判断**
Google AI Edge Gallery 从手机端扩展到 macOS 桌面，提供可下载安装的 Gemma 4 12B 本地运行环境，内置沙盒 Python 环境 + Eloquent 语音交互。这意味着 Google 正在用桌面应用作为端侧 AI 的分发入口，与 Cursor、Claude Code 等 AI 编程工具争夺开发者桌面。

**要点**
- 产品形态：原生 macOS 应用，用户下载后直接在本地跑 Gemma 4 12B，无需 API 调用，完全离线
- 内置能力：沙盒 Python（聊天里写代码执行）、Eloquent 语音交互（对着电脑说话触发 AI）
- 战略意图：Google 在端侧 AI 落地路径上选择了「应用分发」而非「模型输出」——控制入口比卖 API 更有价值

**gakki 锐评**
Google 这步棋醉翁之意不在技术展示，而在于卡位「桌面端 AI 应用分发」。当用户习惯在 Edge Gallery 里跑代码、画图、语音对话，Google 就成了事实上的桌面 AI OS 层——跟 Cursor 抢的不是 IDE，是入口所有权。
