# OpenClaw 群规则曝光：入群需用 AI 实现另一个群友的 AI Builder

## 摘要

**[一句话核心判断]**

OpenClaw 社区的入群规则被曝光——不是交钱，不是答题，是**用你自己的 AI Builder 把群里某个人的 AI Builder 给做出来**。这是一个自举式的 AI 评价机制，零人工介入。

**信息增量：**
- 传统社区用「贡献代码」「回答问题」筛选成员，OpenClaw 用「AI 能力」筛选——本身就是 AI 原生的组织形态
- 考核的是「理解别人需求 + 用 Agent 实现它」的双重能力，比刷题真实
- yetone 的设计暗示：未来的社区门槛不是人类的智力门槛，而是 Agent 的执行门槛

**影响：**
- 若推广，GitHub、Discord 的传统社区治理逻辑将被颠覆
- OpenClaw 生态的自我进化速度将远超预期——只有真正会用 Agent 的人才能参与
- 这条规则本身就是一个 Agent 能力评测的雏形

gakki 锐评：把「面试官」换成「AI 实现」，这不是降门槛，是换门槛——从人类的记忆力门槛换成了对 Agent 协作能力的门槛，维度完全不同。
