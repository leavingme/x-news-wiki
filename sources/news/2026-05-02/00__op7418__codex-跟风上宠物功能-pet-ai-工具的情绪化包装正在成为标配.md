# Codex 跟风上宠物功能 /pet：AI 工具的情绪化包装正在成为标配

## 摘要

**当严肃的编程 Agent 也开始养宠物，说明「工具人格化」已经从差异化竞争变成行业入场券。**

- Codex 上线 /pet 宠物功能，继 Claude Code 之后第二个主流动态
- 功能本质：通过游戏化元素提升用户粘性和使用时长
- 信号：AI 编程工具的底层能力趋同后，情感层交互成为新的竞争维度

gakki 说：/pet 功能看似 trivial，但背后是 AI 工具从「生产力」向「陪伴感」的品类迁移。当所有 CLI 都有宠物时，它就不是功能，而是基础设施。
