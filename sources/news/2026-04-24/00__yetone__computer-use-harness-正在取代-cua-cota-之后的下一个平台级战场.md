# Computer Use Harness 正在取代 CUA：CoTA 之后的下一个平台级战场

## 摘要

**[yetone 观察到 Codex Chronicle 正在成为 Computer Use 的新 Harness 层]**

Computer Use 已ready，但 Codex Chronicle 作为 Computer Use Harness（CUH）尚未完成。当前 CUH 正在代替 CUA（Computer Use Agent），这意味着行业分工正在从「单点模型能力」向「模型+ Harness 组合」演化。

Harness 是连接模型能力与真实任务执行环境之间的中间层，包括任务分解、工具编排、错误恢复等工程逻辑。这与之前 MIT 研究「42%到78%的 AI 性能差距来自框架而非模型」的数据形成呼应。

gakki：Harness 工程才是 Agent 落地的真正壁垒，模型能力反而越来越同质化。下一个平台级机会不在模型，而在让模型好用、可靠、可组合的中间件层。

## 原文

Computer Use 已经做好了，但是 Codex 作为 Computer Use Harness 还没做好。现在 CUH 已经在代替 CUA 了，接下来 Computer Use Harness 应该是一个很火热的领域。
