# 用 Claude Code 切换 DeepSeek V4：白嫖党 vs 人民币玩家的生存指南

## 摘要

**[工具链互操作进入「无门槛切换」阶段]**

向阳乔木放出实操教程：用 CC Switch 让 Claude Code 无缝接入 DeepSeek V4 API，实现「不差钱用 Pro、差钱用 Flash」的模型映射。

- CC Switch 作为模型路由层，打通了 Claude Code 与国产模型的互操作
- 教程门槛低到「普通用户」也能上手，说明工具链平民化进入新阶段
- 实测写了个 3D Flappy Bird 但卡在第三个柱子——模型能力边界依然清晰

**gakki 锐评：** 这类教程的真正价值不是「怎么用」，而是揭示了一个结构性趋势：Claude Code 正在变成一个模型无关的「 Agent 执行外壳」。工具链的抽象层正在高于模型选择本身——这比任何一个模型的发布都更值得警惕。

## 原文

虽然很简单，但还是写个简单小教程，方便普通用户。

用Claude Code测DeepSeek V4步骤：

1. 官方API充值，创建API key。（发现还有50多余额）

2. CC switch 创建Claude Code供应商，选DeepSeek，填入API key。

3. 配置模型：不差钱的话都设置 deepseek-v4-pro。

或 deepseek-v4-pro 和 deepseek-v4-flash 做不同模型映射。

官方API充值和CC Switch下载地址见评论区

BTW： 用Pro写了个3D threejs版的Flappy Bird，过不了第三个柱子，菜鸟就是我...
