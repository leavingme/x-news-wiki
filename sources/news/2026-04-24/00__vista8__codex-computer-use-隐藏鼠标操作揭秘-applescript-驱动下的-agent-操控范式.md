# Codex Computer Use 隐藏鼠标操作揭秘：AppleScript 驱动下的 Agent 操控范式

## 摘要

**隐藏鼠标下的精准点击，背后是 AppleScript 的系统级操控能力。**

用户实测 Codex Computer Use 时发现一个关键细节：它可以在隐藏鼠标的情况下完成应用操作点击。这引发对其底层实现的好奇。答案指向 AppleScript——macOS 原生的自动化脚本语言，能直接操控 UI 元素而不依赖光标位置。

这意味着 Computer Use 的核心能力不是「模拟鼠标」，而是「系统级 UI 操作」——Agent 第一次真正接管了系统控制权而非像素层。


**gakki**：隐藏鼠标 + AppleScript = Agent 操控从「模拟人类」升级为「系统原生调用」。这是 macOS 独有的优势，也解释了为什么 OpenAI 选择 macOS 作为 local-first Agent 的主战场。Windows/Android 的等效能力缺失是国产 Agent 的硬伤。
