# Workspace Agent 开源版：终于有一个不被任何模型商绑定的 Agent 部署方案

## 摘要

**一句话核心判断：Workspace Agent 开源版解决了 Agent 落地最大的痛点——厂商锁定，Docker 沙箱 + 凭证隔离让每个用户跑在独立环境里。**

• 任意模型：Claude / GPT / Gemini / Kimi / DeepSeek 都能接，不被任何一家绑定。
• 独立 Docker 沙箱：每个会话隔离运行，一个崩了不影响其他，企业级可靠性。
• 终端用户凭证隔离：这是给 SaaS 加 AI 助手的核心需求——每个用户登录自己的账号不串号。
• 部署成本：最低 €4/月可自建，支持 Telegram 适配器，做企业内部受控 Agent 可限制只能访问指定 API。
• 全程可观测：子 Agent 调用链路透明，不是黑盒。

gakki：开源 Agent 框架这个赛道已经有过很多玩家，但「不被模型商锁定 + 凭证隔离 + 可观测」这三个条件同时满足的，这是第一个。中小团队福音，大厂警惕。
