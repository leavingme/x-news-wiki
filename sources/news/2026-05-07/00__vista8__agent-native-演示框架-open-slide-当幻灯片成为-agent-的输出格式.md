# Agent-Native 演示框架 open-slide：当幻灯片成为 Agent 的输出格式

## 摘要

**演示工具正在被 Agent 重新定义。**

- open-slide 定位为「A slide framework built for agents」，2.2k stars / 146 forks，活跃度高（343 commits，35 branches）
- 从 commit 记录看，已进入 monorepo 架构阶段，包含 CLI 工具 + 设计系统 + playground，工程化程度不低
- 核心信号：幻灯片不再只是人类的表达工具，Agent 也需要结构化的「交付物格式」，PPT/Slides 正在成为 Agent 的原生输出模态

锐评：Agent 的输出界面从 terminal → chat → 文件 → 演示文稿，这条演进链说明 Agent 正在接管「表达层」。但 open-slide 能否跑出来，取决于它能否成为 Agent 生态的标准输出协议，而不只是又一个前端组件库。
