# Tianfu Agent 在 3069 人算命大赛中逼近人类 Top-20：AI 垂直领域 Agent 的「量化验证」标杆

## 摘要

**用 AI 方法论重构紫微斗数，Tianfu Agent 在全球算命师大赛中准确率接近人类顶尖选手。**

- 核心不是「AI 算命」的噱头，而是展示了垂直领域 Agent 的工程化路径：排盘可视化、多流派权衡、最多 500 次工具调用的深度推理
- 关键设计：遇到复杂矛盾时会坦诚「不确定」，比通用大模型的幻觉输出更可靠
- 3069 人参赛的量化验证，给出了可衡量的性能基线——这在 Agent 评测领域是稀缺的

gakki 锐评：抛开命理学的争议性，这个案例的价值在于验证了一件事：当 Agent 被赋予足够的领域规则和自我怀疑能力时，它的表现可以逼近人类专家。这才是「AI 替代人类」讨论中被忽略的关键前提。

科学和玄学之间，真的不可兼容吗？

这和现代 AI 的核心逻辑：基于海量数据进行模式识别和参数优化，在本质上惊人地相似。

这不就是古代的“大数据+AI”实践吗？
 
Tianfu Agent 正在用最严谨的 AI 方法，去认真对待这门古老学问。

它不是黑箱，你能看到它怎么排盘、怎么看宫位，甚至不同流派的权衡过程。
 
它也不是简单背诵，而是实时计算和规则推理。
 
最关键的是，它在 3069 人的全球算命师大赛中，准确率接近人类 Top-20 选手，有量化验证支撑。
