# OpenAI Codex 记忆终于进化：Chronspan 让 AI 边看屏幕边记上下文

## 摘要

**一句话判断：Codex 的记忆系统从「需要用户复述」升级到「主动感知屏幕内容」，是 Agent 记忆范式的实质性进步。**

要点：
• Chronicle 功能让 Codex 能利用近期屏幕上下文改善记忆，无需用户手动 restate context
• 延续上周 Preview 记忆功能，完成了「复述→感知」的闭环
• 记忆从被动响应转向主动感知，Agent 记忆基建的重要一步

Agent 记忆正在从「用户告知」进化到「自我感知」。这是记忆系统正确的发展方向——让 AI 自己长出感知入口，而不是依赖用户手动喂上下文。

## 原文

Last week, we released a preview of memories in Codex.

Today, we’re expanding the experiment with Chronicle, which improves memories using recent screen context.
