# CC-Switch v3.13.0 查出市面假冒 Claude API：glm-5 和 MiniMax-M2.7 被揭为假货

## 摘要

**[一句话核心判断]**
CC-Switch 新版「使用统计」功能像照妖镜一样照出市面流通的假冒 Claude API——有人拿 glm-5 和 MiniMax-M2.7 伪装成真 Claude 售卖，这个发现对 API 采购者来说是直接的生死牌。

**信息增量：**
- CC-Switch v3.13.0 内置的「使用统计」功能可识别 API 供应商的实际模型来源
- Michael Anti 实测发现：买的多家 Claude API 中，有两家是真的，有一家是 glm-5 和 MiniMax-M2.7 假冒的
- 建议所有通过第三方采购 Claude API 的用户更新到新版本自查

**gakki 锐评：** 这才是真正的安全风险——不是模型本身危险，而是中间商在模型名称上掺假。这种检测能力应该成为行业基础设施。
