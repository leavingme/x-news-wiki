# 小米MiMo-V2-Pro正式亮剑：万亿参数模型API定价仅为Claude Opus 1/5

## 摘要

**[国产大模型首次在「参数规模+定价+生态整合」三线同时发力]**
• MiMo-V2-Pro：1万亿参数/420亿激活，支持100万token上下文，AI综合智能指数第8，定价仅为Claude Opus 4.6的1/5
• MiMo-V2-Omni：全模态Agent模型，支持端到端任务执行，官方演示通过OpenClaw在京东自主浏览、索要优惠并下单
• 联合OpenClaw、OpenCode、KiloCode等框架提供一周限时免费接口
• **影响**：国产模型首次在Agent生态层（而非单纯模型层）进行全球化开发者争夺，价格武器+生态绑定策略清晰

**gakki锐评**：1万亿参数卖Claude 1/5的价格不是「性价比」，是「战争宣言」。小米这次不是在卖模型，是在买开发者心智。

## 原文

小米周四连发3款大模型。正式揭晓了此前Hunter Alpha的匿名模型正是这个旗舰基座大模型MiMo-V2-Pro。模型总参数量达1万亿，激活参数420亿，支持100万token上下文，在AI综合智能指数排第8。支持无人工干预的条件下完成复杂工作流编排、长程规划与精准工具调用，API定价仅为Claude Opus 4.6的1/5。

第二款全模态Agent模型MiMo-V2-Omni支持跨模态理解复杂环境、自主制定并执行计划、在遇到异常时实时修正策略，最终端到端地交付完整结果。官方提供的使用案例中，利用OpenClaw实现通过浏览网页做出购买决策，在京东上索要优惠并下单。

第三款语音合成大模型Xiaomi MiMo-V2-TTS支持多方言、多角色、多语气生成，还可以智能识别文本中的标点符号、语气词、强调标记等各类格式信号。

MiMo-V2-Pro、MiMo-V2-Omni将联合OpenClaw、OpenCode、KiloCode、Blackbox及Cline等智能体开发框架团队，为全球开发者提供为期一周的限时免费接口支持。
