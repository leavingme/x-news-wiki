# Claude Code 正式「出圈」：从终端到 Telegram/Discord 的入口层扩张

## 摘要

**[Anthropic 正在将 Claude Code 从开发者工具重新定义为跨平台 Agent 基础设施]**
• 新功能 Claude Code Channels 支持直接通过 Telegram/Discord 与 Claude Code 交互，打破 CLI 边界
• 支持文件传输（TG 50MB、Discord 25MB×10）、历史消息拉取、权限白名单等企业级功能
• 信号：Agent 工具正在经历从「命令行原生」向「多平台入口」的范式迁移，抢占用户常驻场景

**gakki 锐评**：这不是简单的「多端同步」，而是 Anthropic 对「Agent 入口争夺」的正面回应——当 Cursor/Codex 卷模型能力时，Claude 选择先占领用户的聊天习惯。

## 原文

卧槽

Claude Code 终于支持 Telegram 和 Discord 了

Claude Code 更新到 v2.1.80，带了一个新功能：Claude Code Channels

现在你可以通过绑定Telegram或Discord，直接给Claude Code发消息了

- 支持发图片、文件附件，Telegram单文件50MB，Discord 25MB×10个 

- Claude会自动显示"正在输入..."，处理完再发结果

- 还能编辑之前的消息做进度更新 

- Discord额外支持拉取最近100条历史消息和按需下载附件 

- 完整的权限控制，配对模式/白名单/群组多人，防止陌生人乱发
