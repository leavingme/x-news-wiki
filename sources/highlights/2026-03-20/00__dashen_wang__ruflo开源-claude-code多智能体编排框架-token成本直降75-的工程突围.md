## 原文

Claude Code的多智能体编排神器Ruflo开源，GitHub直接爆21.6k Star，Token成本狂降75%。

最近用Claude Code开发企业级应用真头疼。

上万行代码的项目，模型总"失忆"。

上下文稍长就丢关键信息，逻辑直接断成两截。

更离谱的是，文档整理这种简单活，还非得调贵价Opus模型，token钱像流水似的。

多环节任务得手动切模型角色，每次都要重讲上下文，思路全碎，效率砍半。

这时候刷到GitHub上21k+ Star的Ruflo（前名Claude Flow）。

专为Claude Code设计的多智能体编排框架——单打模型变分工团队。

内置60+专业智能体，覆盖编码、测试、安全审计全链路。

还有持久化向量记忆库，任务经验能存能复用，再也不怕模型失忆。

更绝的是三级路由：简单活本地秒处理，中等任务用轻量模型，复杂部分才调Opus。

官方说API成本能降75%，Claude能力上限直接提2.5倍。

现在问题变成——是自己手搓智能体团队，还是直接用现成的Ruflo省精力？
