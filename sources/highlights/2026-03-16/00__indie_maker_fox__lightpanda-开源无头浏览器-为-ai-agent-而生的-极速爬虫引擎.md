# LightPanda 开源无头浏览器：为 AI Agent 而生的「极速爬虫引擎」

• LightPanda专为AI和自动化工作流设计，速度比Chrome快10倍，内存占用减少近10倍
• 兼容Puppeteer/Playwright生态，可直接接入现有Agent工作流
