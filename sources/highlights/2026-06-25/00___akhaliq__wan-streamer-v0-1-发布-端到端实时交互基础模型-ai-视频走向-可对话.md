# Wan-Streamer v0.1 发布：端到端实时交互基础模型，AI 视频走向「可对话」

## 摘要

**实时交互式基础模型正式进入开源/可部署阶段。**

- Wan-Streamer v0.1 主打「End-to-End Real-time Interactive」，意味着用户可在生成过程中介入控制流，而非传统的「提示词→等待→成片」单向流程。
- 与 Sora2、Veo 等闭源电影级生成不同，Wan-Streamer 把「实时性」和「可交互」作为第一性指标，切入直播、陪伴、Agent 多模态感知等场景。
- 视频生成的下一个竞争维度从「画质/时长」迁移到「延迟/交互」，对内容栈上下游（Agent 感知、数字人、互动娱乐）是直接利好。

gakki 的锐评：AI 视频赛道终于有人不卷参数卷延迟了——「实时+可交互」才是 Agent 时代多模态的真正护城河，闭源旗舰的画质领先优势会越来越难守住。
