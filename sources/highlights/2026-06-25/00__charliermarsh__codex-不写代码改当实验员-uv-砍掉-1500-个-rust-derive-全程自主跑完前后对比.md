# Codex 不写代码改当实验员：uv 砍掉 1500 个 Rust derive 全程自主跑完前后对比

## 摘要

**AI Agent 角色正在从「写代码的」滑向「跑实验的」。**

- Astral CEO Charlie Marsh 让 Codex 自己造了个工具，识别并删掉 uv 里 >1500 个不必要的 Rust derive，全程后台自动跑完 before/after 基准对比。
- 价值点：这不是一次性 prompt 写代码，而是「造轮子 → 跑实验 → 量化收益」闭环，由 Agent 在开发者视野之外完成。
- 信号：AI Coding 工具开始吞噬「好奇心驱动的工程优化」——以前懒得做的事，现在随时可以让 Agent 顺手干完。


gakki 锐评：让 AI 替你做 A/B 是真的杠杆，但这条故事能成立的前提是 harness 工程纪律足够，否则 Agent 跑出来的「实验」只是数字垃圾。

## 原文

You can just run experiments now. Constantly. I was curious if unnecessary Rust derives had any measurable build-time cost. So I had Codex build a tool to identify and remove them, then cut &gt;1500 derives from uv, measured the before and after. All in the background. https://t.co/fu0TJb8KWS
