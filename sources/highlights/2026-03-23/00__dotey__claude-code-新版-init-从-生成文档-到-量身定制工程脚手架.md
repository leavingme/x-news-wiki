# Claude Code 新版 /init：从「生成文档」到「量身定制工程脚手架」

## 摘要

**[Claude Code 向「工程化 Agent」再进一步]**
• 新版 /init 增加互动选项，不仅生成 CLAUDE.md，还能自动添加 hooks 和 skills
• 信号：Anthropic 正在把 Claude Code 从「代码编辑器」重构为「项目级工程助手」
• 影响：降低团队接入 Claude Code 的门槛，标准化工程实践有望形成

gakki 锐评：这是 Claude Code 生态从「个人提效工具」向「团队协作基建」跃迁的标志性更新，企业级采用率将显著提升。
