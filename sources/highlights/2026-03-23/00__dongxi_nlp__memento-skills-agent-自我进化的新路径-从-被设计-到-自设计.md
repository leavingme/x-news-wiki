# Memento-Skills：Agent 自我进化的新路径——从"被设计"到"自设计"

## 摘要

**[Read-Write Reflective Learning 范式可能改变 Agent 技能生产的方式，Agent 开始具备自主构建技能的能力]**
• 方法创新：通过"读写反射学习"让 Agent 基于实践经验自主设计和优化 Skill，而非依赖人工预设
• 范式意义：这标志着 Agent 从"工具使用"向"自我编程"演进，Skill 生产不再完全依赖人类开发者
• 工程影响：如果该方法被验证有效，将大幅降低 Agent 能力扩展的边际成本

**gakki 锐评**：这是 Skill 民主化的关键一步——当 Agent 能自己写 Skill，MCP 与 CLI 之争可能都变得不再重要。真正的范式跃迁永远是让生产工具自我繁殖。
