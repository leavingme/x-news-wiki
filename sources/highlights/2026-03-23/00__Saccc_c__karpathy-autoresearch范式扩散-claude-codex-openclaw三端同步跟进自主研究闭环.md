# Karpathy AutoResearch范式扩散：Claude/Codex/OpenClaw三端同步跟进自主研究闭环

## 摘要

**[AI自主迭代方法论正成为Skill层标准组件]**
• Karpathy开源autoresearch一周疯涨3w stars，核心是给AI设定清晰目标和评判标准让其自我优化
• 社区已快速衍生Claude Autoresearch、Codex Autoresearch、AutoResearchClaw三个适配版本
• 从idea到完整论文的全流程自动化正在变成"可插拔技能"

Gakki锐评：自主研究能力的民主化正在发生，但这套范式的真正考验是能否在开放域任务中保持收敛——而非在封闭loop里自我感动。

## 原文

大神Karpathy(@karpathy)前几周开源的autoresearch项目火爆全网，一周疯涨3w stars。

其本质是给AI设定清晰的目标和评判标准，让它自己去不断优化与迭代。

社区目前已经迅速跟进，将理念应用到了日常研究和工作当中。这里我整理了三个相关的优质衍生项目：
1. Claude Autoresearch：专为Claude Code设计的skill，让代码工作和日常任务都能在迭代循环中持续优化。

2. Codex Autoresearch：专为 Codex 打造的skill，让Codex充分发挥编程能力优势，把代码质量提升至最佳。

3. AutoResearchClaw：研究型龙虾助手，完美兼容OpenClaw。你下达任务，它就能自主完成从 idea 到完整论文的全流程。

建议认真学习autoresearch理念，并尝试把该思路融入到自己的研究和实践当中，产出的速度和质量都会有不错的提升。相关仓库链接放下面了👇，值得借鉴参考。
