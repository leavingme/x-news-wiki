# Claude Code 的「新手友好」升级：/init 从配置工具进化为交互式访谈

## 摘要

**[降低认知摩擦是Agent产品化的关键战场]**
• 新版/init通过主动访谈引导用户配置skills、hooks，从「文档阅读」转向「对话式设置」
• 环境变量开关的测试策略，体现Anthropic的谨慎发布节奏
• 这一改动直指Claude Code的核心用户增长瓶颈——非技术用户的上手门槛

**gakki 锐评：** 这是Claude Code从「工程师玩具」向「大众工具」转型的微观信号。当Agent配置的复杂度被封装进对话流程，「自然语言编程」才真正具备普适性前提。

## 原文

Claude 正在测试 /init 命令的新版本

init 是 Claude Code 里的一个命令，用来初始化项目配置（生成 CLAUDE.md 等）

新版本/init命令会主动访谈你，一步一步的帮你设置 skills（技能）、hooks（钩子）等配置项，比原来更智能。

对新用户更友好...

这个新版本还在测试阶段，需要加一个环境变量才能启用：

bashCLAUDE_CODE_NEW_INIT=1 claude
