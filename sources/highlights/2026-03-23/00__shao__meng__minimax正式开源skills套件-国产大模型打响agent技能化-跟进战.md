# MiniMax正式开源Skills套件：国产大模型打响Agent技能化「跟进战」

## 摘要

**[国内厂商正式入局Agent技能生态]**
• MiniMax开源开发与办公两大类Skills，覆盖前端、全栈、移动端及Office三件套
• 与Anthropic、OpenAI的Skills形成直接对比，开发者可横向评估不同厂商技能质量
• 标志着Skills/MCP范式正从硅谷单点扩散为全球基础设施竞赛

**gakki锐评：** 当MiniMax都开始做Skills，说明Agent技能化不再是「要不要做」的选择题，而是「不做就会被生态抛弃」的生存题。国产模型正在从「模型能力内卷」转向「工程生态卡位」。

## 原文

MiniMax @MiniMax_AI 也开源了自己精选的 Skills，包含开发和办公两大类 Skills，其中开发类包含了前端、全栈、移动端等，办公类包含了 MiniMax Office 三件套、PDF 等，大家可以安装后和 Anthropic、OpenAI 开源的 Skills 做一下对比，看看哪个更适合自己，开源地址：
https://t.co/4XS29caiBD

- Anthropic 和 OpenAI 精选 Skills 集合 -
推荐 Top10：面向通用办公场景的 Agent Skills
https://t.co/HOn5nquTLt
推荐 Top10：面向 前端 / 产品 / UI 的 Agent Skills
https://t.co/HtXfSTh9y6

- MiniMax Skills - 
开发类技能
· frontend-dev：前端开发，不只是写 UI，还覆盖动效、媒体素材生成、文案框架、视觉艺术、依赖检查和质量门禁
· fullstack-dev：全栈开发，强调从需求澄清、架构决策、项目结构、认证、实时通信，到 build/smoke/integration 验证的完整流程
· android-native-dev、ios-application-dev、shader-dev：分别面向 Android、iOS 和 GLSL shader，有较完整的规范文档和参考资料

文档与生产力类技能
· minimax-pdf：生成、填写、重排 PDF，不只是生成内容，而是包含封面、版式、配色、合并、表单填写等整套流程
· pptx-generator：生成、编辑、读取 PPTX，还单独拆出了 plugins/pptx-plugin
· minimax-xlsx：读写、分析、编辑 Excel，重点在于保留格式、修复公式、XML 级编辑
· minimax-docx：Word 文档创建、填充、套模板、校验，走的是 OpenXML SDK 这条非常工程化的路线
