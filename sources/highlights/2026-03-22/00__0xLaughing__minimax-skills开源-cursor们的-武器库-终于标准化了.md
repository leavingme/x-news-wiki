# MiniMax Skills开源：Cursor们的「武器库」终于标准化了

## 摘要

**MiniMax将内部 curated skills 开源，标志着 Agent 开发从「个人经验」向「可复用资产」跃迁。**

• 结构化经验资产化：顶级开发者的工程实践被打包为标准化技能，解决 Agent 长期存在的风格漂移和输出不稳定痛点
• 生产级代码民主化：iOS/Android 开发、GLSL Shader 等原本高门槛领域，现在可通过预置 skill 稳定输出
• 生态分层加速：基础模型 → 工具平台（Cursor/Codex）→ 专业 skill 库的三层架构开始成形，每层都在压缩下游的决策成本

---

**gakki 的锐评**：这不是简单的「模板开源」，而是把「好 Agent 的 DNA」做了提取和封装。当 skills 成为流通货币，Agent 领域的贫富差距会被进一步拉大——会用 skill 的人和不会用的人，产出的将是两种东西。

## 原文

MiniMax 直接把官方 skills 开源了

以前让 Agent 做复杂项目，经常写着写着就幻觉 风格不统一 细节拉胯
