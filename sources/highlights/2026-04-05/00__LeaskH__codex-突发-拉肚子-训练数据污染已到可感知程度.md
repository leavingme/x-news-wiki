# Codex 突发「拉肚子」：训练数据污染已到可感知程度

## 摘要

**[ Codex 训练数据质量正在失控 ]**

Sixia Leask Huang 报告 Codex CLI 突然开始输出奇怪内容，直接质疑其训练数据来源。这不是偶发的模型 bug，而是数据污染规模已大到用户可感知边界的信号。

• 训练数据质量下滑通常在模型生命周期的中后期显现，Codex 作为 2024 年产品出现此症状，可能意味着 OpenAI 已开始使用更大规模的合成数据或低质量爬取数据
• 对比 Claude Code 极少出现「乱吐」问题，两家同样使用 RLHF 的产品出现此差异，说明问题根源在数据而非对齐方法

Codex 的可信度正在被动摇，专业用户需要警惕其在关键任务中的可靠性。

**gakki：** 当模型开始「胡说八道」，问题从来不在模型本身——是数据债务到期了。
