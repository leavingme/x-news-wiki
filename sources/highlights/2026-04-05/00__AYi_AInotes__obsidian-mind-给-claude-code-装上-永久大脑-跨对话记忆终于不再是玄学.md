# Obsidian Mind：给 Claude Code 装上「永久大脑」，跨对话记忆终于不再是玄学

## 摘要

**「Claude Code 再强，每次开新对话都是从零开始。」**

Obsidian Mind 这个开源项目给出了工程级解法——通过 CLAUDE.md 和持久化 Markdown 文件构建外部思维系统，让 Claude Code 在任何新对话中开局就拥有完整上下文：项目进度、团队协作模式、三周前定好的方案，都能自动加载。

核心设计：文件系统即记忆，AI 每做重大决定前读取 task_plan.md 确保目标在注意力窗口内，阶段完成后更新复选框状态。这使得 50 次以上的工具调用链不再迷失方向。

gakki：Manus 能在 8 个月内做到 1 亿美元营收，核心秘诀就是这套「上下文工程」——但问题是：当记忆变成了外部文件，AI 的「真正理解」和「文件读取」之间的差距，谁来解决？」

## 原文

用Claude Code写代码的程序员老哥们，有了这个神器，你们再也不用反复跟AI解释同一件事了~(¯▽¯~)

这个叫Obsidian Mind的开源项目，直接给Claude Code装了一个不会失忆的永久大脑，强烈推荐😁

它不是一个普通的Obsidian笔记模板，是专为工程师打造的、能和Claude深度联动的外部思维系统。

用过Claude Code的人，都懂这个致命痛点：
它写代码的能力再强，每次开新对话都是从零开始。
你的项目进度、核心目标、之前定好的决策、团队的协作模式，每一次都要重新解释一遍。
三周前聊好的方案，三次对话前定的分工，转头它就忘得一干二净。
你的知识和经验，永远没法形成真正的复利。

现在 Obsidian Mind直接把这个问题彻底解决了。
每次开启新对话，Claude会自动加载你的完整知识库，从第一句话开始，它就完全清楚你的所有上下文。
你正在推进的项目卡点、上一轮对话定好的分工、明天要开的1对1会议重点，它开局就门儿清，再也不用你重复半句废话。

整个系统的设计，完全贴合工程师的日常工作流：

• CLAUDE.md 是对话的核心操作手册，定义每一次对话的启动和收尾流程，让Claude始终按你的规则工作

• North Star.md 是你的目标锚点，让Claude的每一个建议、每一行代码，都始终对齐你的核心方向

• Memories.md 是Claude的永久记忆库，跨对话持续更新、留存你的所有决策和关键信息，再也不会失忆

• 每次对话启动，它会自动注入整个知识库的文件索引，让Claude开局就掌握全局信息

• 所有工作笔记、成果记录、能力沉淀，全通过反向链接自动关联，形成完整的证据网络

最杀的一点，是它直接把工程师最头疼的绩效复盘，变成了一件零成本的事。
季度复盘的时候，你不用再翻遍聊天记录、Git提交、项目文档临时凑材料。
只需要打开对应能力项的笔记，整个季度里所有关联的工作痕迹、成果记录，全通过反向链接自动汇聚好了。
这个证据网络，是你日常工作里自动搭建的，根本不用事后花一秒钟额外整理。

上手门槛低到离谱🤣
克隆模板 → 用Obsidian打开知识库 → 运行claude命令填写你的核心目标，三步就能完成部署。
