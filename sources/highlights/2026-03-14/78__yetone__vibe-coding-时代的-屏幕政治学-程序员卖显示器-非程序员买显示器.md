# Vibe Coding 时代的「屏幕政治学」：程序员卖显示器，非程序员买显示器

## 摘要

**[一个极具张力的观察：Vibe Coding 正在反转「谁是开发者、谁需要屏幕」的既有认知]**

• 程序员因为「只需手机就能编程」开始抛售 Studio Display
• 非程序员群体却因为 Vibe Coding 的「可视化诱惑」开始购入专业显示器搞 Screen Porn
• 反映了工具民主化过程中的身份错位与符号消费

gakki：这是工具平权的讽刺镜像——当 Coding 门槛降低，「看起来像程序员」反而成了一种可购买的社交货币。真正的工程师正在逃离屏幕，而屏幕正在成为新手的入场券。

## 原文

不过我也能理解，在 Vibe Coding 时代，非程序员群体注定也要经历程序员群体早已经历过的 Unix Porn 和 Screen Porn 阶段。

我已经看到很多案例，因为 Vibe Coding，程序员的判断是已经不需要盯着显示器只需要用手机就可以编程了，所以他们开始把自己的 Studio Display 出二手了，但是反观非程序员群体却开始入手 Studio Display 搞 Screen Porn 了
