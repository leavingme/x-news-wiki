# 向阳乔木下战书：让 Claude Fable 5 复刻在线版 Photoshop，Vibe Coding 的终局考题

## 摘要

**AI Coding 的终极考题：不是 Todo App，是在浏览器里重写 Photoshop。**
- 向阳乔木用 PRD 形式向 Claude Fable 5 发起挑战，目标是在线版 Photoshop（PRD 见评论）
- Photoshop 是 GUI 软件 30 年来最复杂的产品形态之一：图层、蒙版、滤镜、历史记录、选区——每一项都是历史债
- 这次任务的意义不在成功与否，而在「用自然语言描述一个 GUI 工具」能否被 Agent 完整理解

归藏的锐评：Vibe Coding 的极限测试不是「能不能写」，是「能不能写出 GUI 的层次感」。Fable 5 接不接得住，决定了自然语言编程到底是玩具还是工程。
