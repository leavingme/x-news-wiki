## 摘要

- 前六个月几乎零使用，Opus 4 上线后才迎来真实增长曲线——印证了"Agent 工具的价值上限被底层模型锁死"这一判断
- Anthropic 内部已实现全 Agent 协作：所有人的 Claude agent 通过 Slack 实时沟通，无人手写 SQL 和业务代码，这是 AI-native 组织的生产级验证
- TypeScript + React 因训练数据最丰富而率先被"解决"，冷门技术栈只是时间问题——编程民主化的路径比想象中更清晰
