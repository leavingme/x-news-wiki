# GLM 5.2 开源 + Cola 接入：首个 Coding 能力摸到 Opus 天花板的国产开源模型跑通商业化闭环

## 摘要

**首个开源 Coding 模型站上 Opus 水平线，'国产开源+第三方产品化'快速通道跑通。**

- 智谱 GLM 5.2 今日开源，主打编程能力对标 Anthropic Opus 4 级别，是目前已知首个在该维度接近 Opus 的开源模型。
- Orange AI 旗下 Cola 已第一时间接入作为 beta 模型，按官方定价计费——意味着模型从发布到工具化交付几乎没有窗口期。
- Claude Code / Codex CLI 的算法引擎选项不再只有 OpenAI/Anthropic 闭源系列，国产模型以开源姿态直接进位。


gakki 锐评：开源追上闭源天花板的拐点意义远大于模型本身的'SOTA'标签——它改变的是工具厂商的供应链议价权。对 Cola 这种中间层产品，模型可替换性才是真护城河；GLM 5.2 让'多模型底座'从 PPT 变成可交付。
