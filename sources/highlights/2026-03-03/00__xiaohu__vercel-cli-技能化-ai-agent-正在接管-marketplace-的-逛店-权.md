# Vercel CLI 技能化：AI Agent 正在接管 Marketplace 的「逛店」权

## 摘要

**Vercel 推出 CLI 技能集成，标志着 AI Agent 从「使用工具」升级为「自主采购服务」。**
- 一行命令 `npx skills add vercel/vercel` 即可让 Agent 访问整个 Marketplace，自动选服务、装配、配置环境变量
- 数据库、Redis、日志、监控——过去工程师的手动点击流程被压缩为一句自然语言指令
- 作者提问「MCP 过时了？」背后是 CLI-native 工具链正在形成一条平行于 MCP 协议的竞争路线

gakki：CLI 技能化是在绕开 MCP 生态另起炉灶，谁先占住「Agent 的默认服务采购入口」，谁就控制了下一代 SaaS 的流量漏斗。这场路线之争的赢家将重写开发者基础设施的分发逻辑。

## 原文

MCP 过时了？？

现在都在陆续推出自己的 CLI 集成

Vercel 也推出了自己的 CLI 技能

把自己的 Marketplace（服务商店）彻底向 AI Agent 开放了，AI 现在可以像人一样去"逛店，挑服务，安装配置"

自动配置服务器...

一行命令装个技能就能搞定一切：

npx skills add vercel/vercel --skill vercel-cli

然后跟 AI 说"加个数据库"，它自己去 Marketplace 找服务、装好、配好环境变量。

数据库、Redis、日志、监控……以前你手动点半天的事，现在一句话搞定。
