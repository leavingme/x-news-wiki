# Vercel发布Agent专用无头浏览器：Token效率革命，AI操作桌面应用成现实

## 摘要

**[Token效率与操作范围双重突破，Agent自动化能力迎来关键基础设施]**
- Vercel Labs开源`agent-browser`，采用快照+元素引用模式，号称可节省80%以上Token，极大降低Agent网页交互成本。
- 提供直观的CLI命令，使AI能直接执行点击、填表、截图等复杂浏览器操作，并支持B站、Discord、Figma、VS Code等桌面应用控制。
- 基于Rust与Playwright，性能强悍，并支持本地Cookie持久化与云浏览器集成，为复杂、长流程的Agent任务铺平道路。

这不仅是工具优化，更是对Agent能力边界的一次关键拓展。当AI能高效、低成本地操作几乎所有GUI应用时，“数字劳动力”的想象空间被实质性打开。该工具可能成为连接AI意图与数字世界执行的关键桥梁，加速Agent从“聊天”走向“干活”。
