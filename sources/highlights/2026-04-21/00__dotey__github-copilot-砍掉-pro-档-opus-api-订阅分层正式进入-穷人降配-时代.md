# GitHub Copilot 砍掉 Pro 档 Opus：API 订阅分层正式进入「穷人降配」时代

## 摘要

**[ GitHub Copilot 套餐悬崖：付费墙从四面八方收紧 ]**

GitHub 这次调整传递了一个清晰信号：大模型 API 的低价获客窗口正在关闭。暂停新订阅 + 砍掉 Pro 档 Opus，等于在说「免费版引流、Pro 版走量、Pro+ 才是亲儿子」的三级火箭已成体系。

关键影响：对中小开发者而言，Claude Opus 在 Copilot Pro 里已成历史，必须加钱上 Pro+ 才能用到旗舰模型。这会加速用户分流——愿意付费的流向官方 API，不愿意的被推向第三方平替方案。

另一层隐忧：GitHub 用「保障服务质量」作理由关停新订阅，本质上是算力成本压力下的平滑过渡措辞。

gakki：Copilot 这波操作本质是 API 成本传导——模型越强、Token 单价越高，平台越需要通过订阅分层把账算清楚。免费版是饵，Pro 版是安慰剂，Pro+ 才是利润中心。中小团队要么接受降配，要么自建 API 路由，别指望 GitHub 会良心发现。

## 原文

GitHub 今天（4 月 20 日）对 Copilot 个人版套餐做了一次大动作调整：暂停 Pro、Pro+、Student 三档套餐的新用户注册，收紧使用额度，并从 Pro 套餐里移除 Claude Opus 模型。只有免费版 Copilot Free 继续开放注册。

核心变化有三点：

1. 新注册暂停。

GitHub 的官方说法是要"优先保障现有付费用户的服务质量"，所以关闭了 Pro、Pro+、Student 三档的新订阅入口。现有用户仍然可以在各档位之间升级或切换。

2. 使用额度收紧。

Pro+ 的额度被重新拉开到 Pro 的 5 倍以上，言下之意就是 Pro 用户经常撞上限的话，官方的建议是掏钱升 Pro+。VS Code 和 Copilot CLI 会在接近额度时弹警告，用量进度条据说也会很快上线。

3. Pro 套餐砍掉 Opus。

Anthropic 最新旗舰 Claude Opus 4.7 只保留给 Pro+ 用户。早些时候官方还预告过，Pro+ 上的旧版 Opus 4.5、4.6 也会一起下架，等于 Pro+ 用户以后只剩 Opus 4.7 这一档 Opus 可选。

为什么现在动刀？背景是过去一年 AI 编程工具跑量太凶，Opus 本身就是市面上最贵的推理模型之一，Copilot 按月固定价卖给海量用户，成本账早就算不过来。GitHub 三月底、四月初已经先后对 Pro+ 动过一次限速和模型下架，这次相当于把刀挥得更彻底，同时把 Pro 这档重新定位成"轻度用户入门款"。

受影响最明显的是两类人。一是想新开 Pro 或 Pro+ 订阅的开发者，窗口暂时关了，只能先用免费版。二是 Pro 订阅者里依赖 Opus 做复杂代码任务的，要么掏钱升 Pro+，要么转到 Claude Code、Cursor、Codex 这些竞品。

GitHub 给了台阶：对这些改动不满意的用户，4 月 20 日到 5 月 20 日之间可以联系官方取消订阅，4 月的费用全额退还。

这是主流 AI 编程工具第一次公开把付费注册窗口关上。
