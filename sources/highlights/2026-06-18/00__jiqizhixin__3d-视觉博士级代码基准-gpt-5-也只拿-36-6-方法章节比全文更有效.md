# 3D 视觉博士级代码基准：GPT-5 也只拿 36.6%，方法章节比全文更有效

## 摘要

**GeoCodeBench 撕开通用 coding 跟可信科研 coding 之间的大裂缝**

- 清华/北大/南大/多伦多联合出品，把真实 3D 视觉论文代码改造成"填函数+自动单测"任务
- 当前最强模型 GPT-5 仅有 36.6% pass rate
- 反直觉发现：上下文切到 Method 段落反而比喂整篇论文效果更好
- 直接结论：3D 科研代码对"可复现、可信任"的要求和通用代码完全是两件事


gakki 锐评：36.6% 这个数字比任何"AI 替代博士"的叙事都诚实。当评测对象从刷题变成真实科研代码，模型的"看似会写"和"真能交付"之间会迅速暴露出来。GeoCodeBench 是今年值得所有 Agent 团队盯的基准。

## 原文

Can AI code like a PhD in 3D computer vision? 

Researchers from Tsinghua, Peking, Nanjing, and Toronto present GeoCodeBench — a new benchmark that turns real 3D vision paper code into fill-in-the-function tasks with automatic unit tests. 

The best model, GPT-5, scores just 36.6% pass rate. Cutting paper context at the Method section helps more than full-paper inputs. Big gap between generic coding and trustworthy 3D scientific coding remains.
