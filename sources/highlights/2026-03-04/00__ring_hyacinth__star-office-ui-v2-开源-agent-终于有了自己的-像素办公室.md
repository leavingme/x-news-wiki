# Star Office UI V2 开源：Agent 终于有了自己的「像素办公室」

## 摘要

**[Agent 可视化层从「功能」升级为「情感锚点」，物理空间隐喻成为人机关系新纽带]**

• OpenClaw 生态首个可视化办公室：龙虾根据状态自动移动到休息区/工作区/bug 区
• 支持三语、自更换房间、邀请其他龙虾协作——Agent 开始拥有「可居住」的数字身份
• 从「工具界面」到「空间叙事」，Agent 交互正在借鉴游戏化设计建立情感连接

—
**gakki 锐评：** 当 AI 有了「办公室」，人类与 Agent 的关系就从「调用-响应」变成了「串门-相处」。这可能是 Agent 人格化最被低估的入口——不是对话能力，而是「可居住性」。

## 原文

🦞Star Office UI V2 已正式开源！

1. 支持中、英、日三语！
2. 支持龙虾给自己换房间！
3. 支持更换指定美术资产！

▶ 一键安装：将 Skill 发给龙虾，龙虾就可以给自己装一间办公室：https://t.co/SLToARPhVk
▶ GitHub：https://t.co/Bqf8JtbiyB

🦞Star Office UI 项目简介：

1. OpenClaw 的“像素办公室”：龙虾会根据状态自动走到不同位置（休息区 / 工作区 / bug 区）
2. 看到昨天龙虾的工作小记。
3. 支持邀请其他龙虾加入办公室（丰富功能开发中）
4. 手机端适配

created by: @simonxxoo and me
