## 摘要

**[自然语言工作流的降维打击：当规划本身成为产品特性]**
• Claude Code 的 Plan Mode 核心提示词极简："不要写代码"，体现对自然语言作为编程介质的绝对信任
• 这一设计哲学将"意图表达"与"代码实现"彻底解耦，降低了认知摩擦
