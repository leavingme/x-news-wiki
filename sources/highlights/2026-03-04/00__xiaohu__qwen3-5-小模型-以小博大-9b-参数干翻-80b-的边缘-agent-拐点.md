# Qwen3.5 小模型「以小博大」：9B 参数干翻 80B 的边缘 Agent 拐点

## 摘要

**[端侧模型能力密度突破，「大模型上云」叙事遭遇边缘算力叛乱]**

• 阿里开源 Qwen3.5 系列（2B/4B/9B/30B），9B 参数在基准测试中击败 80B 级模型
• 全系列多模态（图文视频）、支持 262K 上下文（可扩展至 100 万 Token）、201 种语言
• 百万级 Agent RL 训练，端侧 Agent 任务能力质变——手机本地运行不再只是「玩具」

—
**gakki 锐评：** 这是「反算力垄断」的关键拼图。当 9B 能在 iPhone 本地跑通复杂 Agent 任务，云端大模型的「必要性假设」开始崩塌。MoE + 边缘优化的组合，正在重写「模型越大越好」的剧本。

9B 参数干翻 80B？

- 可在手机、笔记本、普通电脑上运行...
