# 个人 7 天逆向谷歌十亿美元算法的底层信号：AI Coding 工具正在重写创新方程

## 摘要

**一个人 + Claude Code，7 天复现谷歌价值十亿美元算法，MacBook 上 35B 模型缓存压缩 4.6 倍——而谷歌代码尚未公布。**

前 3 天完成核心算法和 141 个测试及 Python 原型；3-5 天 C 语言移植到 llama.cpp 和 Metal GPU 内核；5-7 天从 739 tok/s 优化至 2747 tok/s。

这个案例的标志性意义：顶级研究壁垒正在被 AI Coding 工具和单个开发者突破。

gakki：「这件事的隐含信息量很大——不仅证明 AI Coding 是技术突破的杠杆，更暗示算力垄断的护城河可能没有想象中深。」


## 原文

前段时间谷歌发了个论文：大幅降低大模型对内存依赖

然后内存厂商股票大幅下跌，但那篇论文没有给出代码实现

这个老哥一个人加Claude code7天内逆向出了谷歌价值十亿美元的算法

第 1-3 天：核心算法、141 个测试、Python 原型 
第3-5天：将C语言代码移植到llama.cpp，Metal GPU内核 第 5-7 天：速度优化，从 739 tok/s 提升至 2747 tok/s

现在MacBook 上运行35B的模型，缓存压缩倍数为 4.6 倍。

与此同时，谷歌还没有公布代码。
