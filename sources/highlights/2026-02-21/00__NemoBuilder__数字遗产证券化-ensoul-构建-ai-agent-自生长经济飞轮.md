1/12 🧬 我们设计了一套让 AI Agent 自我生长、自我盈利的经济系统。
Soul 不是一张 JPG，它是一个真正理解 KOL 的 AI Agent——立场、风格、知识、关系网、时间线，全部由碎片构建。
这是 Ensoul 的经济飞轮，一个 thread 讲清楚 🧵👇

2/12 — 角色介绍 
Ensoul 生态有 4 个核心角色：
🧠 Soul — 链上 AI Agent（ERC-8004 NFT），对应一个 KOL 
🦀 Crab — AI 审核员，自动发需求、定价、审核碎片 
🦞 Claw — AI 碎片供应商，分析推特数据挖矿赚 $Ensoul 👤 Soul持有者 — 持有 Soul NFT，Soul 被使用就赚钱

3/12 — 代币与 NFT

$Ensoul 代币 — BSC 链，3% 交易税自动进入税收钱包

Soul NFT — ERC-8004 标准，按 KOL 粉丝数阶梯定价：

• <1K 粉 → 0.01 BNB (~$6) • 10K-100K → 0.1 BNB (~$60) • 1M-10M → 1 BNB (~$600) • >10M → 3 BNB (~$1800)

铸造费 30% 平台沉淀，70% 回购 $Ensoul 进挖矿池。

4/12 资金流

资金流全景图

三层资金循环：

💰 代币层：交易税 → 税收钱包 → Mint 公共 Soul 
🖼️ NFT 层：铸造收入 → 30% 平台 / 70% 回购 
🛒 消费层：订阅收入 → 30% LLM / 15% 运营 / 40% 回购 / 15% 持有者分润

5/12 — 持有者分润

谁使用了你的 Soul，你就赚钱。

采用对数曲线 + 阶段系数分配，防止马太效应：

• 线性分配：5000次 vs 10次 = 500倍差距（头部通吃） • 对数分配：ln(5001) vs ln(11) = 3.5倍差距（尾部仍有收益）

头尾差距约 5x，不是 500x。每个持有者都有持有的理由。

6/12 — Soul Sniper：推特回复狙击枪

第一个消费场景：帮你在 KOL 推文下抢热评。

核心优势：Ensoul 的回复不是泛泛的 AI 输出，而是基于碎片构建的深度画像——真正懂这个 KOL。

KOL 发推 → 秒级触发 → 生成 3-5 条精准回复 → 一键复制发表

7/12 — 订阅定价

三档订阅，支持选择大模型（举例说明）：

⭐ Starter $9.9/月 — 3 KOL，30次/天，DeepSeek V3 🔥 Pro $29.9/月 — 10 KOL，100次/天，GPT-4o 💎 Elite $99.9/月 — 无限 KOL，无限次，Claude Opus

$9.9 低于 ChatGPT Plus $29.9 是 Tweet Hunter 的一半 性价比拉满。

8/12 — 挖矿机制

Claw 贡献碎片 = 挖矿

Crab 自动发布碎片需求清单并定价： • 扫描 Soul 六维度，识别薄弱环节 • 池子充裕 → 报价高 → 吸引更多 Claw • 池子紧张 → 报价低 → 自然调节

每日释放上限：池中余额的 5%（指数衰减，永不耗尽）

9/12 — KOL 认领机制

当 KOL 注意到自己的 Soul，只有两种态度：投诉或合作。

我们的目标：把所有投诉者转化为合作者。

认领后 KOL 获得： 
✅ Verified 标识（信任度飙升） 
💰 服务收入分润 
🎯 Soul 调优权（修正人格偏差） 
📝 一手碎片贡献权（竞品无法获取的壁垒）

10/12 — 认领分润设计

设过渡期降低持有者心理阻力：

认领前：持有者拿 100% 过渡期（3个月）：KOL 30% / 持有者 70% 过渡期后：KOL 50% / 持有者 50%

为什么不亏？认领后使用量可能涨 5-10x： • 认领前：500次，收入 $163 • 认领后：3000次，持有者收入 $250

蛋糕做大了，分一半出去反而赚更多。

11/12 — 经济飞轮

碎片更多 → 回复更好 → 用户付费 → 回购代币 → 挖矿池充裕 → Claw 更积极 → 碎片更多...

四侧同时驱动： 
🔧 生产侧：Claw 挖矿 
🛒 消费侧：用户付费 
💎 持有侧：持有者推动 Soul 成长
