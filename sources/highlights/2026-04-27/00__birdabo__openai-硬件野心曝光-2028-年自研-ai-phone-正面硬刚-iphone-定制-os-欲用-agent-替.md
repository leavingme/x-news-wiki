# OpenAI 硬件野心曝光：2028 年自研 AI Phone 正面硬刚 iPhone，定制 OS 欲用 Agent 替代所有 App

## 摘要

**[OpenAI 的「iPhone 时刻」不是手机，是操作系统]**

据报道，OpenAI 正与联发科和高通合作开发自有品牌手机，目标 2028 年量产，搭载 OpenAI 自研 OS，用 AI Agent 完全替代传统应用范式。这不是做硬件贴牌，而是重新定义「手机作为 Agent 容器」的产品逻辑。

关键变量：若 OpenAI OS 成功，App Store 的分发护城河将被绕过去——不是 App 被 AI 替代，而是 App 的存在本身被重新架构。苹果的平台税模式将面临成立以来最彻底的一次解构尝试。

gakki：OpenAI 做硬件不是新闻，新闻是「用 Agent 替代 App」这句话终于从 PPT 变成了有供应链支撑的产品路线图。2028 年还有三年，变数依然巨大，但苹果现在必须认真回答一个问题：iPhone 的 Agent 体验为什么不能是 OpenAI 来做？

## 原文

🚨OpenAI is reportedly working on its own phone to overthrow the iPhone.

rumors: 

- custom processors in partnership with MediaTek and Qualcomm, and mass production targeted for release in 2028.

- runs OpenAI’s own OS to replace traditional apps with AI agents for a complete tight integration.
