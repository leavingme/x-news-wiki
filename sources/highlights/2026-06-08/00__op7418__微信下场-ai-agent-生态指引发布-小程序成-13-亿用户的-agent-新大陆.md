# 微信下场：AI Agent 生态指引发布，小程序成 13 亿用户的 Agent 新大陆

## 摘要

**[微信正式向 AI Agent 开放小程序生态，超级 App 入口战从概念进入实操阶段]**

- 微信罕见以官方指引形式（《开发者接入微信 AI 生态的指引》）引导小程序开发者接入 AI Agent 能力，核心是让 AI 接管和控制小程序，而非仅作为对话框内的助手
- 这意味着 Agent 在中国拿到了真正的「履约层」——AI 不只能推荐、回答问题，还能直接调用国民级应用完成支付、预约、点单等实际操作
- 短期冲击：小程序开发者需要重新设计 Agent 友好的 API 与状态机；长期看，微信极可能成为中美 Agent 战争中中国侧最厚的护城河
- 归藏判断为「微信 AI 上一项非常重要的功能」——这一定位意味着它不是实验性 feature，而是平台级战略

gakki 锐评：之前所有 Agent 都在抢浏览器、桌面、终端入口，但真正的「国民入口」在中国只有一个。这次不是 OpenAI 的 ChatGPT 模式，是 App 时代的「AI 操作系统」模式——所有做 Agent 的人都该停下来重新评估自己的入口选择。

## 原文

微信终于要加 AI Agent 能力了？

微信发布了《开发者接入微信 AI 生态的指引》这篇文章，引导小程序开发者接入微信 AI 生态，让微信的 AI 控制小程序。
