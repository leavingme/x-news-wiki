# Agent Swarm 的管理艺术：利用蒙特卡罗方法对抗不确定性

## 摘要

**[技术探索]**
- 提出“programming an organization”的概念
- 将 Agent swarm 视为蒙特卡罗方法，通过多会话寻找最优解
- 强调在验证者定律下，重点在于碰撞出可被验证的想法

用算力换确定性。既然单一 Agent 不靠谱，那就用多智能体并行试错，这也是未来提升 AI 可靠性的重要路径。

## 原文

关键词：「programming an organization」。这个 Agent swarm 的用例可以称得上是 vibe management 了

Agent swarm 是种蒙特卡罗。与其对抗 LLM 的不确定性，不如将它利用起来，多开几个 session，期望其中某一个能碰到有用的答案。当然 verifier's law 还在，这里碰的更多是能被验证的想法
