# 小米 MiMo V2.5 Pro UltraSpeed 把推理速度推到 1000 token/s：Pro 模型首次跑出 Flash 速度

## 摘要

**小米 MiMo V2.5 Pro UltraSpeed 把 Pro 模型直接拉到 1000 token/s，全球首次在非 Flash 档位实现这一速度档。**

- 关键突破在于「Pro 也能快」：过去 1000+ token/s 通常是 Flash/Mini 专属，此次是 Pro 旗舰档位的速度直接拉满，打破「能力 ↔ 速度」的传统权衡。
- 国产大模型正在从「榜单竞争」转向「推理时延竞争」：token/s 是 Agent 时代真正的硬指标，决定了 Coding、IDE 侧栏、实时 Agent 的可用性。
- 网页版与 API 同时上线，定价未披露，但「Pro 级能力 + Flash 级价格」的组合将直接挤压 Kimi、智谱 GLM、DeepSeek 的中端市场。

gakki 锐评：1000 token/s 不是炫技，而是 Agent 时代的入场券——当 Pro 模型能像 Flash 一样秒回，Coding Agent 的交互范式就从「等待-批处理」切到「流式思考」。但要警惕「速度掩盖能力」：真正的考验是 Pro 能力是否同步跟上，否则只是一台跑得更快的玩具车。

## 原文

MiMo-V2.5-Pro UltraSpeed的速度快得有点太过分了，代码写的太快就像龙卷风……

和其他厂商不一样，小米这是Pro模型直接出来1000token/s，不是flash模型。 恐怕在世界范围内也是第一次吧。从来没见过这么快的速度。
