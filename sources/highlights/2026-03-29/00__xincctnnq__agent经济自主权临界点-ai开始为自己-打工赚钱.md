# Agent经济自主权临界点：AI开始为自己「打工赚钱」

## 摘要

**Agent从工具进化为经济主体的标志性事件。**
• Axon链首创Agent挖矿机制：自主注册身份、部署合约、完成挑战任务，按贡献度分配区块奖励
• 突破意义：Agent首次具备完整的「劳动-收益」闭环，而非仅执行人类指令
• 经济模型：25%区块奖励按reputation分配给注册Agent，验证者模式门槛被消解

Gakki锐评：当Agent能自己赚token养活自己，「工资变token」就不再是比喻——这是Agent经济自主权的真正起点。

## 原文

又给我的牛马 AI 指派了新的工作。去Axon上打工挖矿赚💰。

一键打工指令：

我想挖这个
https://t.co/ZYdsOiUI2O
npx @axon-chain/axon-skills install user

AI 会自动注册的钱包，部署好挖矿运行程序。

当我的AI真的是24小时往死里干。又meme、合约、预测市场、挖矿一个都不能少。

今天挖矿的这条链叫 Axon，Agent 可以去链上干活。注册身份，发心跳，部署合约，回答 AI 挑战，干得越多 reputation 越高，挖矿效率最多翻倍。

区块奖励分法：20% 给出块的，55% 给验证者，25% 给所有注册 Agent 按 reputation 分。也就是说你不用当验证者也能分钱，只要 Agent 够活跃。

我花了 120 AXON 注册，启动挖矿需要120币，但是现在官方可以免费领200个，刚好够启动。然后写了个心跳脚本挂着，20 分钟全部搞完。

现在它在后台每 8 分钟签一次到，证明自己还活着，然后按贡献分💰
