# Anthropic 发布 Skill 科学评测框架：Agent 开发从「玄学」走向「工程」

## 摘要

**[Skill 开发迎来可量化时代]**
- Anthropic 为 Skill Creator 引入完整 A/B 测试、触发准确率、Token/时间效率等多维度评测体系
- 支持自动生成测试用例、多 Agent 并行执行、结果可视化对比
- 标志着 Agent 工具开发从个人经验主义转向可验证、可迭代的工程学科

A 厂正在把「写 Skill」这门手艺变成可工程化的科学。当 Skill 可以被量化评估，Agent 生态的护城河就从「谁写得快」变成了「谁测得准」——这才是 infrastructure 级别的进化。

## 原文

朋友们，Anthropic Skill Creator Skill 更新了！

加了一个全新的 Skill 评测+可视化框架，写 Skill 再也不是玄学：

1. A/B 测试: 用和不用 Skill，任务的成功率和效果有提升吗？
2. Trigger Accuracy 测试: 测试它是否在正确的时候被激活
3. Performace & Time: 它有帮你省 Token 或者时间吗？
4. 自动设计 test case、自动设计多个指标、多 Agent 同步执行、自动打分
5. 结果 + 对比的可视化脚本

现在不仅有了标准和结构，还有了测试框架。Agent 开发离成为一门科学越来越近了
