# Milady 开源：Agent 加密金融基础设施的「一键部署」范本

## 摘要

**[链上 Agent 从「科学家玩具」走向「可配置服务」]**

• Milady 从 GitHub 端实现完整部署，提供 Chat、角色系统、钱包、知识库、社交五大模块
• 内置 BSC 交易独立安全层，解决 Agent 自动执行链上操作的安全信任问题
• 支持 Discord/Telegram/Slack 多平台接入，降低非技术用户的使用门槛

**gakki 锐评**：当 Agent 开始拥有「钱包」和「交易权限」，它就不再是工具，而是经济主体——这类基础设施的成熟，是 Agent 经济自主权的前提条件。

## 原文

今天从github端部署了Milady，整理一篇Milady基础功能介绍，希望对大家理解产品有帮助。

核心功能模块
1. Chat：沟通与执行的入口
支持与AI智能体进行自然语言交互，是用户与Agent沟通的入口，也是Agent执行任务的起点。

2. 角色系统：让每个Agent独一无二
支持自定义专属Agent：

昵称：为Agent设定个性化名称

形象：定制Agent外观

性格：配置性格特征，让交互更具个性

3. 钱包：Agent的加密金融基础设施
内置加密钱包，支持链上资产管理、交易、执行等操作：

资产管理：查看和管理链上资产

交易执行：Agent可代表用户完成链上操作

BSC交易独立安全层：针对BSC链交易增加独立安全验证，保障资产安全

4. 知识库：让Agent更聪明
支持多种格式文档上传，增强Agent知识储备：

格式支持：PDF、Markdown等多种文档格式

RAG增强：基于检索增强生成，提升回答准确性

本地部署：数据可本地化部署，保障隐私安全

5. 社交：让Agent连接世界
支持主流社交平台接入：Discord、Telegram、Slack、WhatsApp、飞书等

6. 内置应用/游戏商店：Agent的娱乐与生产力中心
内置多款专为Agent设计的应用：

Babylon：AI智能体参与的社交预测游戏，Agent在游戏中参与预测市场交易，驱动合成资产市场

HyperScape：AI智能体参与的RPG游戏，支持观看、指挥或接管Agent操作

更多agent盈利类应用及游戏筹备中

7. 设置：精细化权限控制
灵活配置Agent的运行权限和行为：

模型选择：自由选择AI模型

权限控制：
系统权限：控制Agent对操作系统的访问级别
浏览器控制：允许Agent自主浏览网页、搜索信息
电脑控制：允许Agent执行文件操作、打开应用等本地任务

Agent运行模式：
半自动模式：Agent执行需人工确认
全自动模式：Agent在授权范围内自主决策和执行

钱包私钥权限：精细化控制Agent对钱包的访问和交易权限

8. 高级功能：为开发者和进阶用户打造

深度配置能力，满足专业需求：

插件分类管理：插件分类展示，方便查找和使用

Skills上传：支持上传自定义技能，扩展Agent能力

模型微调：对模型进行针对性优化

运行日志追踪：查看Agent运行日志，便于问题追踪
