# Claude Agent SDK 生态爆发：从开发框架到产品基础设施的跃迁

## 摘要

**Anthropic 的工具链正在成为创业公司的默认基建。**

• 基于 Claude Agent SDK 的工具产品快速涌现：笔记软件 Moss、Markdown 任务管理 Moment、多 Agent 开发工具 SUPERSET
• 反映开发者社区对 Claude 产品力的认可，Agent 基础设施层竞争格局初现
• SDK 生态成熟度成为平台粘性的关键变量

---

**gakki 锐评：** 当第三方开发者开始用你的产品造产品，你就从工具变成了平台。Claude Code 正在经历从「好用的编辑器」到「Agent 时代的 Rails」的质变。

## 原文

发现用Claude Agent SDK打造的工具产品越来越多。

一晚上随便测了几个AI工具，全都是基于它搞的。

有笔记软件Moss，有一个Markdown管理开发任务的Moment，有专门为多 Agent开发的工具SUPERSET。

Anthropic公司不管怎么说，产品力是相当可以。 https://t.co/GOAvcB6B2c
