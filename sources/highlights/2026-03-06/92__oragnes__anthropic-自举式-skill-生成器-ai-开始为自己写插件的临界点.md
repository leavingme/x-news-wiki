# Anthropic「自举式」Skill 生成器：AI 开始为自己写插件的临界点

## 摘要

**AI 插件生产的自动化闭环已打通。**

• Anthropic 官方 skill-creator 实现「用 AI 写 AI 能力模块」，开发者只需自然语言描述需求即可生成完整 Skill 代码、目录结构与测试用例
• 标志着 Agent 工具链从「人工组装」向「自举生成」演进，大幅降低生态扩展门槛
• 预示 AI 的 App Store 时代加速到来：技能生产本身正在被自动化

## 原文

Anthropic 官方 GitHub 里悄悄藏了一个“王炸”工具。

AI 正在学会自己给自己写插件！🤯

这个官方项目叫 skill-creator，代表了 AI Agent 架构的一个重要分水岭。

带你一分钟看懂这个趋势（建议收藏，AI 的 App Store 时代来了）👇

🧵到底什么是 skill-creator？

这是 Anthropic 官方 Skills 库里的一个特定 Skill。

它的唯一作用就是：👉 让 AI 自动帮你创建新的 Skill。

本质上，这就是用 AI 来写 AI 的能力模块，机器实现了自我武装。

它是怎么工作的？

以前：开发者苦哈哈地手写代码和 Prompt 组装工具。

现在：你只需要对 skill-creator 说一句：

“我想做一个自动生成 Marketing 报告的 Skill”。

它就会自动帮你生成完整的代码脚本、目录结构、使用说明和测试用例。

AI 自动生成 AI 插件，闭环了。

❌ 过去：Prompt → AI

✅ 现在：Prompt + Skills → AI Agent

🚀 未来：AI = 大模型 + Skills 生态
