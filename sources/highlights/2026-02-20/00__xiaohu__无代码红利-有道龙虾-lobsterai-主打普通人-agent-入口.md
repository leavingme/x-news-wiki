# 无代码红利：有道龙虾（LobsterAI）主打普通人 Agent 入口

## 摘要

**[核心逻辑]** 通过封装沙盒 APP 降低 Agent 使用门槛，旨在解决 OpenClaw 命令行操作对普通用户的阻碍。
- 三分钟零门槛安装，无需 Python 环境或终端操作。
- 独立沙盒运行机制，在保证安全的前提下实现 Agent 功能大众化。
- 开源属性确保了技术的透明度与社区的可扩展性。

[gakki 的锐评]：Agent 的‘施乐时刻’。当强大的工具被装进漂亮、好点的窗口里，它才真正开始改变世界，而不仅仅是极客的玩具。

## 原文

最近后台好多人问我: OpenClaw 这个听着确实牛，但我不会写代码，终端操作太麻烦用不了怎么办?

网易有道刚搞了"中国版 OpenClaw"：LobsterAI（有道龙虾）

专门给普通人设计的

把龙虾封装成了一个APP，安装极其方便而且独立沙盒运行，很安全，对普通人来说很友好

而且是开源的！

体验了下，确实是很好上手
