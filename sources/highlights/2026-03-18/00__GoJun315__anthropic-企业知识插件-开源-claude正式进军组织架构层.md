# Anthropic「企业知识插件」开源：Claude正式进军组织架构层

## 摘要

**[Claude从个人助手向企业知识基础设施跃迁的信号]**
• 官方发布11个垂直岗位插件（产品/法务/营销/数据分析），直接对标企业内部员工角色定义
• 通过MCP打通Slack/Notion/Jira等主流协作工具，完成从「单点工具」到「工作流嵌入」的关键一跃
• 开源策略暗示Anthropic正在构建「Claude Certified」生态标准，与Cursor/Codex形成差异化竞争壁垒

Anthropic的企业化路径比OpenAI更激进——不是卖API，而是直接卖「数字员工」的组织能力。但企业真正买单的前提是权限治理和审计追踪的补齐，目前仍缺一块。

## 原文

Anthropic 官方开源了一套专为 Claude Cowork 打造的插件库：Knowledge Work Plugins，也兼容 Claude Code 使用。

它能将 Claude 变成精通我们团队工具和流程的「专业员工」。

涵盖产品经理、数据分析、法务、营销等 11 个专属岗位插件，并可通过 MCP 无缝打通了 Slack、Notion、Jira 等企业常用工具。

GitHub：https://t.co/AKEMXaTHhO

如果我们正在用 Claude 辅助工作，或者想在企业工作流程落地，官方提供的这套插件，值得收藏研究下。
