# MetaClaw开源：Agent自我迭代的「睡眠学习」范式落地

## 摘要

**Agent终于学会「睡觉长脑子」了。**

- AIMING Lab推出的AutoClaw/MetaClaw在OpenClaw基础上实现自动技能提取与持续进化
- 核心突破：将日常对话转化为学习信号，闲置时云端RL训练，无需本地GPU
- 三种模式覆盖从轻量到激进的不同场景，v0.3.2新增多爪支持与OpenRouter

这套「透明代理层」架构可能是OpenClaw生态的下一个关键分叉点——不是更好的工具，而是更聪明的工具制造机。

## 原文

兄弟们！来自 AIMING Lab的自我迭代的OpenClaw 有点东西啊！

2两半狂揽3.3 k Star 🌟并且还在持续增长！

这项目基于OpenClaw 迭代出一个进化成“会自己变聪明”的版本 ——AutoClaw& MetaClaw 正式开源！🦞🧬

Just talk to your agent，它就自动学习、提取 Skills、持续进化！

核心功能：

- 透明代理层在OpenClaw / CoPaw / IronClaw / NanoClaw 等前面

- 每一次正常聊天都变成学习信号 → 自动注入最相关 Skills + 会话后自动总结新 Skill

- 三种模式：skills_only（轻量） / rl（立即训练） / MadMax（默认！睡觉、开会、闲置时偷偷云端 RL 训练，永不打扰你）

- 完全不用本地 GPU！用 Tinker / MinT 云端 LoRA 就行

- v0.3.2 新增多爪支持 + OpenRouter

两个命令就搞定：
bash metaclaw setup metaclaw start 

GitHub 直达：https://t.co/gF6zx9cYbU
