# OpenAI 押注"超级应用"：ChatGPT 启动史上最大改版，整合编码 + Agent

## 摘要

【OpenAI 终于把 ChatGPT 当成入口而非工具来卖】
- 英国金融时报披露：OpenAI 正在准备 ChatGPT 推出以来规模最大的一次改版
- 战略方向：从对话工具转型为"超级应用"，整合编码工具 + AI Agent
- 高管层认为这一路径能创造更多收入，与 Atlas 浏览器、Codex 平台战略一脉相承
- 与 Anthropic Claude Code 生态、Google Gemini 桌面端形成正面对撞

 gakki 锐评：当 ChatGPT 同时承担 IDE、Agent 平台、浏览器三层角色，用户究竟是用户还是产品？OpenAI 这次"超级应用"赌的不是模型领先，而是入口垄断。

## 原文

据英国金融时报：OpenAI正在准备对ChatGPT进行其推出以来规模最大的一次改版。

该公司计划将ChatGPT转型为结合编码工具和人工智能体的“超级应用”，并添加高管们认为能够创造更多收入的产品。
