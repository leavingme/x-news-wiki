# Codex 内置浏览器变无限画板：Excalidraw 草图直出图，免费不限量

## 摘要

**[Codex 的浏览器+生图能力被严重低估：草图→出图全链路打通]**

要点：
- Codex 打开 Excalidraw，画布上随手画草稿标注，Codex 基于草图直接出图
- 内置 Imagen 模型，几乎不限量；对比按次收费的生图工具 ≈ 免费无限画板
- 配合编程能力可批量生图、统一风格、自动后期，出图一站式完成


锐评：Codex 一直被当作 CLI 编码工具，但内置浏览器+生图能力正在让它变成万能画板。关键问题是这套玩法会不会被 OpenAI 收紧——免费无限的红利窗口可能不会一直开着，先用先赚。
