# Kimi Work 撞脸 Codex：国产 Agent 赛道进入「像素级对标」阶段

## 摘要

**[一句话核心判断]**

余温发现 Kimi Work 与 Codex 在产品界面、交互逻辑上高度相似——项目管理、浏览器自动化、定时任务后台运行等功能一应俱全，被形容为「国产 Codex」。

**信息增量：**
- 产品形态对标已从「功能借鉴」演进到「交互范式借鉴」，说明 Agent 工具的产品形态正在收敛
- Kimi K2 模型被评价为国内日常体验最接近 Claude 的模型，有实质性技术积累
- 开发者对「国产替代」的情绪从质疑转向「希望做好」，用户开始认真评估而非单纯嘲讽

**影响：**
- 国内 Agent 工具竞争从「功能堆砌」进入「体验一致性」比拼
- Codex 确立的产品范式（Project + 自动化 + 后台任务）已成行业默认标准
- 独立开发者和小型团队将更容易在国内生态中找到 Codex 的平替选项


gakki：Kimi Work 像素级对标 Codex，本质上是在验证「Codex 的产品路径已经收敛到可以被复刻的成熟度」——但复刻的边界在于：你能抄界面，能抄交互，但抄不走 Codex 与 GPT 模型深度集成的工作流积累。
