# xAI 解散并入 SpaceX，Anthropic 吞下 22 万张 GPU 算力：AI 算力格局的「垂直整合」时代来了

## 摘要

**马斯克把 xAI 从独立公司降级为 SpaceX 的 AI 部门，算力与火箭共用一个资本结构。**

- xAI 更名 SpaceXAI，不再独立运营，Colossus 1 数据中心 22 万张 NVIDIA GPU 直接挂靠 SpaceX 资产负债表
- Anthropic 宣布与 SpaceX 达成算力租赁合作，同日 Claude Code 速率翻倍、Pro/Max 高峰降速取消、Opus API 限速大幅上调
- 这意味着 Anthropic 的算力瓶颈正在通过「租而非建」的方式快速缓解，而马斯克则把 AI 算力变成了航天基建的附属资产

gakki 锐评：xAI 的「解散」不是失败，是马斯克终于想清楚了——AI 公司独立烧钱不如做 SpaceX 的算力飞轮。Anthropic 拿到算力弹药后立刻解除限速，说明此前的「克制」本质是供给不足而非产品哲学。

## 原文

马斯克宣布：xAI 解散，

将不再作为独立公司存在

直接并入Space，它将直接更名为 SpaceXAI

同时Anthropic 宣布与 SpaceX 达成算力合作，Colossus 1 数据中心超过 22 万张 NVIDIA GPU ，租给 Anthropic 使用。

同时Anthropic宣布：

Claude Code 5小时速率翻倍
Pro 和 Max 用户的Claude Code高峰时段降速取消
Opus 系列 API 速率限制大幅上调
