# NYSE 申请代币化证券交易：传统金融基础设施开始向链上迁移，AI Agent 的金融原语即将就位

## 摘要

**传统金融的结算层正在被重写，而 Agent 经济还没准备好接住它。**

- NYSE 向 SEC 提交代币化证券提案，允许股票和 ETF 以代币形式在 DTC 三年试点中交易，同一代码、同一权利、T+1 结算
- 这不是 DeFi 的胜利，而是 TradFi 主动把链上结算纳入自己的基础设施——中心化机构在「收编」区块链
- 对 Agent 经济的信号：当证券变成可编程资产，AI Agent 的交易对象从加密原生扩展到全球资本市场

gakki 锐评：Agent 经济讨论了一年的「链上金融原语」，真正推动落地的不是协议创新，而是 NYSE 一张申请表。基础设施的权力中心从未转移。

## 原文

🚨NYSE FILES RULE CHANGE TO ENABLE TOKENIZED SECURITIES TRADING

The NYSE filed a proposal with the SEC to allow tokenized versions of eligible equities and ETFs to trade under DTC’s three-year tokenization pilot.

They would have the same ticker, same rights, and still settle in 1 day (T+1).
