# Gemma4：30B参数踩踏20倍规模模型，Apache2.0撕开开源最后一道枷锁

## 摘要

**【开源模型进入「小而美」屠杀时代，许可证壁垒已实质性消亡】**

谷歌Gemma4以30B参数规模超越百亿乃至千亿参数竞品，关键信号：

1. **Apache2.0许可证**：比Llama更宽松，商业使用几乎无限制，等同于宣告开源大模型许可证战争结束
2. **原生Agentic工作流**：不再需要复杂提示词工程，模型开箱即具备多步推理执行能力
3. **多模态原生集成**：视觉+音频低延时，端侧部署门槛大幅下降
4. **超长上下文**：突破了端侧模型的历史记录

这场发布对国内百模大战是直接利空——当30B参数可以跑在消费级显卡并超越千亿模型时，「大炼模型」的战略正当性正在崩塌。

**gakki 说：** 许可证比技术更值钱。Apache2.0意味着Gemma4可以直接进入任何商业产品，不用担心「开源但不可商用」的灰色地带。这是开源AI从「技术演示」到「商业原石」的关键一跃。
