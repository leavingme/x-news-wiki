# Claude Code源码泄露背后：44个功能开关正在打开一个被低估的竞争维度

## 摘要

**[Claude Code的「水面下冰山」比水面上的更值钱]**

泄露源码揭示44个功能开关和20+未发布特性：后台Agent 24/7运行、多Claude调度、Cron定时调度、完整语音指令模式、Playwright浏览器控制、Agent休眠自恢复、跨会话持久记忆。这些特性一旦解锁，Claude Code将从「编程工具」进化为「可定制的AI工作站」。

竞争信号：这些能力与OpenAI Codex、Cursor的功能矩阵高度重叠，但Claude的工程克制哲学（见源码结构）让发布节奏远慢于竞争对手。

**gakki：** 功能开关泄露比功能本身更值得分析——它暴露了Anthropic的产品路线图优先级。持久记忆和语音模式是今年最确定的升级方向，现在知道Claude Code用户可以期待什么。

## 原文

Claude Code 泄露的源码里藏着 44 个功能开关和 20 多个未发布特性：

  - 后台智能体 24/7 不间断运行
  - 一个 Claude 调度多个子 Claude 协同工作
  - 智能体的 Cron 定时调度
  - 完整的语音指令模式
  - 通过 Playwright 实现的真正浏览器控制
  - 智能体可休眠和自恢复
  - 跨会话持久记忆
