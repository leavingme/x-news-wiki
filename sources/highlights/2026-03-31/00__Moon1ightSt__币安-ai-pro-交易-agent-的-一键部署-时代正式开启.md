# 币安 AI Pro：交易 Agent 的「一键部署」时代正式开启

## 摘要

**[交易 Agent 从概念验证走向产品化落地]**
• 整合 ChatGPT、Claude、Qwen、Kimi、MiniMax 等主流模型，提供开箱即用的交易 Agent
• 子账户隔离设计解决安全顾虑，API Key 无提现权限，降低用户信任门槛
• 覆盖现货、永续合约、杠杆借贷等完整交易链路，实现「策略→执行」闭环

gakki 锐评：币安这一步把交易 Agent 的门槛降到了「零代码」水平，但真正的护城河不在功能集成，而在策略市场的沉淀与社区生态的活跃度。

## 原文

亲爱的，币安这次动作真快，AI 叙事火起来就直接跟上了，推出了一个更偏实用型的产品，叫做Binance Ai Pro @binancezh 

月光仔细研究了功能，帮大家把重点整理出来

新产品的本质是把交易、策略和大模型能力打包成一个可以直接用的 AI Agent

先说核心体验，它是走一键激活+云端自动搭建的路线

开通之后，模型接入、账户配置这些流程基本都帮你处理好了，把上手门槛降低

在模型层面，主流的基本都给你接好了，比如 ChatGPT、Claude、Qwen、Kimi、MiniMax 等，同时也整合了币安自身的各类交易相关能力和第三方 AI skills

月光觉得这一点比较关键，相当于不是单纯聊天，而是可以直接调用交易、数据、链上查询这些动作

安全这块也做了隔离设计。系统会自动创建一个 AI 子账户，并绑定 API Key，而且这个 Key 没有提现和转账权限

你需要手动把资金从主账户划转过去，让 AI 在子账户里执行策略，这样即便策略出问题，也不会影响主账户资金

功能上，币安的AI工具覆盖了比较完整的一套交易辅助能力，比如现货和永续合约下单，杠杆借贷相关操作，市场行情分析，链上地址资产分布查询，自定义策略执行

入口也不复杂，安卓 App 和官网都可以直接激活

目前是分批开放，不是所有用户都能第一时间用到，快看看你有没有吧！

整体来看，这个产品更像是币安在把AI + 交易执行这件事产品化，从工具走向半自动化策略助手

如果你本身就在用量化、或者对 AI 辅助交易有兴趣，@binance 的 AI Agent 产品值得关注一下哦！
