# Claude Code 上线 Computer Use：从「写代码」到「操电脑」的质变

## 摘要

**[Anthropic 补全 Agent 闭环最后一环：代码→编译→运行→验证全自动化]**

• Computer use 功能登陆 Claude Code，支持自动启动 App、点击 UI、找 bug、修复、验证
• 覆盖 SwiftUI、Electron 等无 CLI 的 GUI 工具，突破纯代码 Agent 的能力边界
• 与 Dispatch 联动实现「手机派任务→电脑自动执行」的跨端工作流

这是 Claude Code 从「编码助手」向「数字员工」的关键跃迁。GUI 自动化意味着 Agent 可以接管任何软件——不只是有 API 的，而是**所有软件**。

**边界**：macOS 专属、Pro/Max 付费墙，Windows/Linux 用户仍需等待。

## 原文

Computer use 功能现在登陆 Claude Code 

Claude Code现在能自己操控电脑了

以前 Claude Code 的写完代码交给你处理，

现在它可以在写代码、编译后，自己启动 App、点击 UI、找 bug、修复、验证，一句话 全搞定。

全程不用你动手。

SwiftUI、Electron这些没有 CLI 的 GUI 工具，也都支持。

Pro / Max 用户现在就能用

macOS 专属，需要更新到 Claude Code v2.1.85+
