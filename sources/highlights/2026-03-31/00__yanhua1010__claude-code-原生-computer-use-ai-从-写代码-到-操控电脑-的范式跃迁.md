# Claude Code 原生 Computer Use：AI 从「写代码」到「操控电脑」的范式跃迁

## 摘要

**[Claude Code 正式引入 Computer Use，标志着 AI Agent 从代码生成走向端到端任务执行]**

• **场景突破**：原生应用验证（自动编译 Swift/macOS 应用并截图验证）、端到端 UI 测试（无需 Playwright 配置）、视觉 Bug 调试（自动调整窗口复现错位）、操控无 API 工具（设计软件/系统设置）
• **门槛限制**：仅限 macOS + Claude Pro/Max + v2.1.85+，通过 /mcp 启用 computer-use server
• **信号解读**：Anthropic 正将 Claude Code 从「代码助手」重构为「通用计算机操控 Agent」，与 OpenAI 的 Codex 形成差异化竞争

Claude Code 的 Computer Use 不是功能叠加，而是产品定位的根本转向——从「帮你写」到「替你干」。Anthropic 选择 macOS 独占是聪明的：抓住开发者主力阵地，建立生态粘性。但 Pro/Max 门槛意味着这仍是「精英工具」，真正的平民化还需时日。

## 原文

🚀 Claude 可以在 CLI 里直接操控你的电脑了！

Claude Code 正式引入了 Computer use。
几个核心应用场景：
✅ 原生应用验证：让 Claude 自动编译 Swift 或 macOS 应用、启动它、点击各个按钮并截图验证是否崩溃。
✅ 端到端 UI 测试：控制 iOS 模拟器或本地 Electron App 跑通整个注册流程，完全不需要 Playwright 等繁琐的测试配置。
✅ 视觉 Bug 调试：让 Claude 自动调整窗口大小复现 UI 错位，修改 CSS 代码并直接验证修复效果。
✅ 操控无 API 的工具：直接操作设计软件、系统设置或专有工具。

⚠️ 使用门槛：
 1. 仅限 macOS 平台。
 2. 需订阅 Claude Pro 或 Max 计划。
 3. 使用 Claude Code v2.1.85+ 版本，运行 /mcp 并启用 computer-use mcp server。
