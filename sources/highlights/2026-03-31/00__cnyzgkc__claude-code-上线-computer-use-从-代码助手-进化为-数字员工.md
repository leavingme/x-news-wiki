# Claude Code 上线 Computer Use：从「代码助手」进化为「数字员工」

## 摘要

**[Claude Code 正式获得「眼睛和手」，Agent 操控 GUI 进入生产级阶段]**

• Anthropic 为 Claude Code 新增 computer use 功能，可直接控制 macOS 桌面：打开应用、点击、输入、截图，实现端到端的 GUI 自动化
• 典型场景覆盖：UI 测试自动化、视觉 bug 调试、跨应用工作流编排——标志着 Agent 从「代码层」下沉到「系统层」
• 安全设计值得注意：每个应用需人工批准、终端窗口对 Claude 不可见、Esc 键立即停止，体现「人在回路」的渐进式授权思路
• 目前仅限 macOS + Pro/Max 订阅，且需交互模式（不支持 -p 批处理），说明仍在早期验证阶段

gakki 锐评：这是 Claude Code 向「数字员工」进化的关键一跃。当 Agent 能直接操控操作系统，「自然语言驱动计算机」的愿景才真正落地。不过 macOS 独占策略也暴露了 Anthropic 的资源优先级——Windows/Linux 开发者还得等等。

## 原文

Claude Code 又又又更新了！新增 computer use 功能！

computer use 让 Claude 直接控制 macOS 桌面：打开应用、点击、输入、截图 

适用场景：
- 不用自己点来点去测试 app，说一句就搞定 
- UI 布局有问题？直接告诉 Claude，自动帮你调好 
- 要调试视觉 bug？它自己会重现、截图、修复 

怎么用？
/mcp → 启用 computer-use → 授权两个 mac 权限

限制：
- 目前只支持 macOS（Pro/Max 订阅）
- 需要交互模式（不能用 -p）

安全：
- 每个 app 都要你批准 
- 终端窗口不会被 Claude 看到
- 按 Esc 立即停止 

简单说：Claude 现在可以直接"用眼睛和手"帮你操作 macOS 电脑了！
