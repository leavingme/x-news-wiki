# OpenAI 78%扩招背后：API卖不动，逼出「布道师」新物种

## 摘要

**[大模型API的ROI困境催生新型服务岗位]**

OpenAI宣布年底前社招扩招78%，专门设立Technical Ambassador——本质是替代咨询公司、帮客户找到ROI的那批人。谁先帮客户跑通工作流，谁就锁定了未来几年的API消费。

Gimlet Labs获$92M融资则揭示了另一面：推理优化/算力中转赛道正当时，硅基流动、OpenRouter都是同类，国内很网红。

## 原文

今天的重磅消息，OpenAI决定不躺了，给HR定了个KPI，年底之前大幅扩招增员78%，以社招为主，技术人员为主。

一个有意思的岗位类别正在兴起 Technical Ambassadorship ，我问了一下哈吉米，它说最贴切的中文对应：阿里云/腾讯云/华为云的行业解决方案架构师，或者字节/百度大模型团队的AI 技术布道师（Evangelist）。

为什么 OpenAI 要专门设这个岗

大模型 API 不像传统 SaaS，客户买了不会自己用，需要有人手把手帮他们找到 ROI。

这个岗位本质上是在替代传统咨询公司的角色，但更便宜、更快、更懂产品。

谁能帮客户先跑通，谁就锁定了这个客户未来几年的 API 消费。

...

Gimlet Labs 完成 $92M 融资  ，这公司是做AI 推理优化的。换接地气（咱国人）能理解的术语说就是做中转站的，闲鱼上随便一搜一大堆的这种。现在国内比较高调融资的是 硅基流动 , 很多大模型榜单上吹的openrouter也是这类公司，这赛道现在很网红，很火.可以理解成是算力的二房东，租卡，然后按API和token卖钱，暴利。

...

Air Street Capital 这家欧洲基金刚募集了很多钱，这部分资金主要投向AI基础设施 和 AI应用方向，想要卖股权的FA机构和初创公司可以联系看看。公司的成功案例很多，在别人国家，知名程度算是不亚于咱国内的红杉资本这样水平吧。
