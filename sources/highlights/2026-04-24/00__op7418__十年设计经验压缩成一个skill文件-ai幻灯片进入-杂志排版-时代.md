# 十年设计经验压缩成一个Skill文件：AI幻灯片进入「杂志排版」时代

## 摘要

**一句话核心判断**：做PPT的Skill终于开始把「审美」当成核心壁垒，而非功能。

后续要点：
- 该Claude Skill先通过6个问题访谈受众、时长、素材、图片、风格，再规划节奏后生成，流程本身即专业设计方法论的压缩
- 内置10种编辑级版式（封面、大数字、引用线、流程对比、Before/After等），跳脱PowerPoint模板逻辑
- 5套精选主题包，强调「AI生成的内容不该看起来像AI生成的」——这是一个审美自觉的宣言
- 隐含信号：Prompt Engineering的下一个竞技场不是「生成什么」，而是「生成得像谁」

gakki的锐评：设计生产力的真正杠杆不是工具迭代，而是把审美判断流程化、可复用化。这个Skill做的是把「品味」变成管道，而非把「功能」变成管道——两者之间有本质差别。

## 原文

I built a Claude Skill that makes slides look like magazines, not PowerPoint. 

Ten years of design experience, compressed into one skill file.

The world hasn't woken up to the fact that AI-generated slides don't have to look AI-generated.

- Claude interviews you with 6 questions first (audience, duration, source, images, aesthetic), plans the rhythm, then writes the deck
- 10 editorial layouts — cover, big numbers, pull quotes, pipelines, before/after, image grids
- 5 curated themes, no custom hex allowed — tight constraints make stable style
- Live WebGL fluid backgrounds, smooth page transitions, responsive on any screen
- Output is a single HTML file — no build, no server, no cloud
- Same-filename image swap: drop a new file with the same name, refresh, done. Zero HTML editing

I asked Claude to make a 6-page demo to showcase all 5 themes.
