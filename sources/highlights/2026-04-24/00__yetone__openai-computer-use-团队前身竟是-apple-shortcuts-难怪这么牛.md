# OpenAI Computer Use 团队前身竟是 Apple Shortcuts：难怪这么牛

## 摘要

**Apple Shortcuts 的基因决定了 Computer Use 的交互范式。**

OpenAI Computer Use 团队被爆其核心成员来自 Apple 的 Shortcuts 团队。Shortcuts 的本质就是「用自然语言驱动系统操作」——这与 Computer Use 的 Agent 操控电脑逻辑高度同构。难怪 Computer Use 能做到隐藏鼠标下的精准操作点击。

这揭示了一个重要规律：顶级 AI Agent 产品背后，往往是已有多年特定领域人机交互积累的团队，而非纯 LLM 算法背景。交互基因比模型基因更重要。


**gakki**：Computer Use 的成功不是 OpenAI 的胜利，是 Apple 人机交互遗产的胜利。这个信号值得国内大厂反思——与其砸钱买卡，不如收购一个有深度交互积累的团队。

## 原文

昨天听 CEO 说，原来 OpenAI 的 Computer Use 团队以前是在 Apple 做 Shortcuts 的。怪不得这么牛逼。
