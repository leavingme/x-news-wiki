# Sequoia 最新研判：AI Agent 的杠杆已超过 100%，确定性退位给「可能性」

## 摘要

**[顶级 VC 重新定义 AI 时代的工作逻辑：杠杆 > 确定]**

Sequoia 红杉资本提出五大投资趋势，第一条核心论点：工作模式正在从「任务100%确定但杠杆极低」转向「结果不确定但杠杆超过 100% 甚至 1000%」。以销售为例，AI Agent 可以给每个客户配备独立 Agent，追踪动态、发现扩客机会。

这意味着：AI 时代的核心竞争力不再是「把事情做对」，而是「选择让哪些事情被放大」。

gakki：「Sequoia 这番话的潜台词是——还在追求确定性的团队正在主动选择退出牌桌。2026 年的 AI 竞争，判断做什么的权重已经超过了怎么做。」

## 原文

很多人說AI賺不到錢泡沫要破，大筆投資在局中的人則認為AI還在早期局數，未來的機會非常大。

Sequoia 紅杉資本相信 AI 革命將是一場不亞於工業革命的重大轉型。將帶來「10兆美元的機會」，這場認知革命正在重塑投資格局。以下是他們觀察到的五大投資趨勢和未來關鍵投資主題。

#五大投資趨勢

趨勢一： #槓桿勝過確定性 Leverage Over Uncertainty

工作模式正在改變：「從任務槓桿效果很小但結果100%確定，轉向任務槓桿效果超過100%，但結果的具體呈現方式變得不那麼確定。」

以銷售為例，傳統上需要親自管理客戶帳戶，但在AI agent未來，「你可以使用像Rocks這樣的工具，擁有數百個AI agent，為每個客戶配一個AI agent，追蹤進展、觀察變化，顯示重新接觸和擴大客戶關係的機會。」雖然「AI agent的做事方式不會跟你完全一樣」，但能提供「100%以上的槓桿效果，甚至可能是1000%」。

趨勢二：#真實世界驗證 Real-World Validation

測量標準已從學術基準轉向真實世界表現。「今天，如果你想證明卓越，你要在真實世界中證明。」

Expo想證明他們的AI是「世界上最好的AI駭客」，「他們沒有只在學術基準上證明，而是走進真實世界，在Hacker One平台上與全球駭客競爭尋找漏洞」，最終「成為世界第一的駭客。」

趨勢三： #強化學習崛起 Reinforcement Learning

「AI產業談論強化學習已經很長時間了。我們看到它在過去一年登上了中心舞台。」不只大型推理實驗室受益，「許多投資組合公司也從中受益。以Reflection為例，它使用強化學習來訓練一些在程式設計領域最好的開源模型。」

趨勢四： #物理世界中的AI AI in the Physical World

「AI在物理世界真正活躍起來。這不只是人形機器人，還包括用AI創造流程和製造硬體。」以Nominal為例，「它使用AI來加速硬體製造流程，實際上也使用AI在部署到現場後進行品質保證。」

趨勢五： #運算成為生產函數  Compute as Production Function

「新的生產函數是運算，也就是每個知識工作者的浮點運算次數。」投資組合公司「預測每個知識工作者的浮點運算次數，至少會增加10倍」，在樂觀情況下，「我們看到未來每個知識工作者的浮點運算消費，可能增加1000倍或10000倍。」

#五大投資主題

#持久記憶 Persistent Memory

「持久記憶至少有兩個意思：長期記憶和AI身份的持續性，agent在一段時間內保持自己的個性和風格。」這對AI進入更多工作職能至關重要。「我們注意到在持久記憶領域沒有類似規模定律的東西，這個問題還沒有被解決，這是未來的重大機會。」

#通訊協議 Communication Protocols
雖然MCP引起關注，但「讓我們回想一下網際網路革命。TCP/IP不是終點線，而是起跑槍。」AI之間的無縫通訊將帶來重大應用，例如購物領域，「AI將能夠完成整個流程，找到最好的價格，為你執行購買並完成所有操作。」

#AI語音 AI Voice
「我認為一年後AI影片可能會到來，但AI語音就是現在。」因為「不僅保真度提高到日常使用的程度，而且延遲降低了，你可以與AI語音進行即時對話。」在企業領域，「許多物流工作仍然通過語音完成」，AI語音可以「自動化許多物流協調」。

#AI安全 AI Security
「我們在AI安全方面看到了絕對巨大的機會，從開發層一直到消費者層面。」未來「你可以為每個人甚至每個agent配置數百個AI安全agent。與物理世界不同，你不受物理空間甚至相同成本的限制。」

#開源AI  Open Source AI
「開源正處於AI旅程的關鍵時刻。」Sequoia認為「開源競爭並提供一些最好的最先進基礎模型非常重要」，這對於「一個更自由、更開放的未來至關重要，讓AI未來不只限於資金極其充裕的巨頭。」
