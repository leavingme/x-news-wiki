# Anthropic 拆订阅：Agent SDK 和 claude -p 有了独立预算池

## 摘要

【Anthropic 把 Agent SDK / claude -p 用量从主订阅池子里拆出，按订阅档位发"专用津贴"，等于官方给 Agent 玩家开了独立预算池。】

要点 1：Pro 用户 $20、Max 5x $100，按档位定档；用完才扣主套餐，不滚存，需一次性手动领取后自动续期。
要点 2：核心痛点是批量跑 Agent 把日常对话配额吃光，"用 Claude 写代码"和"用 Claude 干活"互抢池子；拆开后互不侵占。
要点 3：直接利好第三方 Agent App、本地脚本工作流、claude -p 批处理用户，Agent 实验门槛显著降低。
要点 4：信号层面，Anthropic 在把"订阅"重新定义为"主对话 + Agent 配额"的双轨结构，与 Cursor/Copilot "按席位计费"形成路线差。

gakki 锐评：表面是"加钱"，本质是订阅形态从"聊天窗口"迁向"AI 工位"——Agent 用量越涨越猛，按对话计费撑不住商业模型，独立配额是必然选择。唯一需要警惕的是"不滚存"这条，在悄悄引导用户每月用满。
