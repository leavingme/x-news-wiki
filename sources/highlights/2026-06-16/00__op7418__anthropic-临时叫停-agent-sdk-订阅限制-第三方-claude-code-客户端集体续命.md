# Anthropic 临时叫停 Agent SDK 订阅限制，第三方 Claude Code 客户端集体续命

## 摘要

**Anthropic 又一次在 ARR 与生态开放之间选择折中，CodePilot 们还能继续薅订阅。**
- 原计划 6 月起禁止基于 Agent SDK 的第三方客户端调用 Claude 订阅额度
- 本次回调意味着 CodePilot 等 Claude Code 替代品仍可使用 Pro/Max 订阅
- 短期利好独立开发者，长期反映 Anthropic 还在试探商业化边界

只要 Pro 订阅按月固定，第三方客户端的套利就堵不死。这次叫停本质是 Anthropic 对自家生态外溢的容忍度测试。

## 原文

Anthropic 叫停了关于 Agent SDK  六月以后不能使用 Claude 订阅额度的这个改动

就是说你还是可以在类似 CodePilot 这种基于 Agent SDK 开发的其他 Claude Code 三方客户端上，使用你的 Claude 订阅额度。
