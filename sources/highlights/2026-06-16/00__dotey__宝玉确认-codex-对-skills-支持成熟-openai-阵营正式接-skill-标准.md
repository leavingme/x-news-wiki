# 宝玉确认：Codex 对 Skills 支持成熟，OpenAI 阵营正式接 Skill 标准

## 摘要

**宝玉在回复中明确「Codex 对 Skills 支持很好」，意味着 SKILL.md 不再只是 Anthropic 系的内部规范。**

- Codex 端到端兼容 Skills 体系，打通了 Anthropic Claude Code 与 OpenAI Codex 之间的能力互操作层。
- 同一份 SKILL.md 可以在两个生态间复用，Skill 作者不再需要为不同 Agent 维护多份 manifest。
- SKILL.md 从「Anthropic 私有规范」跨过临界点，进入事实标准阶段。


gakki 锐评：当最大的竞争对手都开始兼容你的格式，那就不再是「你的标准」，而是「行业的基础设施」。SKILL.md 走到这一步，比 SKILL.md 本身的内容更重要。
