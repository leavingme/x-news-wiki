# 浙大×阿里云发布 Token Economics：首次把 Token 抬到「货币+生产要素」的理论高度

## 摘要

**Token 不再是计费单位，而是 AI 经济的硬通货**

• 浙大与阿里云联合提出 Token Economics 统一框架，把 token 同时定义为生产要素、交换媒介、货币三重角色，平衡输出质量与经济成本。
• 不同于以往分散的 token 优化综述，该框架覆盖单 Agent、多 Agent、生态系统三层，串联起优化、协作、安全三个维度。
• 为 Agent 自主交易、算力结算、激励设计提供了第一份可被引用的理论底座，呼应 NVIDIA / Jensen Huang 提出的 Token 经济学叙事。

academia 终于追上 industry 的叙事速度，Agent 经济最缺的不是模型，而是这本「货币手册」。

## 原文

Why are tokens becoming the currency of AI agents, and why is their exploding cost a crisis?

Researchers from Zhejiang University and Alibaba Cloud introduce Token Economics, the first unified framework that treats tokens as production factors, exchange mediums, and money. Their approach balances output quality against economic cost.

Outperforms fragmented surveys by synthesizing optimization, collaboration, and security across single-agent, multi-agent, and ecosystem levels—laying the theoretical foundation for scalable next-generation agent systems.
