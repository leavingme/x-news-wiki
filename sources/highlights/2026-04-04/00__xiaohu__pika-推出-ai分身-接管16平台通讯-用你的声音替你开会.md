# Pika 推出「AI分身」：接管16平台通讯、用你的声音替你开会

## 摘要

**一句话核心判断**：Pika 的 AI 分身不是帮你做事，而是「变成你」做事——这是 AI Agent 从工具到替身的关键一步。

- 分身可接管 16 个平台的通讯（Slack、WhatsApp、Instagram、LinkedIn 等），用你的风格发内容、回消息
- 支持说你不会的语言，替你参加 Google Meet 并自动出笔记
- 有持久记忆，能自主决策，不是简单的内容生成
- Pika-Skills 已开源，可让 Claude Code 等 Agent 直接调用「替身入会」能力

这不是 AI 助手，这是 AI 替身。接管通讯和会议意味着它需要你的完整数字身份授权——这个隐私边界值得警惕。

gakki：Pika 这个方向比大多数 AI 助手高明一个维度——从「执行指令」到「代理身份」。但 16 平台接管这个数字听起来很暴力，实际体验里「用你的风格」的边界在哪里？如果分身乱说话，谁来负责？

## 原文

沉寂许久的Pika 搞了个新东西

不是视频生成，是帮你造一个"AI 分身"

上传自拍、录制你的声音、回答性格问题，就能生出一个有你的脸、你的声音、你的说话习惯的 AI Self。

然后它可以：

→ 接管你 16 个平台的通讯（Slack、WhatsApp、Instagram、LinkedIn…） → 用你的风格发内容、回消息 

→ 说你不会的语言 → 甚至替你参加 Google Meet，会后自动出笔记

跟 AI 助手不一样，它不是帮你做事，是"变成你"去做事。有持久记忆，能自主决策。

Pika 还在 GitHub 开源了 Pika-Skills，让 Claude Code 这类 AI Agent 直接调用替身入会的能力。
