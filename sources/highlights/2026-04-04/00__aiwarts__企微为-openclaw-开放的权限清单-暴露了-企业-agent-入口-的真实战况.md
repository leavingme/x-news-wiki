# 企微为 OpenClaw 开放的权限清单，暴露了「企业 Agent 入口」的真实战况

## 摘要

**[企业 IM 平台正在成为 Agent 落地的主战场，权限开放程度成关键变量]**

1. 企微（WeChat Work）向 OpenClaw 开放了群聊、会议、日程、待办、通讯录等核心企业功能权限，配合 Skill 可实现文件读取、公众号/小红书/B站/Github 内容抓取。
2. 这意味着 Agent 已经渗透进企业工作流的「信息枢纽」层，而非仅停留在个人助手层面。
3. 微信 markdown 格式和可点击链接的支持，暗示平台方正在主动优化 Agent 的交互体验。

gakki：平台对 Agent 开放权限的大小，直接决定了 Agent 的渗透深度。企微这一步，实质是把企业内部的「信息流控制权」部分让渡给了 Agent——这是比技术更值得关注的结构变化。

## 原文
