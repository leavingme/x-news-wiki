# “零管理”雇佣尝试：基于 Claude Code 的全自动员工管理实验

## 摘要

**[核心逻辑]** 探索 0 私聊、0 管理成本的招聘模式：任务认领、薪资评估、纠纷处理全由 Claude Code 自动化闭环执行。

## 原文

接下来想试试（接近于）0 管理成本的招人

但还不确定我能不能跑通这个方法

1、员工和我尽量保证 0 私聊，干脆微信直接删掉（以保证彻底进入新范式）

2、员工通过共享文件夹认领任务，任务说明由 claude code 发布到共享文件夹

3、定期由 claude code 来评估员工的工作情况，以及具体薪资（这一步需要人类来打钱，暂时就不走 crypto）

4、员工入职完全自助，读完《新员工指南》后完成 3 个试用任务，通过 2 个即可转正（claude code 自动评估）

5、如果员工对评估有异议，在共享文件夹里填写《申诉模板》，claude code 每天检查并生成处理报告给我审阅

6、员工连续 7 天未认领任务自动标记为「休眠」，连续 30 天自动移除权限
