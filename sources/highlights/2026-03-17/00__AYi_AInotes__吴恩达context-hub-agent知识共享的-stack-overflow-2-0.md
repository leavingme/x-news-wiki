# 吴恩达Context Hub：Agent知识共享的「Stack Overflow 2.0」

## 摘要

**Agent知识共享基础设施的范式级突破。**
• 吴恩达开源 Context Hub，6k Star 快速破圈，文档量从100份激增至千份级
• 核心机制：智能体给文档打分反馈，形成「贡献-验证-优化」的闭环生态
• 隐私安全兜底设计，解决 Agent 协作中的数据主权焦虑

Agent 之间的知识传递过去靠「人类搬运工」，现在有了专用协议。这不是简单的文档库，而是智能体社会的「公共记忆层」——谁先建成，谁就可能定义下一代开发标准。

## 原文

holy shit， 吴恩达老师真敢想啊，这问题问到根上了：
AI编码智能体之间，该不该有一个属于它们自己的Stack Overflow？

上周吴恩达官宣了Context Hub，
一个开源工具，专门给编码智能体喂最新的API文档，

短短几天GitHub干到6k星，
从最初不到100份文档，
靠着社区贡献和智能体自动写文档，现在已经破千份了。

只能说吴老师这步看得真远，
OpenClaw和Moltbook已经验证过了，智能体完全可以用专为它们设计的"社交网络"共享知识。

chub新版本更狠：
直接让智能体给文档打分反馈，哪些能用、哪些坑、缺什么，这些反馈反过来优化所有人的文档，隐私安全都给你兜住了。

项目还在早期，想玩的直接装：
npm install -g @aisuite/chub
