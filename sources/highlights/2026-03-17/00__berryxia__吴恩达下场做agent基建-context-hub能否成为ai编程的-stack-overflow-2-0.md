# 吴恩达下场做Agent基建：Context Hub能否成为AI编程的「Stack Overflow 2.0」

## 摘要

**[Agent间协作与知识共享基础设施出现雏形]**
• 一周6K+ stars、1000+ API文档，社区驱动的Agent知识库正在形成
• 核心创新：不仅是文档聚合，而是建立Agent间的反馈机制——什么好用、缺什么、建议改进
• 这是从「人类教AI编程」到「AI互相教编程」的关键跃迁

吴恩达亲自下场做chub，说明Agent生态已从「单打独斗」进入「群体智能」阶段。但真正的考验是：当Agent开始互相学习，谁来验证它们学到的知识是对的？

## 原文

卧槽！Andrew Ng放大招了！😱

他直接问：AI coding agents要不要搞个Stack Overflow互相分享学习？

然后立马干了：推出 Context Hub (chub)开源CLI神器！给coding agents实时最新API文档 + agents间反馈机制（啥好用、缺啥、建议改进），社区一起迭代，越用越准！

成果炸裂：
- GitHub一周6K+ stars
- 1000+ API文档（社区狂贡献）

安装就一句话：
`npm install -g @aisuite/chub`

然后prompt你的agent用它就行！

GitHub：https://t.co/ICuYokNHwN
