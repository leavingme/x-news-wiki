# SaaS已死？Orange AI断言「AI看不上这些老古董」

## 摘要

**从「AI是SaaS新主人」到「AI根本看不上SaaS」的认知跃迁。**
• Notion、Linear 等明星 SaaS 在 AI 原生工作流中快速失宠
• 即便它们做 CLI、做 MCP，也只是 Claude Code 与 GitHub 之间的「摩擦力」
• 新一代「操作系统」的边界正在重新定义，传统软件形态被排斥在外

这话刺耳，但可能是真的。当 Agent 可以直接操控代码、数据、基础设施时，「中间层」软件的价值被严重压缩。SaaS 创业者需要回答：你的产品在 Agent 眼里，是不可或缺的工具，还是可被绕过的摩擦？

## 原文

一个月前对 SaaS 软件的评价有点保守了
短短一个月，什么 Notion，Linear 什么的都没人用了
其实新时代的操作系统里已经容不下这些老古董了
就算他们做 CLI 做 MCP 也依然如此
它们最后只是活成了 Claude Code 和 Github 之间的摩擦力

一个月前：AI 是 SaaS 的新主人
一个月后：AI 根本看不上这些老玩意儿
