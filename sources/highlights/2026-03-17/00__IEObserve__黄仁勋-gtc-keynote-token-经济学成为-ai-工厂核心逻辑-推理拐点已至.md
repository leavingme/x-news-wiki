# 黄仁勋 GTC  keynote：Token 经济学成为 AI 工厂核心逻辑，推理拐点已至

## 摘要

**黄仁勋将 AI 基础设施市场展望从 5000 亿美元上修至 2027 年 1 兆美元，核心逻辑是数据中心从「存储文件」转向「生产 Token」。**

• 过去两年运算需求暴增 100 万倍，来自三大转折：生成式 AI 改变运算方式、推理 AI（o1/o3）让 AI 能拆解问题、Claude Code 让 AI 真正能执行任务
• Token 分层定价体系成形：免费层获客 → 中层 3-6 美元/百万 Token → 高阶 45 美元 → 顶级 150 美元，AI 越聪明价格越高但吞吐量越低
• Grace Blackwell 实现 35-50 倍每瓦效能提升，同一数据中心营收可增加 5 倍
• 演讲明确提及 OpenClaw 龙虾 Agent 趋势，暗示英伟达对 Agent 执行层的高度关注

黄仁勋正在定义 AI 时代的「算力-营收」转换公式。当 Token 成为核心商品，拥有推理基础设施就等于拥有印钞机。这不是泡沫，是定价权的重新分配。

## 原文

老黃在 GTC 2026 描繪了一個以 token 為核心商品的全新運算經濟體系，核心命題是：AI 工廠如何將電力轉化為營收。而且也同樣花了不小的篇幅在講OpenClaw龍蝦agent的趨勢。
https://t.co/CCLRXp1mmZ

他指出過去兩年運算需求暴增 100 萬倍，來自三個轉折：生成式 AI 改變運算方式、推理 AI（o1/o3）讓 AI 能拆解問題、Claude Code 讓 AI 真正能執行任務。「推論的拐點已經到來。」

他將展望從去年的 5000 億美元上修到 2027 年至少 1 兆美元的基礎建設需求。

演講最精華的部分是 #Token經濟學。「你的資料中心過去存放檔案，現在是生產 token 的工廠。」他用分層定價來說明：免費層吸引客戶，中間層每百萬 token 約 3 到 6 美元，高階層 45 美元，頂級服務可達 150 美元。AI 越聰明價格越高，但吞吐量越低，所以架構最佳化決定一切。

Grace Blackwell 相較 Hopper 實現 35 到 50 倍的每瓦效能提升，同一座資料中心營收增加 5 倍。新發布的 Vera Rubin 在最有價值的推論層級再提升 10 倍，營收再翻 5 倍。整合 Groq LPU 後，超高速 token 生成再提升 35 倍。1 GW 工廠中，兩年內 token 生成速率從 200 萬提升到 7 億。

他用一句話點出成本邏輯：「一座 1 GW 工廠攤提 15 年就是 400 億美元，即使上面什麼都不放。你最好確保放上最好的系統。」

#產品路線圖 ，Rubin Ultra 正在 tape out，搭配 Kyber 機架可連接 144 顆 GPU。下一代 Feynman 包含全新 GPU、LP40 LPU、Rosa CPU，同時支援銅線和共封裝光學。承諾每年推出全新架構。

另一個重磅是 #OpenClaw。Jensen Huang 將它定義為 agent 電腦的作業系統：「就像 Windows 讓我們創造個人電腦，OpenClaw 讓我們創造個人 agent。」他預言每家 SaaS 公司都將轉型為 AaaS（Agent as a Service），token 預算將成為矽谷新的招聘工具。
