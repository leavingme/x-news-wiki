# 「一个人的华尔街」：Claude + Public.com API 让量化交易 Agent 零门槛落地

## 摘要

**[金融 Agent 的「极简主义」范式——用自然语言替代专业终端]**
• Claude Desktop + Public.com 免费 API 实现实时股票报价、期权链、希腊值查询，月成本仅 $20
• 无需编程，通过自然语言配置即可完成「研-判-执」闭环，打破 Bloomberg Terminal 的专业壁垒
• 标志着个人投资者与机构级信息差的结构性缩小

当 $20/月的 AI 能调取实时市场数据并执行分析，华尔街的信息护城河正在被「自然语言接口」填平。这不是工具平权，这是认知平权的前奏。

## 原文

堪称一个人的华尔街！

Jason Luongo 发布了一个教程，宣布 Claude AI 的桌面版现在可以通过 https://t.co/EYMIC70GPV 的免费 API 接入实时股票报价、期权链、希腊值和个人持仓数据。

核心要点总结如下：

- 需要 Claude Pro 订阅（20美元/月）+ Claude 桌面应用（支持 MCP 工具连接）。
- https://t.co/EYMIC70GPV 开免费账户即可获得实时市场数据 API 权限（无佣金、无 PFOF）。
- 连接步骤非常简单：
  1. 注册 Public 免费账户。
  2. 下载并安装 Claude Desktop 应用。
  3. 在终端运行一条命令：pip install publicdotcom-py。
  4. 在 Claude 的 Cowork 界面直接让 AI 帮你编辑配置文件（输入你的 API key 和账户 ID，无需手动改代码）。
  5. 重启 Claude 应用，验证是否能拉取实时数据（如问 NVDA 当前股价）。

完成后，你可以在同一个聊天窗口里：
- 直接拉取完整期权链（含 bid/ask、成交量、未平仓量）。
- 查询任意合约的希腊值（delta、IV 等）。
- 查看自己 Public 账户的真实持仓和盈亏。
- 结合 Claude 的网页搜索功能做策略分析（例如同时查价格 + IV rank + 给出卖 put 或买 LEAP 的建议）。

全部数据接入免费，设置一次后所有股票相关对话都有实时数字支持。目前没有其他 AI 能做到同样程度的实时经纪商数据整合。

（GitHub 链接也在帖子里：https://t.co/8Fr8e8lY0I）
