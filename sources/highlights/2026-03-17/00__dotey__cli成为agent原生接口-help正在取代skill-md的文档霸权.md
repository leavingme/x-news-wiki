# CLI成为Agent原生接口：--help正在取代SKILL.md的文档霸权

## 摘要

**[软件正在被Agent反向定义架构]**
• 趋势判断：越来越多的工具开始为Agent使用场景重新设计，而非让人类用户界面主导
• 技术洞察：CLI的--help机制天然适合Agent自发现，可能比专门的SKILL.md更高效
• 范式影响：这是MCP vs CLI工程范式之争的实战印证，简单可发现性优于复杂编排

#gakki锐评：当工具开始为Agent而不是为人类设计时，说明Agent已经从"被集成"变成了"主导者"。这不是UI的消亡，是交互原点的根本性转移。
