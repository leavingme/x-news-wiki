# 上交+CMU联手：DataMaster把数据工程变成「树搜索」，MLE-Bench获奖率猛拉32%

## 摘要

**当数据工程被当作 tree search 来解：自主 Agent 在 MLE-Bench Lite 上把奖牌率拉高 32.27%**

- 上海交大 + CMU 等机构发布 DataMaster：把数据工程拆成「分支探索外部数据集 → 筛选清洗 → 共享记忆复用」的树搜索结构
- 在 MLE-Bench Lite 上比起点分数提升 32.27%，PostTrainBench 上也有显著增益——证明「数据准备」比「算法调参」更可能是模型卡点的源头
- 核心思路不是更大的模型，而是把工程师的隐性经验（选数据集、清洗 pipeline）外化成可复用的搜索策略

## 原文

Is your AI model stuck because of bad data, not bad algorithms?

Researchers from Shanghai Jiao Tong University, Carnegie Mellon University, and other top institutions present DataMaster: an autonomous agent that treats data engineering like a tree search—branching to discover external datasets, selecting and cleaning them, then reusing what works via shared memory and a central pool.

Results: On MLE-Bench Lite, it boosts the medal rate by 32.27% over the starting score. On PostTrainBench, it edges past the instruct model on GPQA (31.02% vs 30.35%). No algorithm changes needed.
