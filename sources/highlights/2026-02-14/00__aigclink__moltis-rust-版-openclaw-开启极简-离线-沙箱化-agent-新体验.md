# moltis：Rust 版 OpenClaw 开启极简、离线、沙箱化 Agent 新体验

## 摘要

**[核心逻辑]** moltis 以单个静态二进制文件实现零 Node.js 依赖，支持环境变量自动脱敏与 Docker 隔离执行。

## 原文

Rust版的OpenClaw：moltis，单个静态二进制文件，零Node.js依赖

统一接口接入Codex、Copilot以及本地LLM，支持故障转移链

内置本地模型支持，自动下载配置模型，可完全离线运行

所有命令在Docker或Apple Container沙箱中会话级隔离执行；环境变量自动脱敏（覆盖明文、base64、hex 形式）；首次运行终端打印设置码，杜绝默认密码风险

基于SQLite的混合存储，向量语义搜索结合全文检索；文件监听实时同步；上下文窗口达95%时自动压缩精简
