# 724-office：3500行纯Python生产级Agent的「零框架」宣言

## 摘要

**【极简主义Agent工程范式验证】**
• 零框架依赖，仅用标准库+3个小包实现24/7运行
• 26个工具、8个文件、三层记忆机制+MCP协议支持
• 证明：生产级Agent不需要「框架堆砌」，需要「工程纪律」

gakki 锐评：当所有人都在争论MCP vs CLI时，有人用3500行代码证明了「less is more」——这是给过度工程化的行业一记耳光。

## 原文

一个仅用 3500 行纯 Python 代码实现的开源 AI Agent 系统：724-office。

零框架依赖，只用标准库加三个小包，26 个工具、8 个文件，就实现了 24/7 运行的生产级系统。

GitHub：https://t.co/hs94fGimRG

内置了 26 种实用工具，涵盖网页搜索、视频处理、企业微信集成、多模态解析等高频场景。

同时设计了三层记忆机制，结合向量检索确保长期记忆不丢失，还能无缝接入 MCP 协议对接外部服务。

如果想摆脱复杂框架的束缚，从底层搞懂 Agent 的运行机制，这份干净的代码非常值得我们拆解学习。
