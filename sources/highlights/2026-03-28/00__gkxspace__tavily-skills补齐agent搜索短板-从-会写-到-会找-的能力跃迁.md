# Tavily Skills补齐Agent搜索短板：从「会写」到「会找」的能力跃迁

## 摘要

**[搜索能力已成为AI Agent的体验分水岭]**
• Tavily Skills将search/extract/crawl/map/research整合为统一技能层
• Claude Code等编码Agent的弱搜索痛点被精准击中，信息获取不再依赖人工切换
