# 飞书CLI开源：中国协作平台的Agent-Native转折

## 摘要

**[飞书率先将核心业务能力全面CLI化，等于向所有Agent敞开了大门]**

• 11个业务域、200+命令全量开放，日历/文档/消息/任务皆可命令调用
• 标志着中国企业软件从GUI向Agent-Native架构的战略迁移
• 与Claude Code/Codex的潜在组合，可能诞生比「小龙虾」更强大的飞书原生Agent体验

飞书这次走在了钉钉和企业微信前面——当美国还在吵MCP协议时，中国玩家直接把API变成了生产力管道。
