# CyberStrikeAI：100+安全工具AI编排，渗透测试进入"对话式自动化"时代

## 摘要

**[Agent在垂直安全领域的深度整合]**
• 自然语言调度 100+ 安全工具，自动生成攻击链图谱与审计日志
• 12种预定义角色+20+技能库，覆盖 SQL 注入、XSS、云安全等场景
• 支持钉钉/飞书机器人，实现移动端指令下发

**gakki 锐评**：安全测试的复杂性天然适合 Agent 编排——攻击链的生成、追溯、协作，正是多 Agent 架构的典型用武之地。

## 原文

做渗透测试，工具一堆，命令行一堆，还要自己拼接攻击链，记录漏洞，整理报告，一个项目下来工作量颇大。

不妨试下 CyberStrikeAI 这个开源项目，用 AI 把整个安全测试流程串起来了，从对话到漏洞发现全自动化。

通过自然语言对话，就能调度 100 多个安全工具，自动生成攻击链图谱，还能追溯每一步操作。

GitHub：https://t.co/CuFh7RoIbC

内置了 12 种预定义安全测试角色，涵盖渗透测试、CTF、Web 扫描、API 安全等，每个角色有专门的提示词和工具限制。

同时还配了 20 多个安全测试技能库，包含 SQL 注入、XSS、云安全等等，AI 可以按需调用。

以及支持批量任务队列，多个测试任务按顺序执行，每个任务都有独立的对话记录和状态追踪。

提供 Web 控制台、攻击链可视化、漏洞管理、任务监控等功能，所有操作都有审计日志可追溯。

除此之外，还支持钉钉和飞书机器人，手机上也能下发测试指令。

如果你想把零散的安全工具和测试流程整合成一套可追溯、可协作的自动化平台，这个项目值得试试。
