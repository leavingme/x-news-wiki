# 金融背景文科生72小时冲上GitHub全球前30：AI组队的"一人公司"样本

## 摘要

**[技术门槛的坍塌正在创造新的人口红利]**
• 零代码背景，通过 Echo/Elon/Henry 三个核心 Agent 分工协作完成项目
• 本人仅负责需求定义与方向把控，Agent 承担执行、提 PR、推广等全链路工作
• OpenClaw 项目获得 Sam Altman 亲自官宣，验证了"非技术背景+AI"模式的可行性

**gakki 锐评**：这不是个例，而是一个信号——发现真实需求的能力正在取代编码能力，成为创业的第一门槛。

## 原文

一个文科生72小时冲上GitHub全球贡献榜前30！

这哥们叫杨天润，Naughty Labs CEO，本科研究生都是金融专业，以前做并购投资的

一行代码都没写过，前几天才搞懂PR是啥

然后他直接用AI组了个小团队：三个核心 Agent
Echo管协调，Elon负责写代码和提PR，Henry帮忙推推广

自己就负责说清楚需求、盯住方向

项目就是OpenClaw，Sam Altman亲自发推官宣创始人加入OpenAI的那个19万星AI Agent开源项目

💡所以，在这个AI时代，代码真不是门槛了
